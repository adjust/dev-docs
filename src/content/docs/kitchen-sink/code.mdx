Use code blocks to visually distinguish snippets of code inside documents. This site uses [Expressive code](https://github.com/expressive-code/expressive-code/) to add features to code blocks.

## Syntax highlighting

Syntax highlighting is added by default when you add the code language name after the backtick fence.

````markdown "swift"
```swift
let authorizationStatus = Adjust.appTrackingAuthorizationStatus()
```
````

```swift
let authorizationStatus = Adjust.appTrackingAuthorizationStatus()
```

## Titles

You can add a title to a code block by passing a `title` directive after the backtick fence.

````markdown 'title="cURL"'
```console title="cURL"
$ curl -X POST https://app.adjust.com/api/v1/create
```
````

```console title="cURL"
$ curl -X POST https://app.adjust.com/api/v1/create
```

## Content highlights

You can highlight parts of a code block to draw attention to them.

### Highlight lines

To highlight lines, add the lines or line ranges you wish to highlight in curly braces after the backtick fence.

````markdown "{3, 5-8}"
```json {3, 5-8}
[
   {
      "id": "string",
      "name": "string",
      "short_name": "",
      "section": "",
      "formatting": "money",
      "increase_is_negative": false,
      "description": "",
      "app_token": ["string"],
      "tokens": ["string"]
   }
]
```
````

```json {3, 5-8}
[
   {
      "id": "string",
      "name": "string",
      "short_name": "",
      "section": "",
      "formatting": "money",
      "increase_is_negative": false,
      "description": "",
      "app_token": ["string"],
      "tokens": ["string"]
   }
]
```

### Highlight words

You can highlight individual words or sentences by including them in quotes after the backtick fence.

<Callout type="important">

Items that are contained in quotes within the code block must be *escaped*.

</Callout>

````markdown "'\"short_name\"'" '"false"'
```json '"short_name"' "false"
[
   {
      "id": "string",
      "name": "string",
      "short_name": "",
      "section": "",
      "formatting": "money",
      "increase_is_negative": false,
      "description": "",
      "app_token": ["string"],
      "tokens": ["string"]
   }
]
```
````


## Collapsible code

If you want to hide some information in a code block, you can collapse it to hide it from view until the reader selects the expand option. To do this, pass a `collapse` directive with the line or range of lines you want to collapse in curly braces.

````markdown "collapse={5-8}"
```json collapse={5-8}
[
   {
      "id": "string",
      "name": "string",
      "short_name": "",
      "section": "",
      "formatting": "money",
      "increase_is_negative": false,
      "description": "",
      "app_token": ["string"],
      "tokens": ["string"]
   }
]
```
````

```json collapse={5-8}
[
   {
      "id": "string",
      "name": "string",
```
