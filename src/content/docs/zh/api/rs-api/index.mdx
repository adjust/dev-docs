通过报告服务 API，您可以查询 Adjust 的报告数据集，获取聚合数据。使用这些终端来轻松过滤、排列和提取您的 Adjust 数据。

## 1\. 认证您的调用 {#1-authenticate-your-calls}

要使用报告服务 API 终端，您首先需要获取 Adjust API 识别码。按照此[认证文档](/zh/api/rs-api/authentication)中的说明进行操作，了解如何在 API 中应用识别码。

## 2\. 找到您的指标 {#2-find-your-metrics}

Adjust 支持您使用多种指标过滤报告。其中很多指标可基于单个应用或事件应用。

您可以在我们的[Datascape 指标术语表](https://help.adjust.com/zh/article/datascape-metrics-glossary)中查看完整指标列表。您还可以使用[过滤器数据终端](/zh/api/rs-api/filters-data)来程序化搜索指标。您还可以通过[事件终端](/zh/api/rs-api/events)返回由机器生成的事件 slug，以便在 CSV 报告中使用。

## 3\. 选择您的响应格式 {#3-choose-your-response-format}

报告服务 API 包含三种不同报告格式的终端。选择您需要接收格式对应的终端。

-  报告终端：返回 JSON 格式的数据。
-  CSV 报告终端：返回 CSV 格式的数据。
-  透视报告终端：返回 JSON 格式的数据，其中包含一个按维度细分总计数据的字段。
