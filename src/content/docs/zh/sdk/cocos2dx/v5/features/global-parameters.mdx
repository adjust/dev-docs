您在 Adjust 控制面板中[注册回传 URL](https://help.adjust.com/zh/article/recommended-placeholders-callbacks)，SDK 监测到会话后，Adjust 会向您的回传 URL 发送一个 GET 请求。

## 统一回传参数 {#global-callback-parameters}

<Callout type="note">

Adjust SDK 会合并会话回传参数和事件回传参数。事件回传参数优先级高于会话回传参数。这意味着，如果将一个参数键同时添加到事件和会话，那么 SDK 会发送事件参数。

</Callout>

您可以设置发送到服务器的统一回传参数。Adjust SDK 会将所有统一回传参数附加到您的回传 URL 中。您可以利用该信息，通过自己的 BI 系统分析用户应用内行为。

### 添加统一回传参数 {#add-global-callback-parameters}

使用 `std::string` 键值对参数调用`Adjust2dx::addGlobalCallbackParameter`方法，以此添加统一回传参数。多次调用该方法可添加多个参数。

```cpp
Adjust2dx::addGlobalCallbackParameter("foo", "bar");
```

### 移除统一回传参数 {#remove-global-callback-parameters}

您可以移除不再需要的特定统一回传参数。为此，请将参数 `key` 传递至 `Adjust2dx::removeGlobalCallbackParameter` 方法。

```cpp
Adjust2dx::removeSessionCallbackParameter("foo");
```

### 重置统一回传参数 {#reset-global-callback-parameters}

如果您不再需要统一参数，则可以将所有参数一并移除。为此，请调用 `Adjust2dx::removeGlobalCallbackParameters` 方法。

```cpp
Adjust2dx::removeGlobalCallbackParameters();
```

## 统一合作伙伴参数 {#global-partner-parameters}

您可以添加统一[合作伙伴参数](https://help.adjust.com/zh/article/data-sharing-ad-network#map-parameters)，向渠道合作伙伴发送额外的信息。

Adjust 可向您设置的[外部合作伙伴](https://help.adjust.com/zh/article/integrated-partners)发送统一合作伙伴参数。这些信息可用来进行更精细的数据分析，开展再营销活动。您设置好参数并为合作伙伴启用参数转发后，Adjust 服务器就会将这些参数转发给合作伙伴。

<Callout type="note">

合作伙伴参数不会默认出现在原始数据导出中。您可以添加`{partner_parameters}`占位符，在同一个字符串中接收参数。

</Callout>

### 添加统一合作伙伴参数 {#add-global-partner-parameters}

通过`std::string`键值参数调用`Adjust2dx::addGlobalPartnerParameter`方法来发送统一合作伙伴参数：多次调用该方法可添加多个参数。

```cpp
Adjust2dx::addGlobalPartnerParameter("foo", "bar");
```

### 移除统一合作伙伴参数 {#remove-global-partner-parameters}

您可以移除不再需要的特定统一合作伙伴参数。为此，请将参数 `key` 传递至 `Adjust2dx::removeGlobalPartnerParameter` 方法。

```cpp
Adjust2dx::removeGlobalPartnerParameter("foo");
```

### 重置统一合作伙伴参数 {#reset-global-partner-parameters}

如果您不再需要统一合作伙伴参数，则可以将所有参数一并移除。为此，请调用 `Adjust2dx::removeGlobalPartnerParameters` 方法。

```cpp
Adjust2dx::removeGlobalPartnerParameters();
```
