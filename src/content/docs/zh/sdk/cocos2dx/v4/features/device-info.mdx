Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## Adjust 设备标识符

<CodeBlock title="方法签名">

```cpp
static std::string getAdid();
```

</CodeBlock>

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `getAdid` 方法来通过 **字符串** 的形式返回此 ID。

```cpp
std::string adid = Adjust2dx::getAdid();
```

## 广告主 ID

<CodeBlock title="方法签名">

```cpp
static std::string getIdfa();
```

</CodeBlock>

广告主 ID \(IDFA\) 是 Apple 设备的独特标识符。调用 `getIdfa` 方法来通过 **字符串** 的形式返回此 ID。

```cpp
std::string idfa = Adjust2dx::getIdfa();
```

## Google Play 服务广告 ID

<CodeBlock title="方法签名">

```cpp
static void getGoogleAdId(void (*adIdCallback)(std::string adId));
```

</CodeBlock>

Google Play 服务广告 ID \(GPS ADID\) 是安卓设备的独特标识符。

用户可以通过启用设备的 "选择停用广告个性化功能" 设置来停止分享自己的 GPS ADID。当用户启用该设置后，Adjust SDK 试图读取 GPS ADID 时会返回一串零。

您可以通过在后台线程中调用 `getGoogleAdId` 方法来获取该值。指定一个委托 \(delgate\) 函数，以访问 GPS ADID 值。

```cpp
static void adIdCallbackMethod(std::string adId) {
    CCLOG("\nAdvertising identifier = %s", adId.c_str());
    CCLOG("\n");
}

// ...

Adjust2dx::getGoogleAdId(adIdCallbackMethod);
```

## Amazon 广告主 ID

<CodeBlock title="方法签名">

```cpp
static std::string getAmazonAdId();
```

</CodeBlock>

Amazon 广告 ID \(Amazon Ad ID\) 是安卓设备的独特标识符。调用 `getAmazonAdId` 方法来通过 **字符串** 的形式返回此 ID。

```cpp
std::string idfa = Adjust2dx::getAmazonAdId();
```
