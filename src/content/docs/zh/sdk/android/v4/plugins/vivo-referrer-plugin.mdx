通过 Vivo referrer 插件，Adjust SDK 可以读取设备上的 Vivo 应用商店 install referrer。SDK 随后会将此信息作为安装 payload 的一部分发送至 Adjust 服务器。

所有使用 Vivo 应用商店 5\.3\.0\.0 或以上版本的设备都可以使用 Vivo install referrer。

要将 Adjust SDK 集成进应用，请按 Android 入门指南中的说明操作。

## 安装 Vivo referrer 插件 {#install-the-vivo-referrer-plugin}

安装 Vivo referrer 插件有两种方式：

### Maven {#maven}

将以下依赖添加至 Adjust SDK 声明下的`build.gradle`文件。

<CodeBlock title="build.gradle">

```groovy
implementation 'com.adjust.sdk:adjust-android:$ANDROID_V4_VERSION'

implementation 'com.adjust.sdk:adjust-android-vivo-referrer:$ANDROID_V4_VERSION'

```

</CodeBlock>

### JAR {#jar}

您可以在[GitHub 发布页面](https://github.com/adjust/android_sdk/releases/latest)找到 Vivo referrer 插件。

## 使用插件 {#use-the-plugin}

Vivo referrer 插件默认为 **开启** 状态。您可以使用`AdjustVivoReferrer`类方法来开启或关闭 Vivo install referrer 信息的读取。

要 **禁用** install referrer 读取，请在初始化 SDK 之前调用 `AdjustVivoReferrer.doNotReadVivoReferrer()`。

<Tabs>
<Tab title="Java" sync="java">

```java
AdjustVivoReferrer.doNotReadVivoReferrer();
//...
Adjust.onCreate(config);
```

</Tab>
<Tab title="Kotlin" sync="kotlin">

```kotlin
AdjustVivoReferrer.doNotReadVivoReferrer()
//...
Adjust.onCreate(config)
```

</Tab>
</Tabs>

要 **启用** install referrer 读取，请在初始化 SDK **之前** 调用 `AdjustVivoReferrer.readVivoReferrer(applicationContext)`。

<Tabs>
<Tab title="Java" sync="java">

```java
AdjustVivoReferrer.readVivoReferrer(applicationContext);
//...
Adjust.onCreate(config);
```

</Tab>
<Tab title="Kotlin" sync="kotlin">

```kotlin
AdjustVivoReferrer.readVivoReferrer(applicationContext)
//...
Adjust.onCreate(config)
```

</Tab>
</Tabs>
