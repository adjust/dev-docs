如果您想使用 Admost SDK 监测广告收入，可以借助 SDK 到 SDK 集成，将数据发送到 Adjust 服务器。

<Callout type="note">

如果您对 Admost 广告收入监测有任何疑问，请联系您的专属技术客户经理，或发送邮件至 [support@adjust.com](mailto:support@adjust.com)。

</Callout>

## 操作前须知 \{\#before\-you\-begin\}

要启用该功能，请先联系您的 Admost 代表，为您的应用启用 SDK 到 SDK 收入数据分享。

### 要求{#requirements}

- 在您的应用中集成 Admost安卓SDK v2\.4\.5 或更新版本。
- 在您的应用中集成 Adjust安卓SDK v4\.28\.5 或更新版本。
