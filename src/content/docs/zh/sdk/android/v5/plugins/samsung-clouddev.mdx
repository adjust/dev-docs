Adjust SDK 通过插件支持 [Samsung CloudDev SDK](https://developer.samsung.com/instant-plays/samsung-clouddev-sdk/get-started.html)

## 操作前须知 {#before-you-begin}

1. 要将 Adjust SDK 集成进应用，请按[入门指南](/zh/sdk/android)中的说明操作。
2. 向您的 Samsung 索取一份 Samsung CloudDev SDK 的副本，将其加入您的项目。

## 安装 Samsung CloudDev 插件 {#install-the-samsung-clouddev-plugin}

安装 Samsung CloudDev 插件有两种方式：

### 通过 Maven 安装 {#install-with-maven}

将以下依赖添加至 Adjust SDK 声明下的`build.gradle`文件。

<CodeBlock title="build.gradle">

```groovy
implementation 'com.adjust.sdk:adjust-android-samsung-clouddev:4.38.3'
```

</CodeBlock>

### 作为 JAR 或 AAR 添加 {#add-as-a-jar-or-aar}

Samsung CloudDev 插件以 Java Archive \(JAR\) 和 Android Archive \(AAR\) 形式存在于[Adjust SDK 库](https://github.com/adjust/android_sdk/releases/latest)中
