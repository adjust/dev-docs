Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## Adjust 设备标识符{#adjust-device-identifier}

<CodeBlock title="方法签名">

```cs
public static string GetAdid();
```

</CodeBlock>

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `GetAdid` 方法来通过 `string` 的形式返回此 ID。

```cs
string adid = Adjust.GetAdid();
```

## Windows 广告标识符{#windows-advertising-identifier}

<CodeBlock title="方法签名">

```cs
public static string GetWindowsAdId();
```

</CodeBlock>

Windows 广告 ID \(Windows ADID\) 是 Windows 设备的独特标识符。调用 `GetWindowsAdId` 方法来通过 `string` 的形式返回此 ID。

```cs
var windowsAdid = Adjust.GetWindowsAdId();
```
