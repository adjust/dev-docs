您可以在初始化后隐藏智能横幅，在需要时再次显示。为此请调用`AdjustSmartBanner.hide`方法。

<Callout type="note">

隐藏智能横幅后并不会将其从 DOM 中移除。

</Callout>

```js
AdjustSmartBanner.hide();
```

要再次显示智能横幅，请调用`AdjustSmartBanner.show`方法。

<Callout type="important">

如果您的网页应用是 SPA \(单页应用\)，则应该在出现导航且页面 URL 发生变化后调用该方法。此操作会强制 SDK 读取更新后的页面 URL，从而为当前页面找到合适的横幅，或在通过动态深度链接构建链接时使用更新后的 GET 参数。

</Callout>

```js
AdjustSmartBanner.show();
```
