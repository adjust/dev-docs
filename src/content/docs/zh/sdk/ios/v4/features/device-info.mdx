Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## Adjust 设备标识符 {#adjust-device-identifier}

<CodeBlock title="Method signature">

```objc
+ (nullable NSString *)adid;
```

</CodeBlock>

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `adid` 方法来通过 `NSString` 的形式返回此 ID。

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let adid = Adjust.adid()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *adid = [Adjust adid];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
var adid = Adjust.getAdid();
```

</Tab>
</Tabs>

## 广告主 ID {#id-for-advertisers}

<CodeBlock title="Method signature">

```objc
+ (nullable NSString *)idfa;
```

</CodeBlock>

IDFA \(广告主 ID\) 是 Apple 设备的独特标识符。调用 `idfa` 方法来通过 `NSString` 的形式返回此 ID。

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let idfa = Adjust.idfa()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *idfa = [Adjust idfa];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
Adjust.getIdfa(function (idfa) {
   // …
});
```

</Tab>
</Tabs>
