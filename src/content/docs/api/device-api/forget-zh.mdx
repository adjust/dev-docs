通过 Adjust 忘记设备终端来清除 Adjust 上的设备历史记录，并在同一设备上测试多个安装。
<callout type="seealso" title="操作前须知"> 

您需要拥有 **管理员** 或 **编辑员** 权限才能遗忘设备。如果您拥有 **自定义编辑员** 权限，则需要对目标应用有编辑权限。要了解更多信息，请查看[用户权限](https://help.adjust.com/en/article/users)。
</Callout> 

```http
POST https://api.adjust.com/device_service/api/v1/forget_device
```

清除设备
----

从 Adjust 清除设备信息。
<table> 

|       参数       | 数据类型 | 位置 |      描述       |
|----------------|------|----|---------------|
| `app_token`\* | 字符串  | 数据 | 应用的 12 位数标识符  |
| `adid`\*      | 字符串  | 数据 | 设备的 Adjust ID |

</Table> 

示例
---

```console
$ curl \
--header "Authorization: Token token=ask43jskdp2tg2hg87" \
--location --request POST "https://api.adjust.com/device_service/api/v1/forget_device" \
--data "adid=acf8534f2f052395e617a38730682ccc&app_token=gwzpeepw8uf8"
```

```http
200: Forgot device
```

<table>
