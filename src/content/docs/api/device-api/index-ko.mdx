디바이스 API를 통해 사용자의 디바이스 정보를 확인하고, 테스트 목적으로 진행한 Adjust에서 디바이스 활동 내역을 삭제해보시기 바랍니다.

## 작동 방식

1. **콜 인증**

   디바이스 API 엔드포인트를 사용하려면 Adjust API 토큰이 필요합니다. [인증 가이드](/ko/api/device-api/authentication) 내 설명을 참조하여 해당 토큰을 API와 함께 사용하는 방법을 확인하시기 바랍니다.

2. **디바이스 세부사항 검사**

   [디바이스 엔드포인트 검사\(inspect device endpoint\)](/ko/api/device-api/inspect)를 사용하여 기기 및 사용자 활동에 대한 세부 정보를 반환합니다.

3. **디바이스 삭제**

   Adjust에서의 디바이스 내역을 삭제하려면 [Forget device 엔드포인트](/ko/api/device-api/forget)를 사용하시기 바랍니다. 이는 동일한 장치에서 발생한 다수의 설치를 테스트할 때 유용합니다.
