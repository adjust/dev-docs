レポートサービスAPI（Report Service API）により、Adjustのレポートデータセットにクエリを実行して、集計データを取得することができます。これらのエンドポイントを使って、Adjustから簡単にデータを取得できます。

1\. コールを認証する \{\#authenticate\-your\-calls\}
--------------------------------------------------

このレポートサービスAPIエンドポイントを使用するには、Adjust APIトークンが必要です。このトークンをAPIで使用するには、[認証ガイド](/en/api/rs-api/authentication)の手順に従ってください。

2\. 指標を見つける \{\#find\-your\-metrics\}
-------------------------------------------

Adjustは、レポートのフィルタリングに使用できる多数の指標をサポートしています。これらの指標の多くは、アプリごとまたはイベントごとに存在します。

指標の全リストは[Datascape 指標用語集](https://help.adjust.com/en/article/datascape-metrics-glossary)をご覧ください。[フィルターデータのエンドポイント](/en/api/rs-api/filters-data)を使用して、KPIを検索することができます。また、[イベントエンドポイント](/en/api/rs-api/events)を使って、機械生成したslugを返し、CSVレポートのイベントに使用することも可能です。

3\. レスポンスの形式を選択する \{\#choose\-your\-response\-format\}
-------------------------------------------------------------

レポートサービスAPIには、3つの異なるレポート形式のためのエンドポイントが含まれています。受け取りたい形式と一致するエンドポイントを選択してください。

* レポートエンドポイント：JSON形式でデータを返します。
* CSVレポートエンドポイント：データをCSV形式で返します。
* ピボットレポートエンドポイント：合計をディメンションに分類するフィールドと共にJSON形式でデータを返します。
