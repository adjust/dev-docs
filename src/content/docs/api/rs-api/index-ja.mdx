レポートサービス（Report Services）APIにより、Adjustのレポートデータセットにクエリを実行して、集計データを取得することができます。これらのエンドポイントを使って、Adjustから簡単にデータを取得できます。

1\. コールを認証する
-------------

このレポートサービスAPIエンドポイントを使用するには、Adjust APIトークンが必要です。このトークンをAPIで使用するには、[認証ガイド](/en/api/rs-api/authentication)の手順に従ってください。

2\. 指標を見つける
------------

Adjustは、レポートのフィルタリングに使用できる多数の指標をサポートしています。これらの指標の多くは、アプリごとまたはイベントごとに存在します。

指標の全リストは、 [Datascape指標用語集](https://help.adjust.com/en/article/datascape-metrics-glossary)をご覧ください。 また、フィルターデータのエンドポイントを使ってKPIを検索することもできます。 さらに、イベントエンドポイントで機械生成したslugを返し、CSVレポートのイベントに使用することも可能です。

3\. レスポンスのフォーマットを選択する
----------------------

レポートサービスAPIには、3つの異なるレポート形式のためのエンドポイントが含まれています。受け取りたい形式と一致するエンドポイントを選択してください。

* レポートエンドポイント：JSON形式でデータを返します。
* CSVレポートエンドポイント：データをCSV形式で返します。
* ピボットレポートエンドポイント：合計をディメンションに分類するフィールドと共にJSON形式でデータを返します。
