리포트 서비스 API를 사용하면 Adjust의 리포팅 데이터 세트를 쿼리하여 집계된 데이터를 검색할 수 있습니다. 이러한 엔드포인트를 통해 Adjust 데이터의 간편하게 필터링, 분류 및 검색이 가능합니다.

[1\. 콜 인증](1-authenticate-your-calls)
--------------------------------------

리포트 서비스 API 엔드포인트를 사용하려면 Adjust API 토큰이 필요합니다. [인증 가이드](/en/api/rs-api/authentication) 내 설명을 참조하여 해당 토큰을 API와 사용하는 방법을 확인하시기 바랍니다.

[2\. 지표 찾기](2-find-your-metrics)
---------------------------------

Adjust는 리포트 필터링에 사용할 수 있는 다양한 지표를 지원합니다. 대부분의 지표는 앱 또는 이벤트별로 존재합니다.

전체 지표 목록은 [Datascape 지표 용어집](https://help.adjust.com/en/article/datascape-metrics-glossary)을 참고하시기 바랍니다. [필터 데이터 엔드포인트](/en/api/rs-api/filters-data) 를 사용하여 지표를 검색하고 [이벤트 엔드포인트](/en/api/rs-api/events) 를 사용하여 CSV 리포트에서 사용하기 위해 머신 생성 이벤트 슬러그를 반환할 수 있습니다.

[3\. 응답 형식 선택하기](3-choose-your-response-format)
------------------------------------------------

리포트 서비스 API는 3개의 리포트 유형에 대한 엔드포인트를 포함합니다. 수신하고자 하는 형식과 일치하는 엔드포인트를 선택하시기 바랍니다.

* 리포트 엔드포인: JSON으로 데이터 반환.
* CSV 리포트 엔드포인트: CSV로 데이터 반환.
* 피봇 리포트 엔드포인트: JSON으로 데이터 반환. 총합을 항목에 따라 분류한 필드가 포함되어 있음.
