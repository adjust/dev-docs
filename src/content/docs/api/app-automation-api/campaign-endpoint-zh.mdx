使用推广活动终端，为您的应用程序创建跟踪链接。

```http
POST https://settings.adjust.com/api/campaign
```

## 创建新的跟踪链接

使用该终端为您的应用创建新跟踪链接。如果您的合作伙伴需要设置，请务必在请求中包含该信息。

### 参数

<Table>

| 名称                     | 数据类型 | 位置 | 描述                                                                                                        |
| ------------------------ | -------- | ---- | ----------------------------------------------------------------------------------------------------------- |
| `store_id`\*             | `String` | 正文 | Bundle ID \(Android\) \| Store ID \(iOS\)                                                                   |
| `platform`\*             | `String` | 正文 | `android` 或 `ios`。                                                                                        |
| `channel`\*              | `String` | 正文 | 请查看下方[广告渠道](#ad-networks)。                                                                        |
| `campaign_name`          | `String` | 正文 | 已弃用。出于现有 API 兼容性考虑，未从列表中移除。                                                           |
| `channel_setup`          | `Object` | 正文 | 合作伙伴特定参数。请参阅[渠道源 \(channel\) 设置](/zh/api/app-automation-api/channel-setup)，了解更多信息。 |
| `android_signature_hash` | `String` | 正文 | 从 keystore 文件中生成的 SHA1 哈希，用来创建 SDK 签名。                                                     |

</Table> <Accordion title="广告渠道">

-  `apple_search_ads`
-  `applovin`
-  `chartboost`
-  `crossinstall`
-  `crosspromotion`
-  `digitalturbine`
-  `facebook`
-  `gameanalytics`
-  `google`
-  `ironsource`
-  `snapchat`
-  `tapjoy`
-  `unityads`
-  `vungle`
-  `inmobi`

</Accordion>

### 响应参数

<Table>

| 名称               | 数据类型 | 描述                                             |
| ------------------ | -------- | ------------------------------------------------ |
| `click_url`\*      | `String` | 特定应用和途径的 URL，用作新推广活动的点击 URL。 |
| `impression_url`\* | `String` | 特定应用和途径的 URL，用作新推广活动的展示 URL。 |

</Table>

```json
{
   "click_url": "String",
   "impression_url": "String"
}
```

## 示例

```console
$ curl \
--header "AdjustAuthorization: Token <adjust api token id="sl-md0000000">" \
--header "Content-Type: application/json" \
--data '{"store_id": "test.bundle.app", "platform": "ios", "channel": "unityads"}' \
-X POST https://settings.adjust.com/api/campaign
```

```json
{
   "click_url": "https://app.adjust.com/1lwd85",
   "impression_url": "https://view.adjust.com/impression/1lwd85"
}
```
