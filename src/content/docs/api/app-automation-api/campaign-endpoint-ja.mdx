キャンペーンエンドポイントを使用して、アプリのトラッカーURLを作成します。

```http
POST https://settings.adjust.com/api/campaign
```

新規トラッカーURLを作成する
---------------

このエンドポイントを使用して、アプリの新規トラッカーURLを作成します。設定が必要なパートナーと連携している場合、この情報を含める必要があります。

### パラメータ

<table> 

|            名前            | データタイプ | 入力する値 |                                         説明                                         |
|--------------------------|--------|-------|------------------------------------------------------------------------------------|
| `store_id`\*            | 文字列    | ボディ   | AndroidのバンドルID \| iOSのストアID。                                                       |
| `platform`\*            | 文字列    | ボディ   | `android` または`ios`。                                                                |
| `channel`\*             | 文字列    | ボディ   | 以下の[アドネットワーク](#ad-networks)を参照してください。                                              |
| `campaign_name`          | 文字列    | ボディ   | 廃止済み。既存のAPIとの互換性の理由から残されています。                                                      |
| `channel_setup`          | オブジェクト | ボディ   | パートナー固有のパラメーター。詳細は[チャネル設定に関する記事](/en/api/app-automation-api/channel-setup)をご覧ください。 |
| `android_signature_hash` | 文字列    | ボディ   | SHA1 ハッシュ値。SDKシグネチャーをビルドするため生成されるキーストアファイル。                                        |

</Table> <accordion title="アドネットワーク"> 

* `apple_search_ads`
* `applovin`
* `chartboost`
* `crossinstall`
* `crosspromotion`
* `digitalturbine`
* `facebook`
* `gameanalytics`
* `google`
* `ironsource`
* `snapchat`
* `tapjoy`
* `unityads`
* `vungle`
* `inmobi`

</Accordion> 

### レスポンスパラメーター

<table> 

|         名前          | データタイプ |                       説明                        |
|---------------------|--------|-------------------------------------------------|
| `click_url`\*      | 文字列    | 新規キャンペーンのクリックURLとして使用する、アプリ固有およびチャネル固有のURL。     |
| `impression_url`\* | 文字列    | 新規キャンペーンのインプレッションURLとして使用する、アプリ固有およびチャネル固有のURL。 |

</Table> 

```json
{
   "click_url": "String",
   "impression_url": "String"
}
```

例
---

```console
$ curl \
--header "AdjustAuthorization: Token <adjust api token id="sl-md0000000">" \
--header "Content-Type: application/json" \
--data '{"store_id": "test.bundle.app", "platform": "ios", "channel": "unityads"}' \
-X POST https://settings.adjust.com/api/campaign
```

```json
{
   "click_url": "https://app.adjust.com/1lwd85",
   "impression_url": "https://view.adjust.com/impression/1lwd85"
}
```
