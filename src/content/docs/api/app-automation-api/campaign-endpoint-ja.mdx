Use the Campaign endpoint to create links for your application.
<codeblock title="エンドポイント" highlight="/campaign"> 

```http
POST https://settings.adjust.com/api/campaign
```

</CodeBlock> 

[Create new links](create-new-links)
------------------------------------

Use this endpoint to create new links for your app. If you are working with partners that require setup, you need to include this information.

### [パラメーター](parameters)

<table> 

|            名前            |  データタイプ  | 入力する値 |                                         説明                                         |
|--------------------------|----------|-------|------------------------------------------------------------------------------------|
| `store_id`\*            | `String` | ボディ   | AndroidのバンドルID \| iOSのストアID。                                                       |
| `platform`\*            | `String` | ボディ   | `android` または`ios`。                                                                |
| `channel`\*             | `String` | ボディ   | 以下の[アドネットワーク](#ad-networks)を参照してください。                                              |
| `campaign_name`          | `String` | ボディ   | 廃止済み。既存のAPIとの互換性の理由から残されています。                                                      |
| `channel_setup`          | `Object` | ボディ   | パートナー固有のパラメーター。詳細は[チャネル設定に関する記事](/en/api/app-automation-api/channel-setup)をご覧ください。 |
| `android_signature_hash` | `String` | ボディ   | SHA1 ハッシュ値。SDKシグネチャーをビルドするため生成されるキーストアファイル。                                        |

</Table> <accordion title="アドネットワーク"> 

* `apple_search_ads`
* `applovin`
* `chartboost`
* `crossinstall`
* `crosspromotion`
* `digitalturbine`
* `facebook`
* `gameanalytics`
* `google`
* `ironsource`
* `snapchat`
* `tapjoy`
* `unityads`
* `vungle`
* `inmobi`

</Accordion> 

### [レスポンスパラメーター](response-parameters)

<table> 

|         名前          |  データタイプ  |                       説明                        |
|---------------------|----------|-------------------------------------------------|
| `click_url`\*      | `String` | 新規キャンペーンのクリックURLとして使用する、アプリ固有およびチャネル固有のURL。     |
| `impression_url`\* | `String` | 新規キャンペーンのインプレッションURLとして使用する、アプリ固有およびチャネル固有のURL。 |

</Table> <codeblock title="レスポンスの形式"> 

```json
{
   "click_url": "String",
   "impression_url": "String"
}
```

</CodeBlock> 

[例](example)
------------

<codeblock title="cURL" highlight="store_id, platform, channel"> 

```console
$ curl \
--header "AdjustAuthorization: Token <adjust api token id="sl-md0000000">" \
--header "Content-Type: application/json" \
--data '{"store_id": "test.bundle.app", "platform": "ios", "channel": "unityads"}' \
-X POST https://settings.adjust.com/api/campaign
```

</CodeBlock> <codeblock title="成功レスポンス"> 

```json
{
   "click_url": "https://app.adjust.com/1lwd85",
   "impression_url": "https://view.adjust.com/impression/1lwd85"
}
```

</CodeBlock>
