「AdjustアプリオートメーションAPI」を使用して、アプリの作成と更新を行います。このエンドポイントにより、アプリの管理とアプリの設定に伴う作業を自動化できます。

ダッシュボードでアプリを一つずつ設定する代わりに、テンプレートアプリを利用できます。テンプレートアプリには、キャンペーンに必要な全ての設定が含まれます。テンプレートアプリの作成後、AdjustアプリオートメーションAPIを使用し、このテンプレートに基づいて新しいアプリを作成できます。

新しいアプリごとに、テンプレートアプリまたは選択した別のアプリの設定が複製されます。アプリ固有の設定をパスして、テンプレートを解除または新しい情報を追加できます。

このAPIを使用すると、新しいアプリごとに1つのリクエストを作成するだけで済むため、時間を大幅に節約できます。

[ユース ケース](use-cases)
--------------------

AdjustオートメーションAPIは、新しいアプリケーションを多数作成する場合に最も役立ちます。新しいアプリやキャンペーンを頻繁にリリースする場合、新しいアプリをそれぞれ設定するには追加の作業が必要です。このAPIを使用すると、余分な作業を最小限に抑えられます。

1. テンプレートから設定を複製することで、アプリの **作成** が簡素化されます。Adjustのテクニカル アカウント スペシャリストがテンプレート設定のお手伝いをします。
2. アプリの **更新** が簡素化されます。パートナー固有およびアプリ固有の設定をAPIにパスすることで、いつでもアプリを更新できます。
3. アプリ作成の経過を追うことができます。つまり、各アプリが作成過程のどの段階にあるかを正確に把握できます。
4. アプリに関する情報をプログラムで取得できます。ダッシュボードで各アプリのリンクを確認する代わりに、このAPIを使用して情報を取得できます。

[開始方法](get-started)
-------------------

### [リクエストを認証する](authenticate-your-requests)

アプリオートメーションAPIを使用するには、コールを認証する必要があります。使用を開始するには、[認証ガイド](/en/api/app-automation-api/authentication)の手順に従ってください。

### [新規アプリを作成する](create-new-apps)

<callout type="note"> 

テクニカル アカウント スペシャリストがテンプレートアプリ設定のお手伝いをします。使用を開始するには、担当のアカウントマネージャーにお問い合わせください。
</Callout> 

[`/app`エンドポイント](/en/api/app-automation-api/app-endpoint)を使用して、アプリに関する情報を含むPOSTリクエストを送信してアプリを作成します。新しいアプリのリクエストを送信すると、Adjustによってそのエントリーが作成され、新しいアプリトークンが返されます。

[パートナー固有の設定](/en/api/app-automation-api/channel-setup)をパスして、パートナーモジュールを設定します。

Adjustによってテンプレートアプリから設定がコピーされて、新しいアプリに入力されます。

### [新しいアプリのステータスを確認する](check-the-status-of-your-new-app)

[`/status`エンドポイント](/en/api/app-automation-api/status-endpoint)にGETリクエストを送信することで、新しいアプリのステータスを確認できます。これは、テンプレートアプリから新規アプリに設定をコピーするバックグラウンドでのジョブステータスを示しています。

### [アプリに関する情報を取得する](get-information-about-your-app)

新しいアプリを作成したら、[`/app/{adjust_app_token}`](/en/api/app-automation-api/app-endpoint#app-token-parameters)エンドポイントにGETリクエストを送信して、そのアプリに関する情報を取得できます。

また、アプリのストアIDとプラットフォームを入力して[`/app`エンドポイント](/en/api/app-automation-api/app-endpoint)にGETリクエストを送信して、情報を取得することもできます。

### [アプリを更新する](update-your-app)

テンプレートアプリの設定を変更すると、新しい情報でアプリを更新できます。アプリを更新するために変更する設定を含む`force_update`リクエストを、[`/app`エンドポイント](/en/api/app-automation-api/app-endpoint)に送信します。[パートナー固有の設定](/en/api/app-automation-api/channel-setup)でアプリを更新して、パートナーモジュールを設定します。
