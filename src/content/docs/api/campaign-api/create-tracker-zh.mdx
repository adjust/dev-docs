使用此终端来为应用创建新的跟踪链接和子跟踪链接。

```http
POST https://api.adjust.com/public/v1/apps/{app_token}/trackers
```

## [创建跟踪链接](create-a-tracker)

创建新的与应用关联的跟踪链接或子跟踪链接。

### [参数](parameters)

<Table>

| 参数           | 数据类型 | 位置          | 描述                                           |
| -------------- | -------- | ------------- | ---------------------------------------------- |
| `app_token`\*  | `String` | 路径 \(Path\) | 应用的 12 位数标识符。                         |
| `name`\*       | `String` | 正文          | 跟踪链接名称。                                 |
| `parent_token` | `String` | 正文          | 父级跟踪链接名称，只有在创建子跟踪链接时需要。 |

</Table>

### [响应参数](response-parameters)

<Table search>

| 字段                | 数据类型  | 描述                                                                                                       | 示例                                        |
| ------------------- | --------- | ---------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| `name`              | `String`  | 跟踪链接名称。                                                                                             | `Adroll`                                    |
| `label`             | `String`  | 跟踪链接名称中的最后一个层级部分。                                                                         | `Adroll`                                    |
| `level`             | `Integer` | 跟踪链接层级。 <ul><li>渠道 = `1`</li><li>推广活动 = `2`</li><li>广告组 = `3`</li><li>素材 = `4`</li></ul> | `1`                                         |
| `archived`          | `Boolean` | 跟踪链接是否已被归档。                                                                                     | `true`                                      |
| `has_subtrackers`   | `Boolean` | 跟踪链接是否存在子跟踪链接。                                                                               | `true`                                      |
| `partner_id`        | `Integer` | 附加在跟踪链接上的合作伙伴 ID。                                                                            | `3`                                         |
| `cost_data_enabled` | `Boolean` | 跟踪链接是否启用了cost_data。                                                                              | `false`                                     |
| `impression_url`    | `String`  | 展示跟踪链接的 URL。                                                                                       | `https://view.adjust.com/impression/abc123` |
| `url`               | `String`  | 跟踪链接 URL。                                                                                             | `https://app.adjust.com/abc123`             |
| `click_url`         | `String`  | 点击跟踪链接的 URL。                                                                                       | `https://app.adjust.com/abc123`             |

</Table>

```json
{
   "name": "String",
   "token": "String",
   "label": "String",
   "level": 1,
   "archived": false,
   "has_subtrackers": false,
   "partner_id": 3,
   "cost_data_enabled": false,
   "url": "String",
   "click_url": "String",
   "impression_url": "String"
}
```

#### [跟踪链接层级](tracker-levels)

-  渠道 = `1`
-  推广活动 = `2`
-  广告组 = `3`
-  素材 = `4`
