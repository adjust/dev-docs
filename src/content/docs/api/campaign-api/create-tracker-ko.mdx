새 트래커 및 하위 트래커 생성을 위해 다음 엔드포인트를 사용하시기 바랍니다.

```http
POST https://api.adjust.com/public/v1/apps/{app_token}/trackers
```

트래커 생성
------

새 트래커 또는 앱에 해당하는 하위 트래커 생성.

### 파라미터

<table> 

|      파라미터      |  데이터 유형  |  In  |                           설명                            |
|----------------|----------|------|---------------------------------------------------------|
| `app_token`\* | `String` | Path | 앱의 12자리 ID                                              |
| `name`\*      | `String` | Body | 새 트래커의 이름                                               |
| `parent_token` | `String` | Body | 상위 트래커\(parent tracker\)의 이름 하위 트래커를 생성하는 경우에만 필요합니다. |

</Table> 

### 응답 파라미터

<table search> 

|         필드          |  데이터 유형   |                                                      설명                                                      |                     예시                      |
|---------------------|-----------|--------------------------------------------------------------------------------------------------------------|---------------------------------------------|
| `name`              | `String`  | 트래커 이름                                                                                                       | `Adroll`                                    |
| `label`             | `String`  | 트래커 이름의 마지막 레벨 부분                                                                                            | `Adroll`                                    |
| `level`             | `Integer` | 트래커의 레벨. 트래커의 레벨.<br/><ul><li>네트워크 = `1`</li><li>캠페인 = `2`</li><li>광고그룹 = `3`</li><li>크리에이티브 = `4`</li></ul> | `1`                                         |
| `archived`          | `Boolean` | 트래커의 아카이브 여부.                                                                                                | `true`                                      |
| `has_subtrackers`   | `Boolean` | 트래커의 하위 트래커 포함 여부.                                                                                           | `true`                                      |
| `partner_id`        | `Integer` | 트래커에 포함될 파트너 ID.                                                                                             | `3`                                         |
| `cost_data_enabled` | `Boolean` | 트래커의 cost\_data 활성화 여부.                                                                                     | `false`                                     |
| `impression_url`    | `String`  | 노출 트래킹에 대한 트래커 URL.                                                                                          | `https://view.adjust.com/impression/abc123` |
| `url`               | `String`  | 트래킹에 대한 트래커 URL.                                                                                             | `https://app.adjust.com/abc123`             |
| `click_url`         | `String`  | 클릭 트래킹에 대한 트래커 URL.                                                                                          | `https://app.adjust.com/abc123`             |

</Table> 

```json
{
   "name": "String",
   "token": "String",
   "label": "String",
   "level": 1,
   "archived": false,
   "has_subtrackers": false,
   "partner_id": 3,
   "cost_data_enabled": false,
   "url": "String",
   "click_url": "String",
   "impression_url": "String"
}
```

#### 트래커 레벨

* 네트워크 = `1`
* 캠페인 = `2`
* 광고그룹 = `3`
* 크리에이티브 = `4`
