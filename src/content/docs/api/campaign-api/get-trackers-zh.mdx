使用此终端，为您指定的应用获得渠道层跟踪链接。
<apiversion version="v2"> 

```http
GET https://api.adjust.com/public/v2/apps/{app_token}/trackers
```

</ApiVersion> <apiversion version="v1"> 

```http
GET https://api.adjust.com/public/v1/apps/{app_token}/trackers
```

</ApiVersion> 

调取跟踪链接
------

返回与应用关联的渠道层级跟踪链接列表。
<apiversion version="v2"> <table> 

|       参数       | 数据类型 |      位置       |         描述          |
|----------------|------|---------------|---------------------|
| `app_token`\* | 字符串  | 路径 \(Path\) | 应用的 12 位数标识符。       |
| `cursor`       | 字符串  | 查询            | 包含结果集合的唯一游标位置。      |
| `limit`        | 整数   | 查询            | 响应中包含的最大元素数，须为正整数值。 |

</Table> </ApiVersion> <apiversion version="v1"> <table> 

|       参数       | 数据类型 |      位置       |         描述          |
|----------------|------|---------------|---------------------|
| `app_token`\* | 字符串  | 路径 \(Path\) | 应用的 12 位数标识符。       |
| `after`        | 字符串  | 查询            | 从下一页获取元素的游标。        |
| `before`       | 字符串  | 查询            | 从上一页获取元素的游标。        |
| `limit`        | 整数   | 查询            | 响应中包含的最大元素数，须为正整数值。 |

</Table> </ApiVersion> 

### 分页

来自此终端的结果会通过游标进行分页。每页返回前一页和后一页的位置。您可以将这些信息包含在调用中来返回指定页。您还可以限定每页包含的项目数量。默认上限为 `50`。

每个结果页都包含一个"paging"对象，其中包含游标位置。您可以使用这些游标位置来在不同页间程序化移动。例如，第一页可能包含如下结果：
<apiversion version="v2"> 

```json
"paging": {
   "next": "https://api.adjust.com/public/v2/apps/yxs12pfewq/trackers?cursor=g2wAAAACYhW1_gxkAANuaWxq&limit=50",
   "cursor": "g2wAAAACYhW1_gxkAANuaWxq"
}
```

要前往第二页，请使用`next`链接来返回来自下一个指定`cursor`位置的结果。

```console
$ curl "https://api.adjust.com/public/v2/apps/yxs12pfewq/trackers?cursor=g2wAAAACYhW1_gxkAANuaWxq&limit=50"
```

</ApiVersion> <apiversion version="v1"> 

```json
"paging": {
   "page_size": "50",
   "collection_size": "49",
   "total": "199",
   "cursors": {
      "after": "g2wAAAACYhW1_gxkAANuaWxq",
      "before": null
   },
   "next": "https://api.adjust.com/public/v1/apps/yxs12pfewq/trackers?after=g2wAAAACYhW1_gxkAANuaWxq&limit=50",
   "previous": null
}
```

要访问第二页，在下一个请求中的`after`参数中添加`after`值。例如：

```console
$ curl "https://api.adjust.com/public/v1/apps/yxs12pfewq/trackers?after=g2wAAAACYhW1_gxkAANuaWxq"
```

</ApiVersion> 

### 响应参数

<table search> 

|         字段          | 数据类型 |                                            描述                                            |                     示例                      |
|---------------------|------|------------------------------------------------------------------------------------------|---------------------------------------------|
| `name`              | 字符串  | 跟踪链接名称。                                                                                  | `Adroll`                                    |
| `label`             | 字符串  | 跟踪链接名称中的最后一个层级部分。                                                                        | `Adroll`                                    |
| `level`             | 整数   | 跟踪链接层级。 <ul><li>渠道 = `1`</li><li>推广活动 = `2`</li><li>广告组 = `3`</li><li>素材 = `4`</li></ul> | `1`                                         |
| `archived`          | 布尔   | 跟踪链接是否已被归档。                                                                              | `true`                                      |
| `has_subtrackers`   | 布尔   | 跟踪链接是否存在子跟踪链接。                                                                           | `true`                                      |
| `partner_id`        | 整数   | 附加在跟踪链接上的合作伙伴 ID。                                                                        | `3`                                         |
| `cost_data_enabled` | 布尔   | 跟踪链接是否启用了cost\_data。                                                                    | `false`                                     |
| `impression_url`    | 字符串  | 展示跟踪链接的 URL。                                                                             | `https://view.adjust.com/impression/abc123` |
| `url`               | 字符串  | 跟踪链接 URL。                                                                                | `https://app.adjust.com/abc123`             |
| `click_url`         | 字符串  | 点击跟踪链接的 URL。                                                                             | `https://app.adjust.com/abc123`             |

</Table> 

```json
{
   "name": "String",
   "token": "String",
   "label": "String",
   "level": 1,
   "archived": false,
   "has_subtrackers": false,
   "partner_id": 3,
   "cost_data_enabled": false,
   "url": "String",
   "click_url": "String",
   "impression_url": "String"
}
```
