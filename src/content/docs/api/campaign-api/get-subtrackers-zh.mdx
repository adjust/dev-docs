使用此终端，为指定的应用获得子跟踪链接。

<ApiVersion version="v2">

```http
GET https://api.adjust.com/public/v2/apps/{app_token}/trackers/{tracker_token}/children
```

</ApiVersion>

<ApiVersion version="v1">

```http
GET https://api.adjust.com/public/v1/apps/{app_token}/trackers/{tracker_token}/children
```

</ApiVersion>

## 调取子跟踪链接

返回与应用关联的子跟踪链接列表。

<ApiVersion version="v2">

<Table>

| 参数              | 数据类型  | 位置          | 描述                                   |
| ----------------- | --------- | ------------- | -------------------------------------- |
| `app_token`\*     | `String`  | 路径 \(Path\) | 应用的 12 位数标识符。                 |
| `tracker_token`\* | `String`  | 路径 \(Path\) | 父级跟踪链接跟踪码。                   |
| `cursor`          | `String`  | 查询          | 包含结果集合的唯一游标位置。           |
| `limit`           | `Integer` | 查询          | 响应中包含的最大元素数，须为正整数值。 |

</Table>

</ApiVersion>

<ApiVersion version="v1">

<Table>

| 参数              | 数据类型  | 位置          | 描述                                   |
| ----------------- | --------- | ------------- | -------------------------------------- |
| `app_token`\*     | `String`  | 路径 \(Path\) | 应用的 12 位数标识符。                 |
| `tracker_token`\* | `String`  | 路径 \(Path\) | 父级跟踪链接跟踪码。                   |
| `after`           | `String`  | 查询          | 从下一页获取元素的游标。               |
| `before`          | `String`  | 查询          | 从上一页获取元素的游标。               |
| `limit`           | `Integer` | 查询          | 响应中包含的最大元素数，须为正整数值。 |

</Table>

</ApiVersion>

### 分页

来自此终端的结果会通过游标进行分页。每页返回前一页和后一页的位置。您可以将这些信息包含在调用中来返回指定页。您还可以限定每页包含的项目数量。默认上限为 `50`。

每个结果页都包含一个"paging"对象，其中包含游标位置。您可以使用这些游标位置来在不同页间程序化移动。例如，第一页可能包含如下结果：

<ApiVersion version="v2">

```json
"paging": {
   "next": "https://api.adjust.com/public/v2/apps/gwzpeepw8uf8/trackers/abc123/children?cursor=g2wAAAACYhW1_gxkAANuaWxq",
   "cursor": "g2wAAAACYhW1_gxkAANuaWxq"
}
```

要前往第二页，请使用`next`链接来返回来自下一个指定`cursor`位置的结果。

```console
$ curl "https://api.adjust.com/public/v2/apps/gwzpeepw8uf8/trackers/abc123/children?cursor=g2wAAAACYhW1_gxkAANuaWxq"
```

</ApiVersion>

<ApiVersion version="v1">

```json
"paging": {
   "page_size": "50",
   "collection_size": "49",
   "total": "199",
   "cursors": {
      "after": "g2wAAAACYhW1_gxkAANuaWxq",
      "before": null
   },
   "next": "https://api.adjust.com/public/v1/apps/gwzpeepw8uf8/trackers/abc123/children?after=g2wAAAACYhW1_gxkAANuaWxq&limit=50",
   "previous": null
}
```

要访问第二页，在下一个请求中的`after`参数中添加`after`值。例如：

```console
$ curl "https://api.adjust.com/public/v1/apps/gwzpeepw8uf8/trackers/abc123/children?after=g2wAAAACYhW1_gxkAANuaWxq"
```

</ApiVersion>

### 响应参数

<Table search>

| 字段                | 数据类型  | 描述                                                                                                       | 示例                                        |
| ------------------- | --------- | ---------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| `name`              | `String`  | 跟踪链接名称。                                                                                             | `Adroll`                                    |
| `label`             | `String`  | 跟踪链接名称中的最后一个层级部分。                                                                         | `Adroll`                                    |
| `level`             | `Integer` | 跟踪链接层级。 <ul><li>渠道 = `1`</li><li>推广活动 = `2`</li><li>广告组 = `3`</li><li>素材 = `4`</li></ul> | `1`                                         |
| `archived`          | `Boolean` | 跟踪链接是否已被归档。                                                                                     | `true`                                      |
| `has_subtrackers`   | `Boolean` | 跟踪链接是否存在子跟踪链接。                                                                               | `true`                                      |
| `partner_id`        | `Integer` | 附加在跟踪链接上的合作伙伴 ID。                                                                            | `3`                                         |
| `cost_data_enabled` | `Boolean` | 跟踪链接是否启用了cost_data。                                                                              | `false`                                     |
| `impression_url`    | `String`  | 展示跟踪链接的 URL。                                                                                       | `https://view.adjust.com/impression/abc123` |
| `url`               | `String`  | 跟踪链接 URL。                                                                                             | `https://app.adjust.com/abc123`             |
| `click_url`         | `String`  | 点击跟踪链接的 URL。                                                                                       | `https://app.adjust.com/abc123`             |

</Table>

```json
{
   "name": "String",
   "token": "String",
   "label": "String",
   "level": 2,
   "archived": false,
   "has_subtrackers": false,
   "partner_id": null,
   "cost_data_enabled": false,
   "url": "String",
   "click_url": "String",
   "impression_url": "String"
}
```
