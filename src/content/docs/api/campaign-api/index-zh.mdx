使用推广活动 API 获得跟踪链接、子跟踪链接和合作伙伴信息。创建新跟踪链接和使用新的合作伙伴信息更新已有跟踪链接。

工作原理
----

1. **认证您的调用** 

   要使用推广活动 API 终端，您首先需要获取 Adjust API 识别码。按照此[认证文档](/en/api/campaign-api/authentication)中的说明进行操作，了解如何在 API 中应用识别码。
2. **调取与应用关联的渠道跟踪链接列表。** 

   使用跟踪链接终端调取与应用相关的任意跟踪链接的详细信息。您可以调取[顶层跟踪链接](/en/api/campaign-api/get-trackers)列表或[子跟踪链接](/en/api/campaign-api/get-subtrackers)列表。
3. **创建新渠道跟踪链接** 

   您可以通过[发送 POST 请求至 trackers 终端](/en/api/campaign-api/create-tracker)来向应用添加新的跟踪链接。该终端支持创建顶层跟踪链接和子跟踪链接。
4. **调取合作伙伴列表。** 

   使用 [partners 终端](/en/api/campaign-api/get-partners)获取合作伙伴列表。您可以借此更新跟踪链接。
5. **使用新合作伙伴更新跟踪链接** 

   向 [trackers 终端](/en/api/campaign-api/update-tracker)发送 PATCH 请求来使用新的合作伙伴和成本数据信息更新已有跟踪链接。
