Campaign API를 사용하면 트래커, 하위 트래커, 파트너 정보를 얻을 수 있습니다. 새 트래커를 생성하고, 기존 트래커를 새 파트너 정보로 업데이트 하세요.

## 작동 방식

1. 콜 인증

캠페인 API 엔드포인트를 사용하려면 Adjust API 토큰이 필요합니다. [인증 가이드](/ko/api/campaign-api/authentication) 내 설명을 참조하여 해당 토큰을 API와 함께 사용하는 방법을 확인하시기 바랍니다.

2. 앱과 관련된 네트워크 트래커 목록 수신

   트래커 엔드포인트를 사용해 앱과 관련된 모든 트래커에 대한 상세 정보를 얻을 수 있습니다. [상위 레벨 트래커](/ko/api/campaign-api/get-trackers) 또는 [하위 트래커](/ko/api/campaign-api/get-subtrackers) 목록을 받을 수 있습니다.

3. 새 네트워크 트래커 생성

   [트래커 엔드포인트로 POST 요청](/ko/api/campaign-api/create-tracker)을 전송하여 앱에 새 트래커를 추가할 수 있습니다. 이 엔드포인트는 상위 레벨 트래커와 하위 트래커 생성을 지원합니다.

4. 파트너 목록 수신

   [파트너 엔드포인트](/ko/api/campaign-api/get-partners)를 사용하면 파트너 목록을 받을 수 있으며, 트래커 업데이트 시 활용할 수 있습니다.

5. 새 파트너로 트래커 업데이트

   [트래커 엔드포인트에 PATCH 요청](/ko/api/campaign-api/update-tracker)을 전송하여 기존 트래커를 새 파트너와 비용 데이터 정보로 업데이트할 수 있습니다.
