본 엔드포인트를 사용하여 앱의 기존 트래커 및 하위 트래커를 업데이트 하시기 바랍니다. 트래커에 새 파트너를 추가하거나 비용 데이터의 활성화 여부를 지정할 수 있습니다.

```http
PATCH https://api.adjust.com/public/v1/apps/{app_token}/trackers/{tracker_token}
```

트래커 업데이트
--------

앱에 해당하는 트래커 또는 하위 트래커를 업데이트 하시기 바랍니다.

### 파라미터

<table> 

|        파라미터         |  데이터 유형   |  In  |                               설명                               |
|---------------------|-----------|------|----------------------------------------------------------------|
| `app_token`\*      | `String`  | Path | 앱의 12자리 ID                                                     |
| `tracker_token`\*  | `String`  | Path | 업데이트 하려는 트래커의 토큰                                               |
| `partner_id`        | `Integer` | Body | 트래커에 첨부하려는 파트너의 ID. 파트너 엔드포인트를 사용하여 파트너 및 파트너 ID 목록 검색이 가능합니다. |
| `cost_data_enabled` | `Boolean` | Body | 트래커의 비용 데이터 활성화 여부.                                            |

</Table> 

### 응답 파라미터

<table search> 

|         필드          |  데이터 유형   |                           설명                            |                     예시                      |
|---------------------|-----------|---------------------------------------------------------|---------------------------------------------|
| `name`              | `String`  | 트래커 이름                                                  | `Adroll`                                    |
| `label`             | `String`  | 트래커 이름의 마지막 레벨 부분                                       | `Adroll`                                    |
| `level`             | `Integer` | 트래커의 레벨. 아래의 \[Tracker levels\] tracker\-levels\) 참조. | `1`                                         |
| `archived`          | `Boolean` | 트래커의 아카이브 여부.                                           | `true`                                      |
| `has_subtrackers`   | `Boolean` | 트래커의 하위 트래커 포함 여부.                                      | `true`                                      |
| `partner_id`        | `Integer` | 트래커에 포함될 파트너 ID.                                        | `3`                                         |
| `cost_data_enabled` | `Boolean` | 트래커의 cost\_data 활성화 여부.                                | `false`                                     |
| `impression_url`    | `String`  | 노출 트래킹에 대한 트래커 URL.                                     | `https://view.adjust.com/impression/abc123` |
| `url`               | `String`  | 트래킹에 대한 트래커 URL.                                        | `https://app.adjust.com/abc123`             |
| `click_url`         | `String`  | 클릭 트래킹에 대한 트래커 URL.                                     | `https://app.adjust.com/abc123`             |

</Table> <accordion title="트래커 레벨"> 

* 네트워크 = `1`
* 캠페인 = `2`
* 광고그룹 = `3`
* 크리에이티브 = `4`

</Accordion> 

```json
{
   "name": "String",
   "token": "String",
   "label": "String",
   "level": 1,
   "archived": false,
   "has_subtrackers": false,
   "partner_id": 3,
   "cost_data_enabled": false,
   "url": "String",
   "click_url": "String",
   "impression_url": "String"
}
```
