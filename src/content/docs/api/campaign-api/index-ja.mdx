キャンペーンAPIは、トラッカーやサブトラッカー、パートナーの情報を取得するための機能です。新規トラッカーを作成したり、既存のトラッカーを新規パートナーの情報に更新することができます。

仕組み
---

1. コールを認証する

キャンペーンAPIエンドポイントを使用するには、はじめにAdjustのAPIトークンが必要となります。このトークンをAPIに使用する方法については[認証ガイド](/en/api/campaign-api/authentication)をご覧ください。

2. アプリに関連するネットワークトラッカーの一覧を取得する

   trackersエンドポイントを使用して、アプリに関連するトラッカーについての詳細を取得します。[トップレベルのトラッカー](/en/api/campaign-api/get-trackers)もしくは[サブトラッカー](/en/api/campaign-api/get-subtrackers)の一覧が取得できます。
3. 新規のネットワークトラッカーを作成する

   [trackersエンドポイント](/en/api/campaign-api/create-tracker)にPOSTリクエストを送信することで、アプリに新しいトラッカーを追加することができます。このエンドポイントは、トップレベルのトラッカーとサブトラッカーの作成をサポートします。
4. パートナーの一覧を取得する

   [partnersエンドポイント](/en/api/campaign-api/get-partners)を使用して、パートナーの一覧を取得します。トラッカーを更新する際に使用してください。
5. トラッカーを新規パートナーに更新する

   [trackersエンドポイント](/en/api/campaign-api/update-tracker)にPATCHリクエストを送信し、既存のトラッカーを新規パートナーとコストデータの情報に更新します。
