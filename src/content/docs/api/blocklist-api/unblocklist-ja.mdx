<Callout type="important">

Adjustでは、「ブラックリスト」を「ブロックリスト」、「ホワイトリスト」を「許可リスト」にそれぞれ変更しました。現在、この更新作業をAdjustのUIとドキュメントに行なっています。Adjustではコードベースの更新も予定していますが、その期間中APIエンドポイントに「blacklist」と表示されることがありますのでご留意ください。

</Callout>

リンクをブロックリストから解除すると、エンゲージメントの計測、アトリビューション、トラッカーおよびトラッカーと関連するサブトラッカーへのコールバックを再開します。

```http
POST https://api.adjust.com/dashboard/api/trackers/{tracker}/unblacklist
```

## リンクをブロックリストから解除する

特定のリンクをブロックリストから解除します。

<Table>

| パラメーター  | データタイプ | 入力する値 | 説明                                                     |
| ------------- | ------------ | ---------- | -------------------------------------------------------- |
| `{tracker}`\* | `String`     | パス       | アルファベットと数字で構成されたAdjustトラッカートークン |

</Table>

```json
      "partner_id": "string",

      "archive": "string",

   }
}
```
