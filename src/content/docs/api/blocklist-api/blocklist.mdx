<Callout type="important">

"Blacklist" and "whitelist" have been renamed to "blocklist" and "allowlist" respectively. Currently this change applies to the UI and documentation. While Adjust works towards updating the codebase, you'll still see the term "blacklist" in the API endpoints.

</Callout>

Blocklist a link to block engagement measurement, attribution and callbacks to it and any related sublinks.

<CodeBlock title="Endpoint" highlight="{link}/blacklist">

```http
POST https://api.adjust.com/dashboard/api/trackers/{link}/blacklist
```

</CodeBlock>

## [Blocklist a link](blocklist-a-link)

Blocklist the specified link.

### [Parameters](parameters)

The following parameters MUST be placed in the query path:

`link` (`String`)
: An alphanumeric Adjust link token.
