<Callout type="important">

"블랙리스트" 및 "화이트리스트"라는 단어는 각각 "차단 리스트" 및 "허용 리스트"로 변경됩니다. 이러한 변경은 현재 UI 및 자료들에 반영됩니다. Adjust가 코드베이스를 업데이트하는 동안 API 엔드포인트에 변경 전 용어인 "블랙리스트"가 표시될 수 있습니다.

</Callout>

링크를 차단 리스트에서 해제하면 인게이지먼트 측정, 어트리뷰션, 콜백 및 관련 하위 링크가 재활성화됩니다.

<CodeBlock title="엔드포인트" highlight="{link}/unblacklist">

```http
POST https://api.adjust.com/dashboard/api/trackers/{link}/unblacklist
```

</CodeBlock>

## [링크 차단 리스트 등록 해제](unblocklist-a-link)

특정 링크를 차단 리스트에서 해제합니다.

<Table>

| 파라미터   | 데이터 유형 | In   | 설명                                    |
| ---------- | ----------- | ---- | --------------------------------------- |
| `{link}`\* | `String`    | Path | 소문자와 숫자로 조합된 Adjust 링크 토큰 |

</Table>
