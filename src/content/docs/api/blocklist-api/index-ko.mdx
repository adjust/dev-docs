<Callout type="important">

"블랙리스트" 및 "화이트리스트"라는 단어는 각각 "차단 리스트" 및 "허용 리스트"로 변경됩니다. 이러한 변경은 현재 UI 및 자료들에 반영되었습니다. Adjust가 코드베이스를 업데이트하는 동안 API 엔드포인트에 변경 전 용어인 "블랙리스트"가 표시될 수 있습니다.

</Callout>

Adjust 차단 리스트 API는 허위 또는 오래된 트래커 URL에서 발생한 트래픽을 차단합니다. 차단 리스트에 추가된 트래커 URL은 대시보드에 **BLOCKLISTED** 라고 표시됩니다.

다이나믹 및 모듈 파트너의 트래커 URL을 차단 리스트에 등록할 수 있습니다. SAN 파트너의 트래커 URL은 차단 리스트에 등록할 수 없습니다. SAN 파트너 URL의 트래픽을 차단하려면 SAN 파트너를 삭제해야 합니다.

<Callout type="seealso">

SAN 파트너 목록은 [셀프 어트리뷰션 네트워크 설정](https://help.adjust.com/ko/suite/article/self-attributing-network-san-setup) 페이지를 참고하시기 바랍니다.

</Callout>

## [1\. 콜 인증](1-authenticate-your-calls)

Adjust 차단 리스트 API를 사용하기 위해서는 Adjust API 토큰이 필요합니다. 인증 가이드 내 설명을 참조하여 해당 토큰을 API와 함께 사용하는 방법을 확인하시기 바랍니다.

## [2\. 링크 차단 리스트 등록](2-blocklist-a-link)

다음의 정보 수신을 차단하기 위해 링크를 [차단 리스트에 등록](/ko/api/blocklist-api/blocklist)하시기 바랍니다.

-  인게이지먼트 수

   -  클릭
   -  노출 수

-  파트너와 네트워크에게 전달되는 콜백
   -  모듈 콜백
   -  다이나믹 콜백

해당 링크 연관된 기존의 어트리뷰션은 연결된 상태로 유지됩니다. 어트리뷰션된 사용자의 데이터는 사용자가 다른 트래커에 다시 어트리뷰션될 때까지 해당 링크에 연결됩니다.

## [3\. 링크 차단 리스트 등록 해제](3-unblocklist-a-link)

링크 URL을 재활성화하려면 [차단 리스트에서 해제](/ko/api/blocklist-api/unblocklist)할 수 있습니다. 링크를 차단 리스트에서 해제하면 Adjust가 인게이지먼트 및 콜백 트래킹을 재개합니다.

<Callout type="note">

링크를 차단 해제한 즉시 링크가 재활성화됩니다. 차단 리스트에 포함된 기간에 링크와 연결된 어트리뷰션 데이터는 고려되지 않습니다. 링크가 활성화되었을 때 수신된 인게이지먼트만 어트리뷰션 퍼널에서 고려됩니다.

</Callout>

## [4\. 링크의 차단 리스트 등록 여부 확인](4-view-link-status)

[Campaign Lab](https://help.adjust.com/ko/article/campaign-setup) 에서 차단 리스트에 추가된 링크를 확인할 수 있습니다. 다음을 수행하시기 바랍니다.

1. Campaign Lab으로 이동합니다.
2. 파트너를 선택하면 관련 링크 목록이 표시됩니다.

차단 리스트에 있는 링크는 **차단 리스트에 등록됨** 으로 표시됩니다.

차단 리스트에 등록된 링크는 편집할 수 없으나, Campaign Lab에 표시하고 싶지 않은 경우 이를 숨길 수 있습니다. 링크 숨김에 대한 자세한 내용은 [Campaign Lab의 숨김 링크](https://help.adjust.com/ko/article/manage-ad-network-setup#hidden-links)를 참조하시기 바랍니다.
