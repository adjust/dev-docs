<callout type="important"> 

"Blacklist\(黑名单\)" 及 "whitelist\(白名单\)" 已被分别更名为 "blocklist\(屏蔽名单\)" 和"allowlist\(许可名单\)"。当前此变更适用于我们的用户界面及文档。在我们完全更新代码库前，"blacklist" 可能仍会存在于 API 终端中。
</Callout> 

通过将链接移出屏蔽名单 \(unblocklist\) 来恢复对此跟踪链接及其相关子链接的交互监测、归因和回传。

```http
POST https://api.adjust.com/dashboard/api/trackers/{tracker}/unblacklist
```

[将链接移出屏蔽名单](unblocklist-a-link)
-------------------------------

将指定链接移出屏蔽名单。
<table> 

|       参数       |   数据类型   |      位置       |         描述          |
|----------------|----------|---------------|---------------------|
| `{tracker}`\* | `String` | 路径 \(Path\) | 由字母数字组成的 Adjust 跟踪码 |

</Table>
