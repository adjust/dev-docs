파트너 광고 비용 API를 사용하여 캠페인과 관련된 광고 비용 정보를 전송할 수 있습니다. API는 측정을 위해 캠페인과 관련된 cost id가 필요합니다.

광고 비용 API를 성공적으로 설정하는 방법:

1. 비용 ID 설정하기

   **비용 ID** 는 Adjust가 클릭 또는 노출과 비용 데이터를 매칭할 수 있도록 해주는 고유한 비용 인게이지먼트 식별자입니다. 설정 관련 상세 정보는 [비용 ID 설정 문서](/en/api/ad-spend-api/cost-id)를 참조하시기 바랍니다.

2. API 연동 설정하기

   Adjust는 `/cost` 엔드포인트를 통해 광고 비용 데이터를 받습니다. POST 요청 설정에 관한 자세한 정보는 [광고 비용 엔드포인트 문서](/en/api/ad-spend-api/cost-endpoint)의 설명을 참조하시기 바랍니다.

3. Adjust와의 연동 테스트 및 최종 확인하기

   필수 파라미터를 보낼 수 있도록 서버 구성이 완료되면 [wizards@adjust.com](mailto:wizards@adjust.com)으로 연락하시기 바랍니다. Adjust가 이후 연동을 위한 테스트를 시작합니다.

   Adjust는 여러 차례의 테스트를 제공합니다. 이를 통해 연동이 라이브되기 전에 각각의 비용 유형 내 불일치를 확인하고, 문제를 파악할 수 있습니다. Adjust는 또한 Adjust API 테스트를 통해 API 연동이 올바르게 작동하는지 확인합니다.

   테스트 결과가 만족스러운 경우, Adjust는 고객사를 공식 Adjust 광고 비용 API 파트너 목록에 광고 비용 연동 파트너로 추가합니다.
