Use this endpoint to fetch a list of partners and associated data.

## Endpoint

```http "/partners"
GET https://api.adjust.com/public/v1/partners
```

### GET request

Fetch a list of partners and associated data. Data are returned in an array of partner objects.

#### Parameters

<ListTable>

-  -  Parameter
   -  Data type
   -  In
   -  Description
-  -  `after`
   -  String
   -  Query
   -  Cursor to get elements from the next page.
-  -  `before`
   -  String
   -  Query
   -  Cursor to get elements from the previous page.
-  -  `limit`
   -  Integer
   -  Query
   -  The maximum number of elements included in the response. Needs to be a positive integer value.

</ListTable>

<Pagination />

```json
{
   "id": 1,
   "display_name": "String",
   "supports_cost_data": true
}
```

#### Response parameters

<ListTable>

-  -  Field
   -  Data type
   -  Description
   -  Example
-  -  `id`
   -  Integer
   -  The partner's ID.
   -  `174`
-  -  `display_name`
   -  String
   -  The partner's name.
   -  `Adcolony`
-  -  `support_cost_data`
   -  Boolean
   -  Whether the partner supports cost data.
   -  `true`

</ListTable>

## Example

```console
$ curl --location --request GET 'https://api.adjust.com/public/v1/partners?limit=1' \
--header 'Authorization: Token token={API_TOKEN}'
```
