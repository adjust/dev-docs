Use tabs to group information together and enable readers to switch between the content the want to see.

## Props

Use props to control the behavior of tabs

<Table>

| Prop   | Data type | Required | Description                                                                                                                                             |
| ------ | --------- | -------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `icon` | IconName  | No       | An optional icon that appears on the left of the panel. See [the list of icons in ATLAS](https://atlas.adeven.com/docs/components/Icon#available-icons) |
| `sync` | String    | No       | The sync ID of the tab. Add the same `sync` ID to all tabs you want to synchronize.                                                                     |

</Table>

## Basic tabs

Tabs must be arranged inside a `<Tabs>` container. You can create a simple set of tabs that don't synchronize by passing no props. The first `<h3>` element (`###`) is taken as the title.

```mdx
<Tabs>
<Tab>

### Tab 1

This is tab information in tab 1

</Tab>
<Tab>

### Tab 2

This is tab information in tab 2

</Tab>
</Tabs>
```

## Tab icons

You can add [Atlas icons](https://atlas.adeven.com/docs/components/Icon#available-icons) to tabs to add visual clarity.

```mdx /icon=\".*\"/
<Tabs>
<Tab icon="PlatformIos">

### iOS

This is information about iOS

</Tab>
<Tab icon="PlatformAndroid">

### Android

This is information about Android

</Tab>
</Tabs>
```

## Synchronized tabs

If you want tabs to update as the reader chooses related tabs, pass a `sync` property. When the reader selects another tab with the same sync ID, every instance of that ID will be made active.

```mdx /sync=\".*\"/
<Tabs>
<Tab icon="PlatformIos" sync="ios">

### iOS

This is information about iOS

</Tab>
<Tab icon="PlatformAndroid" sync="android">

### Android

This is information about Android

</Tab>
</Tabs>

<Tabs>
<Tab icon="PlatformIos" sync="ios">

### App Store

This is information about the App Store

</Tab>
<Tab icon="PlatformGooglePlay" sync="android">

### Play Store

This is information about the Play Store

</Tab>
</Tabs>
```
