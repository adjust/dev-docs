---
title: Delay first session
description: Use these methods delay first session to Adjust.
slug: en/sdk/ios/features/first-session-delay
---

Enable first session delay

<Tabs>
   <Tab title="Objective-C" sync="objc">
      ```objc NSString *yourAppToken = @"{YourAppToken}"; NSString *environment
      = ADJEnvironmentSandbox; ADJConfig *adjustConfig = [[ADJConfig alloc]
      initWithAppToken:yourAppToken environment:environment]; // ...
      [adjustConfig enableFirstSessionDelay]; ```
   </Tab>
</Tabs>

End first session delay

<Tabs>
   <Tab title="Objective-C" sync="objc">
      ```objc [Adjust endFirstSessionDelay]; ```
   </Tab>
</Tabs>

Coppa compliance in delay

<Tabs>
   <Tab title="Objective-C" sync="objc">
      ```objc [Adjust enableCoppaComplianceInDelay]; ```
   </Tab>
</Tabs>

<Tabs>
   <Tab title="Objective-C" sync="objc">
      ```objc [Adjust disableCoppaComplianceInDelay]; ```
   </Tab>
</Tabs>

External device id in delay

<Tabs>
<Tab title="Objective-C" sync="objc">

```objc
[Adjust setExternalDeviceIdInDelay:@"yourExternalDeviceId"];
```

</Tab>
</Tabs>
