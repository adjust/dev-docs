<Function
  lang="objc"
  name="sendFirstPackages"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L208"
>

<Callout type="important">

This method has an effect only if you delay the start of the Adjust SDK.

</Callout>

The Adjust SDK sends information to Adjust's servers as soon as it starts. If you delay the start of the SDK, you can use this method to send packages before the delay ends.

import SendFirstPackages from "@ios-fragments/Adjust/_sendFirstPackages.mdx"

<SendFirstPackages />

</slot>

<Function
  lang="objc"
  name="setOfflineMode"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L158"
>

Controls whether the SDK is in offline mode. When in offline mode, the SDK queues all events in a file until offline mode is disabled. When offline mode is disabled, the SDK sends all queued events.

<slot slot="snippet">

import SetOfflineMode from "@ios-fragments/Adjust/_setOfflineMode.mdx"

<SetOfflineMode />

</slot>

<slot slot="parameters">

`enabled` (BOOL)
: Whether the SDK is in offline mode

</slot>

</Function>
