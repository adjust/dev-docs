Use these methods to send information about your app to Adjust.

<Function
  lang="objc"
  name="trackEvent"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L99"
>

Record event information using an `ADJEvent` object and an [Adjust event token](https://help.adjust.com/en/article/basic-event-setup#create-an-event-token).

<slot slot="snippet">

import TrackEvent from "@ios-fragments/Adjust/trackEvent.mdx"

<TrackEvent />

</slot>

<slot slot="parameters">

`event` (ADJEvent)
: An event object containing the information you want to send to Adjust.

</slot>

</Function>
