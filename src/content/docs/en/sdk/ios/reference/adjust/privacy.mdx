Use these methods to handle user privacy in your app.

<Function
  lang="objc"
  name="disableThirdPartySharing"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L274"
>

Disable sharing of information with third parties for all users.

<slot slot="snippet">

import DisableThirdPartySharing from "@ios-fragments/Adjust/disableThirdPartySharing.mdx"

<DisableThirdPartySharing />

</slot>

</Function>

<Function
  lang="objc"
  name="trackThirdPartySharing"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L281"
>

Disable or enable sharing of information with third parties on a per-user basis. Accepts a third party sharing object initialized with a **boolean** value.

<slot slot="snippet">

import TrackThirdPartySharing from "@ios-fragments/Adjust/trackThirdPartySharing.mdx"

<TrackThirdPartySharing />

</slot>

<slot slot="parameters">

`thirdPartySharing` (ADJThirdPartySharing)
: The third party sharing object

</slot>

</Function>

<Function
  lang="objc"
  name="trackMeasurementConsent"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L288"
>

Set whether [data collection and retention rules](https://help.adjust.com/en/article/manage-data-collection-and-retention) apply to a user.

<slot slot="snippet">

import TrackMeasurementConsent from "@ios-fragments/Adjust/trackMeasurementConsent.mdx"

<TrackMeasurementConsent />

</slot>

<slot slot="parameters">

`enabled` (BOOL)
: Whether data collection and retention rules apply for the user.

</slot>

</Function>

<Function
  lang="objc"
  name="gdprForgetMe"
  link="https://github.com/adjust/ios_sdk/blob/20c2666082c372f9f2c0adf871d8cef195d011f7/Adjust/Adjust.h#L261"
>

Send an <Abbr>RTBF (Right To Be Forgotten)</Abbr> request to Adjust's servers. This erases all information about the user the source app. The SDK stops sending requests for the user.

<slot slot="snippet">

import GdprForgetMe from "@ios-fragments/Adjust/gdprForgetMe.mdx"

<GdprForgetMe />

</slot>

</Function>
