---
title: SDK Signature library integration
description: Add the Signature library to your Android app to secure communications between the Adjust SDK and Adjust's servers.
slug: en/sdk/android/integrations/signature-library
---

Adjust SDK v5 supports [SDK signature verification](https://help.adjust.com/en/article/sdk-signature) natively. If you previously integrated the SDK signature library, you should uninstall it and remove all related code.
