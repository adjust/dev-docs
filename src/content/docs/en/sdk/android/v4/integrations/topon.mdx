<MinorVersion added="v4.36.0">

If you want to record your ad revenue with the TopOn SDK, you can use the SDK-to-SDK integration to pass this information to Adjust's servers.

</MinorVersion>

<Callout type="note">

If you have any questions about ad revenue measurement with TopOn, contact your dedicated Technical Account Manager or send an email to support@adjust.com.

</Callout>

## [Before you begin](before-you-begin)

Follow the [integration guide](/en/sdk/android) to integrate the Adjust SDK in your app.

## [Example](example)

To record ad revenue information with Adjust, create an `AdjustAdRevenue` instance inside your `handleAdjustRevenueReport` function. See the [Send ad revenue information guide](/en/sdk/android/features/ad-revenue) for more information.

```java
public void handleAdjustRevenueReport(ATAdInfo atAdInfo) {
    AdjustAdRevenue adjustAdRevenue = new AdjustAdRevenue( AdjustConfig.AD_REVENUE_TOPON);
    adjustAdRevenue.setRevenue(atAdInfo.getPublisherRevenue(), atAdInfo.getCurrency());
    adjustAdRevenue.setAdRevenueNetwork(String.valueOf(atAdInfo.getNetworkFirmId()));
    adjustAdRevenue.setAdRevenueUnit(atAdInfo.getAdsourceId());
    adjustAdRevenue.setAdRevenuePlacement(atAdInfo.getPlacementId());

    Adjust.trackAdRevenue(adjustAdRevenue);
}
```
