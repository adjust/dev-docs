Configure these properties to set up the Adjust SDK.

<Function>

## logLevel

Set the verbosity of logs you want to receive from the Adjust SDK.

<slot slot="signature">

import LogLevelSig from "@flutter-signatures/AdjustConfig/logLevel.mdx";

<LogLevelSig />

</slot>

<slot slot="snippet">

import LogLevel from "@flutter-examples/AdjustConfig/logLevel.mdx";

<LogLevel />

</slot>

<slot slot="parameters">

`logLevel` (AdjustLogLevel)
: The level of logging output by the SDK.

<Table height="full-height">

| Log level                 | Description                                |
| ------------------------- | ------------------------------------------ |
| `AdjustLogLevel.verbose`  | Enable all logging                         |
| `AdjustLogLevel.debug`    | Enable debug logging                       |
| `AdjustLogLevel.info`     | Only show info level logs (default option) |
| `AdjustLogLevel.warn`     | Disable info logging                       |
| `AdjustLogLevel.error`    | Disable warning level logging and below    |
| `AdjustLogLevel.assert`   | Disable error level logging and below      |
| `AdjustLogLevel.suppress` | Suppress all logging                       |

</Table>

</slot>

</Function>
