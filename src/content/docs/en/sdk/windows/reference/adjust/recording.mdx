Use these methods to send information about your app to Adjust.

## TrackEvent

Record event information using an `AdjustEvent` object and an [Adjust event token](https://help.adjust.com/en/article/add-events#add-event).

<slot slot="signature">

import TrackEventSig from "@windows-signatures/Adjust/TrackEvent.mdx";

<TrackEventSig />

</slot>

<slot slot="snippet">

import TrackEvent from "@windows-examples/Adjust/TrackEvent.mdx";

<TrackEvent />

</slot>

<slot slot="parameters">

`event` (AdjustEvent)
: An event object containing the information you want to record.

</slot>

</Function>
