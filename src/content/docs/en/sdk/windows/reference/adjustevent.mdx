This class contains information about events triggered in your application. You can send this information to Adjust's servers by passing an `AdjustEvent` object to the <Tooltip>`TrackEvent` method || <TrackEvent /></Tooltip>.

## PurchaseId

Sets a deduplication ID on your event to avoid recording duplicates. The SDK stores the last ten identifiers and skips revenue events with duplicate IDs.
