Configure these properties to set up the Adjust SDK.

<Function>

## setLogLevel

Set the verbosity of logs you want to receive from the Adjust SDK.

<slot slot="signature">

import SetLogLevelSig from "@react-native-signatures/AdjustConfig/setLogLevel.mdx";

<SetLogLevelSig />

</slot>

<slot slot="snippet">

import SetLogLevel from "@react-native-examples/AdjustConfig/setLogLevel.mdx";

<SetLogLevel />

</slot>

<slot slot="parameters">

`logLevel` (LogLevel)
: The level of logging output by the SDK.

<Table height="full-height">

| Log level                       | Description                                |
| ------------------------------- | ------------------------------------------ |
| `AdjustConfig.LogLevelVerbose`  | Enable all logging                         |
| `AdjustConfig.LogLevelDebug`    | Enable debug logging                       |
| `AdjustConfig.LogLevelInfo`     | Only show info level logs (default option) |
| `AdjustConfig.LogLevelWarn`     | Disable info logging                       |
| `AdjustConfig.LogLevelError`    | Disable warning level logging and below    |
| `AdjustConfig.LogLevelAssert`   | Disable error level logging and below      |
| `AdjustConfig.LogLevelSuppress` | Suppress all logging                       |

</Table>

</slot>

</Function>
