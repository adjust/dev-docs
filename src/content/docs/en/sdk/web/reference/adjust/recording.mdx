Use these methods to send information about your app to Adjust.

<Function>

## trackEvent

Record an event object with an [Adjust event token](https://help.adjust.com/en/article/add-events#add-event) and additional parameters.

<slot slot="signature">

import TrackEventSig from "@web-signatures/Adjust/trackEvent.mdx"

<TrackEventSig />

</slot>

<slot slot="snippet">

import TrackEvent from "@web-examples/Adjust/trackEvent.mdx"

<TrackEvent />

</slot>

<slot slot="parameters">

`params` (EventParams)
: Your Adjust event parameters.

<Accordion>

### EventParams

`eventToken` (string)
: Your Adjust event token.

`revenue` (number)
: The amount of money associated with the event.

`currency` (string)
: The [ISO 4217 code](https://www.iban.com/currency-codes) of the currency used for the transaction.

`deduplicationId` (string)
: An optional identifier to avoid measuring duplicate events.

`callbackParams` (Array\<GlobalParams \>)
: A list of key-value pairs to append to your callback URL.

`partnerParams` (Array\<GlobalParams \>)
: A list of key-value pairs to send to your external partners.

</Accordion>

<Accordion>

### GlobalParams

`key` (string)
: A unique key for the parameter.

`value` (string)
: The parameter value.

</Accordion>

</slot>

</Function>

## getAttribution

Returns the user's attribution information.

<slot slot="signature">

import GetAttributionSig from "@web-signatures/Adjust/getAttribution.mdx"

<GetAttributionSig />

</slot>

<slot slot="snippet">

import GetAttribution from "@web-examples/Adjust/getAttribution.mdx"

<GetAttribution />

</slot>

<slot slot="returns">

`attribution` (Attribution)
: The user's attribution information

<Accordion>

### Attribution

`adid` (string)
: The device's unique Adjust identifier.

`tracker_token` (string)
: The campaign token associated with the attribution.

`tracker_name` (string)
: The name of the campaign link.

`network` (string)
: The network associate with the campaign.

`campaign` (string)
: The name of the campaign associated with the attribution.

`adgroup` (string)
: The adgroup associated with the attribution.

`creative` (string)
: The creative associated with the attribution.

`click_label` (string)
: The [click label](https://help.adjust.com/en/article/user-rewards) associated with the attribution.

`state` (string)
: The current state of the attribution. Either `installed` or `reattributed`.

</Accordion>

</slot>
