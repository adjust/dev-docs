The Adjust SDK provides configurable log levels to return different amounts of information. The following log levels are available:

<Table height="full-height">

| Value     | Description                                  |
| --------- | -------------------------------------------- |
| `verbose` | Enable full logging                          |
| `info`    | Return information, warnings, and error logs |
| `warning` | Return warnings and error logs               |
| `error`   | Return only error logs                       |
| `none`    | Disable all logging                          |

</Table>

You can set the log level by specifying an `logLevel` argument in the <Tooltip>`initSdk` method || <InitSdk /></Tooltip>. The SDK defaults to `error` if no value is passed.

```js {4}
Adjust.initSdk({
   appToken: "YOUR_APP_TOKEN",
   environment: "sandbox",
   logLevel: "verbose",
});
```

## Set log output

You can delegate a log output location in your web app to show logs directly on the screen. To do this, specify an HTML selector in the `logOutput` argument in the <Tooltip>`initSdk` method || <InitSdk /></Tooltip>. The SDK logs will print to this container.

```js {4}
Adjust.initSdk({
   appToken: "YOUR_APP_TOKEN",
   environment: "sandbox",
   logOutput: "#logDiv",
});
```
