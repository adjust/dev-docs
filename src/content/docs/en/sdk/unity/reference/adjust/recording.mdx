Use these methods to send information about your app to Adjust.

<Function>

## trackEvent

Record event information using an `Adjust` object and an [Adjust event token](https://help.adjust.com/en/article/basic-event-setup#create-an-event-token).

<slot slot="signature">

import TrackEventSig from "@unity-signatures/Adjust/trackEvent.mdx"

<TrackEventSig />

</slot>

<slot slot="snippet">

import TrackEvent from "@unity-examples/Adjust/trackEvent.mdx"

<TrackEvent />

</slot>

<slot slot="parameters">

`event` (AdjustEvent)
: An event object containing the information you want to send to Adjust.

</slot>

</Function>
