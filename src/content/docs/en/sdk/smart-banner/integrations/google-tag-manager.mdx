You can integrate the Adjust Smart Banner SDK into Google Tag Manager to distribute smart banners via GTM containers. Follow the steps in this guide to set this up.

## Integrate the SDK in Google Tag Manager {#integrate-the-sdk-in-google-tag-manager}

1. Navigate to Google Tag Manager.
2. Select **Tags -> New**.
3. Select **Tag Configuration**, then select **Custom HTML**.
4. Add the [SDK loading snippet](/en/sdk/smart-banner#using-a-cdn) in the **HTML** field and ensure the **Firing Triggers** field is set to **Initialization - All Pages**.

   ![A screenshot of the GTM tag configuration screen](@images/smart-banner-sdk/gtm/1-gtm-configuration.png)

5. Add another Custom HTML tag and enter your SDK initialization call between the `<script></script>` tags. This call must include an `appToken`. Check out the [SDK initialization options](/en/sdk/smart-banner/init-options) to configure the SDK using the `init` method.

   ```js
   window.AdjustSmartBanner.init({ appToken: "TOKEN" });
   ```

   ![A screenshot of the custom HTML editor with init options](@images/smart-banner-sdk/gtm/2-init-options.png)

6. Create a new trigger by selecting the plus icon (`+`) in the top right corner of the screen.
7. Navigate to **Trigger Configuration** and select **Page View** to load the script on each page view. You can configure whether to trigger the initialization on every page or enter a specific hostname.

   ![A screenshot of the firing trigger configuration](@images/smart-banner-sdk/gtm/3-trigger-config.png)

8. Save the trigger configuration and navigate back to **Tag Configuration**.
9. Verify that everything is configured correctly, then publish your GTM container.
