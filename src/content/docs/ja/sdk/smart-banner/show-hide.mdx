初期化後にスマートバナーを非表示にし、必要に応じて再度表示することができます。これを行うには、`AdjustSmartBanner.hide`メソッドを呼び出します。

<Callout type="note">

スマートバナーを非表示にしても、DOMからは削除されません。

</Callout>

```js
AdjustSmartBanner.hide();
```

スマートバナーを再度表示するには、`AdjustSmartBanner.show`メソッドを呼び出します。

<Callout type="important">

WebアプリがSPAシングルページアプリケーションの場合、ナビゲーションが発生し、ページURLが変更された後にこのメソッドを呼び出す必要があります。これにより、SDKはページの更新されたURLを読み取るようになり、現在のページに適したバナーを見つけたり、ダイナミックディープリンクでリンクを作成する際に更新されたGETパラメーターを使用したりできるようになります。

</Callout>

```js
AdjustSmartBanner.show();
```
