Adjust SDKには、デバイス情報を返すヘルパーメソッドが含まれています。これらのメソッドを使用することでコールバックに詳細を追加し、レポートを改善させましょう。

## AdjustのデバイスID {#adjust-device-identifier}

<CodeBlock title="メソッドシグネチャー">

```cs
public static string GetAdid();
```

</CodeBlock>

Adjustは各デバイスに固有のAdjust Device ID（ADID）を生成します。`GetAdid`メソッドを呼び出すと、このIDを`string`として返します。

```cs
string adid = Adjust.GetAdid();
```

## Windowsの広告ID {#windows-advertising-identifier}

<CodeBlock title="メソッドシグネチャー">

```cs
public static string GetWindowsAdId();
```

</CodeBlock>

Windows Advertising ID（Windows ADID）は、Windows端末の端末固有の識別子です。`GetWindowsAdId`メソッドを呼び出すと、このIDを`string`として返します。

```cs
var windowsAdid = Adjust.GetWindowsAdId();
```
