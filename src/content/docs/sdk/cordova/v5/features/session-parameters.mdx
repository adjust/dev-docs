If you [register a callback URL](https://help.adjust.com/en/article/recommended-placeholders-callbacks) in the Adjust dashboard, Adjust sends a GET request to your callback URL when the SDK measures a session.

## Global callback parameters {#global-callback-parameters}

<Callout type="note">

The Adjust SDK merges global callback parameters with event callback parameters. Event callback parameters take priority over global callback parameters. This means that if you add a parameter key to both an event and a global callback, the SDK sends the event parameter.

</Callout>

You can configure callback parameters to your servers. Once you configure global parameters, the SDK appends them to your callback URL. You can use this information to analyze your users' in-app behavior with your BI system.

### Add global callback parameters {#add-global-callback-parameters}

<CodeBlock title="Method signature">

```ts
addGlobalCallbackParameter(key: string, value: string): void
```

</CodeBlock>

Add callback parameters to your event by calling the `addGlobalCallbackParameter` method with `string` key-value arguments. You can add multiple parameters by calling this method multiple times.

```js
Adjust.addGlobalCallbackParameter("key", "value");
```

### Remove global callback parameters {#remove-global-callback-parameters}

<CodeBlock title="Method signature">

```ts
removeGlobalCallbackParameter(key: string): void
```

</CodeBlock>

You can remove specific global callback parameters if they're no longer required. To do this, pass the parameter `key` to the `removeGlobalCallbackParameter` method.

```js
Adjust.removeGlobalCallbackParameter("key");
```

### Reset global callback parameters {#reset-global-callback-parameters}

<CodeBlock title="Method signature">

```ts
removeGlobalPartnerParameter(): void
```

</CodeBlock>

You can remove all global parameters if they're no longer required. To do this, call the `removeGlobalCallbackParameters` method.

```js
Adjust.removeGlobalCallbackParameters();
```

## Global partner parameters {#global-partner-parameters}

You can send extra information to your network partners by adding [partner parameters](https://help.adjust.com/en/article/data-sharing-ad-network#map-parameters).

Adjust sends partner parameters to [external partners](https://help.adjust.com/en/article/integrated-partners) you've set up. This information is useful for more granular analysis and retargeting purposes. Adjust's servers forward these parameters once you've set them up and enabled them for a partner.

<Callout type="note">

Partner parameters don't appear in raw data by default. You can add the `{partner_parameters}` placeholder to receive them as a single string.

</Callout>

### Add global partner parameters {#add-global-partner-parameters}

<CodeBlock title="Method signature">

```ts
addGlobalPartnerParameter(key: string, value: string): void
```

</CodeBlock>

Send partner parameters with your session by calling the `addGlobalPartnerParameter` method with `string` key-value arguments. You can add multiple parameters by calling this method multiple times.

```js
Adjust.addGlobalCallbackParameter("key", "value");
```

### Remove global partner parameters {#remove-global-partner-parameters}

<CodeBlock title="Method signature">

```ts
removeGlobalPartnerParameter(key: string): void
```

</CodeBlock>

You can remove specific global partner parameters if they're no longer required. To do this, pass the parameter key to the `removeGlobalPartnerParameter` method.

```js
Adjust.removeGlobalPartnerParameter("key");
```

### Reset global partner parameters {#reset-global-partner-parameters}

<CodeBlock title="Method signature">

```ts
removeGlobalPartnerParameters(): void
```

</CodeBlock>

You can remove all global partner parameters if they're no longer required. To do this, call the `resetGlobalPartnerParameters` method.

```js

```
