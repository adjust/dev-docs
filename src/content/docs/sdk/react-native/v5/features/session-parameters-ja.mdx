Adjustダッシュボードで[コールバックURLを登録](https://help.adjust.com/ja/article/recommended-placeholders-callbacks)した場合、SDKがセッションを計測するとAdjustはコールバックURLにGETリクエストを送信します。

## グローバルコールバックパラメーター {#global-callback-parameters}

<Callout type="note">

Adjust SDKは、グローバルコールバックパラメーターをイベントコールバックパラメーターとマージします。イベントコールバックパラメーターは、グローバルコールバックパラメーターより優先されます。つまり、パラメーターキーをイベントとグローバルコールバックの両方に追加した場合、SDKはイベントパラメーターを送信します。

</Callout>

コールバックパラメーターをサーバーに設定できます。グローバルパラメーターを設定すると、SDKはそれらをコールバックURLに追加します。この情報を使用して、自社のBIシステムでユーザーのアプリ内行動を分析することが可能です。

### グローバルコールバックパラメーターを追加する {#add-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
addGlobalCallbackParameter: (key: string, value: string) => void
```

</CodeBlock>

イベントにコールバックパラメーターを追加するには、`string` key\-value引数を指定して`addGlobalCallbackParameter`メソッドを呼び出してください。このメソッドを複回呼び出すことで、複数のパラメーターを追加することができます。

```jsx
Adjust.addGlobalCallbackParameter("key", "value");
```

### グローバルコールバックパラメーターを削除する {#remove-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
removeGlobalCallbackParameter: (key: string) => void
```

</CodeBlock>

不要になった場合は、特定のセッションコールバックパラメーターを削除できます。これを行うには、グローバルパラメーター`key`を`removeGlobalCallbackParameter`メソッドにパスします。

```jsx
Adjust.removeGlobalCallbackParameter("key");
```

### グローバルコールバックパラメーターをリセットする {#reset-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
removeGlobalCallbackParameters: () => void
```

</CodeBlock>

不要になった場合は、全てのグローバルパラメーターを削除できます。これを行うには、`removeGlobalCallbackParameters`メソッドを呼び出します。

```jsx
Adjust.removeGlobalCallbackParameters();
```

## グローバルパートナーパラメーター {#global-partner-parameters}

[パートナーパラメーター](https://help.adjust.com/ja/article/data-sharing-ad-network#map-parameters)を追加することで、ネットワークパートナーに追加情報を送信することができます。

Adjustは、設定した[外部パートナー](https://help.adjust.com/ja/article/integrated-partners)にパートナーパラメーターを送信します。この情報は、より詳細な分析やリターゲティング施策にご利用いただけます。パートナー用にパラメーターの設定が完了し、機能が有効化されると、Adjustのサーバーはデータ転送を開始します。

<Callout type="note">

パートナーパラメーターはデフォルトではローデータに表示されません。ご希望の場合は、プレースホルダー`{partner_parameters}`を追加することで、それらを単一の文字列として受信することができます。

</Callout>

### グローバルパートナーパラメーターを追加する {#add-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
addGlobalPartnerParameter: (key: string, value: string) => void
```

</CodeBlock>

セッションと一緒にパートナーパラメーターを送信するには、`string` key\-value引数を指定して`addGlobalPartnerParameter`メソッドを呼び出してください。このメソッドを複回呼び出すことで、複数のパラメーターを追加することができます。

```jsx
Adjust.addGlobalCallbackParameter("key", "value");
```

### グローバルパートナーパラメーターを削除する {#remove-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
removeGlobalPartnerParameter: (key: string) => void
```

</CodeBlock>

不要になった場合は、特定のグローバルパートナーパラメーターを削除できます。これを行うには、パラメーターキーを`removeGlobalPartnerParameter`メソッドにパスします。

```jsx
Adjust.removeGlobalPartnerParameter("key");
```

### グローバルパートナーパラメーターをリセットする {#reset-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```ts
removeGlobalPartnerParameters: () => void
```

</CodeBlock>

不要になった場合は、全てのグローバルパートナーパラメーターを削除できます。これを行うには、`resetGlobalPartnerParameters`メソッドを呼び出します。

```jsx
Adjust.removeGlobalPartnerParameters();
```
