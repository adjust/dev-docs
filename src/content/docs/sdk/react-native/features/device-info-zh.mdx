Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## [Adjust 设备标识符](adjust-device-identifier)

<CodeBlock title="方法签名">

```ts
getAdid: (callback: (adid: string) => void) => void
```

</CodeBlock>

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `getAdid` 方法来通过 `string` 的形式返回此 ID。

```jsx
Adjust.getAdid((adid) => {
   console.log("ADID = " + adid);
});
```

## [广告主 ID](id-for-advertisers)

<CodeBlock title="方法签名">

```ts
getIdfa: (callback: (idfa: string) => void) => void
```

</CodeBlock>

广告主 ID \(IDFA\) 是 Apple 设备的独特标识符。调用 `getIdfa` 方法来通过 `string` 的形式返回此 ID。

```jsx
Adjust.getIdfa((idfa) => {
   console.log("IDFA = " + idfa);
});
```

## [Google Play 服务广告 ID](google-play-services-advertising-id)

<CodeBlock title="方法签名">

```ts
getGoogleAdId: (callback: (adid: string) => void) => void
```

</CodeBlock>

Google Play 服务广告 ID \(GPS ADID\) 是安卓设备的独特标识符。

用户可以通过启用设备的 "选择停用广告个性化功能" 设置来停止分享自己的 GPS ADID。当用户启用该设置后，Adjust SDK 试图读取 GPS ADID 时会返回一串零。

您可以通过在后台线程中调用 `getGoogleAdId` 方法来获取该值。指定一个委托 \(delgate\) 函数，以访问 GPS ADID 值。

```jsx
Adjust.getGoogleAdId((googleAdId) => {
   console.log("Google Ad ID = " + googleAdId);
});
```
