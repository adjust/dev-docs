<SdkVersion version="v5">

Adjust 的[短链接解决方案](https://help.adjust.com/zh/article/short-branded-links)能将复杂的长链接变成简洁的短链接。短链接会保留深度链接和推广活动信息，如果用户尚未安装您的应用，则会将用户转到应用商店。

使用本节中描述的方法来解析您的短链接。

## Set up deep linking in the Adjust SDK {#set-up-deep-linking-in-the-adjust-sdk}

在 Adjust SDK 中添加对以下内容的支持：

-  [URI 方案](/zh/sdk/react-native/features/deep-links)
-  [延迟深度链接](/zh/sdk/react-native/features/deep-links#deferred-deep-linking)

<Callout type="note">

您无需设置 LinkMe 或链接解析即可使用 Adjust 短链接解决方案。

</Callout>

## Set up the Adjust SDK to resolve short links {#set-up-the-adjust-sdk-to-resolve-short-links}

<CodeBlock title="方法签名">

```ts
processAndResolveDeeplink: (adjustDeeplink: AdjustDeeplink, callback: (resolvedLink: string) => void) => void
```

</CodeBlock>

调用 `processAndResolveDeeplink` 方法来解析将用户深度链接至应用的 Adjust 短链接 URL 。

```jsx
Adjust.processAndResolveDeeplink(new AdjustDeeplink('url'),  function(resolvedLink) {
    console.log(`Resolved link = ${resolvedLink}`)
};
```

如果您发送至`processDeeplink`方法的链接为短链接，那么`resolvedLink`就会返回延长后的原始链接。如果您传递给该方法的链接未被缩短，那么`resolvedLink`包含的链接与您传递的链接相同。

</SdkVersion>
