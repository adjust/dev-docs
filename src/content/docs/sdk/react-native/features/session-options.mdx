## [Send granular session information](send-granular-information)

<CodeBlock title="Method signature">

```ts
public addGranularOption(partnerName: string,, key: string, value: string): void
```

</CodeBlock>

You can attach granular information to send to send to a partner every time a user starts a session. This is information to communicate preferences to a partner. To do this, call the `addGranularOption` method with the following parameters:

-  `partnerName` (`string`): The name of the partner for whom the granular option applies.
-  `key` (`number`): The option key.
-  `value` (`string`): The option value.

```jsx
Adjust.addGranularOption("PartnerA", "key", "value");
```

### [Manage Facebook Limited Data Use](manage-facebook-limited-data-use

<Callout type="important">

The Adjust SDK sends infotmation to Facebook as soon as the app is installed. You need to make sure you call this method **before** initializing the SDK.

</Callout>

Facebook provides a feature called Limited Data Use (LDU) to comply with the California Consumer Privacy Act (CCPA). This feature enables you to notify Facebook when a California-based user is opted out of the sale of data. You can also use it if you want to opt all users out by default.

You can update the Facebook LDU status by passing the following properties to the `addGranularOption` method:

<Callout type="seealso" title="An optional title">

See Meta's LDU documentation [here](https://developers.facebook.com/docs/marketing-apis/data-processing-options#audience-network-sdk).

</Callout>

<Table>

| Parameter                         | Description                                                                                                                                                                                                                                      |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `partner_name`                    | Use `facebook` to toggle LDU.                                                                                                                                                                                                                    |
| `data_processing_options_country` | The country in which the user is located.<ul><li>`0`: Request that Facebook use geolocation.</li><li>`1`: United States of America.</li></ul>                                                                                                    |
| `data_processing_options_state`   | Notifies Facebook in which state the user is located.<ul><li>`0`: Request that Facebook use geolocation</li><li>`1000`: California.</li><li>`1001`: Colorado</li><li>`1002`: Connecticut</li><li>`1002`: Florida</li><li>`1004`: Texas</li></ul> |

</Table>

<Callout type="note">

If you call this method with a `0` value for **either** `data_processing_options_country` or `data_processing_options_state`, the Adjust SDK pass **both** fields back as `0`.

</Callout>

```jsx
adjustThirdPartySharing.addGranularOption(
   "facebook",
   "data_processing_options_country",
   "1",
);
Adjust.addGranularOption("facebook", "data_processing_options_state", "1105");
```
