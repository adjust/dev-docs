The following methods enable you to send event information from your app to Adjust.

## AdjustEvent class

The `AdjustEvent` class is used to hold information about an event. Each event is represented by a unique `AdjustEvent` instance.

### Constructor

Instantiates an `AdjustEvent` object.

<CodeBlock title="Method signature">

```ts
constructor(eventToken: string)
```

</CodeBlock>

<Table>

| Argument     | Type     | Description                                                                                                                       |
| ------------ | -------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `eventToken` | `string` | Your Adjust event token. See [Add events](https://help.adjust.com/en/article/add-events#manage-your-events) for more information. |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
```

### [Record event revenue](record-event-revenue)

Sets the revenue amount and currency code of any revenue associated with the event.

<CodeBlock title="Method signature">

```ts
public setRevenue(revenue: number, currency: string): void
```

</CodeBlock>

<Table>

| Argument   | Type     | Description                                                                     |
| ---------- | -------- | ------------------------------------------------------------------------------- |
| `revenue`  | `number` | The amount of revenue generated by the event.                                   |
| `currency` | `string` | The [ISO 4217 code](https://www.iban.com/currency-codes) of the event currency. |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
adjustEvent.setRevenue(0.01, "EUR");
```

### [Add callback parameters](add-callback-parameters)

Adds callback parameters. When Adjust receives your `AdjustEvent` instance, all callback parameters are sent to your callback URL.

<CodeBlock title="Method signature">

```ts
public addCallbackParameter(key: string, value: string): void
```

</CodeBlock>

<Table>

| Argument | Type     | Description                      |
| -------- | -------- | -------------------------------- |
| `key`    | `string` | The name (key) of the parameter. |
| `value`  | `string` | The value of the parameter.      |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
adjustEvent.addCallbackParameter("key", "value");
```

### [Add partner parameters](add-partner-parameters)

Adds partner parameters. When Adjust receives your `AdjustEvent` instance, all partner parameters are sent to any external partners you've configured.

<CodeBlock title="Method signature">

```ts
public addPartnerParameter(key: string, value: string): void
```

</CodeBlock>

<Table>

| Argument | Type     | Description                      |
| -------- | -------- | -------------------------------- |
| `key`    | `string` | The name (key) of the parameter. |
| `value`  | `string` | The value of the parameter.      |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
adjustEvent.addPartnerParameter("key", "value");
```

### [Set deduplication ID](deduplicate-revenue-events)

Sets a unique identifier for the `AdjustEvent` instance to deduplicate revenue events. The SDK stores the last ten identifiers and skips revenue events with duplicate transaction IDs.

<CodeBlock title="Method signature">

```ts
public setTransactionId(transactionId: string): void
```

</CodeBlock>

<Table>

| Argument        | Type     | Description              |
| --------------- | -------- | ------------------------ |
| `transactionId` | `string` | A unique transaction ID. |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
adjustEvent.setTransactionId("{transactionId}");
```

### [Set a callback identifier](add-a-callback-identifier)

Sets a unique identifier for Adjust's servers to report on in event callback.

<CodeBlock title="Method signature">

```ts
public setCallbackId(callbackId: string): void
```

</CodeBlock>

<Table>

| Argument     | Type     | Description           |
| ------------ | -------- | --------------------- |
| `callbackId` | `string` | A unique callback ID. |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
adjustEvent.setCallbackId("your_callback_id");
```

## [Send an event](send-an-event)

Use the `Adjust.trackEvent()` method to send your `AdjustEvent` instance to Adjust.

<CodeBlock title="Method signature">

```ts
trackEvent: (adjustEvent: AdjustEvent) => void
```

</CodeBlock>

<Table>

| Argument      | Type          | Description                 |
| ------------- | ------------- | --------------------------- |
| `adjustEvent` | `AdjustEvent` | Your `AdjustEvent` instance |

</Table>

```jsx
var adjustEvent = new AdjustEvent("abc123");
Adjust.trackEvent(adjustEvent);
```
