The `AdjustEvent` class contains properties which you can populate with event information to send to Adjust.

In this tutorial, you will:

1. Create a function with the `AdjustEvent` instance and populate it with the required properties. In the example below, the function is called `_onPress_trackEvent`.
2. Within this function, instantiate and populate an `AdjustEvent` object. In the example below, the following properties are set:
   -  The event revenue is set to _`0.25`_ and the currency code is set to _`EUR`_.
   -  The event deduplication ID is set to _`5e85484b-1ebc-4141-aab7-25b869e54c49`_.
   -  The event token and revenue amount are added as callback parameters.
   -  The ID of the associated product and ID of the user who triggered the event are added as partner parameters.
   -  The callback ID is set to _`f2e728d8-271b-49ab-80ea-27830a215147`_.
3. At the end of your function, send the information to Adjust by calling `Adjust.trackEvent()`.
4. Call your function in response to an action in your app. In the example below, the function is called whenever the `Send Event` button is pressed.

```jsx
function _onPress_trackEvent() {
   var adjustEvent = new AdjustEvent("g3mfiw");
   var uniqueId = "5e85484b-1ebc-4141-aab7-25b869e54c49";
   var callbackId = "f2e728d8-271b-49ab-80ea-27830a215147";
   adjustEvent.setRevenue(0.25, "EUR");
   adjustEvent.setTransactionId(uniqueId);
   adjustEvent.setCallbackId(callbackId);
   adjustEvent.addCallbackParameter("event_token", "g3mfiw");
   adjustEvent.addCallbackParameter("revenue_amount", "0.05");
   adjustEvent.addPartnerParameter("product_id", "29");
   adjustEvent.addPartnerParameter("user_id", "835");
   Adjust.trackEvent(adjustEvent);
}
<>
   <View style={styles.container}>
      <TouchableHighlight style={styles.button} onPress={_onPress_trackEvent}>
         <Text>Send Event</Text>
      </TouchableHighlight>
   </View>
</>;
```
