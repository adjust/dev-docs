The `AdjustEvent` class contains properties which you can populate with event information to send to Adjust. In this tutorial, you'll learn how to:

-  Instantiate the `AdjustEvent` class using the Adjust event token, `g3mfiw`.
-  Record event revenue by setting the `revenue` property of this event to _`0.25`_ and the `currency` property to _`EUR`_.
-  Add the revenue event deduplication ID by setting `uniqueId` to `5e85484b-1ebc-4141-aab7-25b869e54c49` using the `setTransactionId` method.
-  Add the `event_token` and `revenue_amount` callback parameters to your event.
-  Add the `product_id` (ID of the associated product) and `user_id` (ID of the user who triggered the event) partner parameters to your event.
-  Add a custom callback identifier, `callbackId` to `f2e728d8-271b-49ab-80ea-27830a215147` to your event.

### [Tutorial](tutorial)

In this tutorial, you will:

1. Create a function with the `AdjustEvent` instance and populate it with the required properties.
2. Send the information to Adjust using `Adjust.trackEvent()`.
3. Add a call to the event function to a button in your app.

```jsx
function _onPress_trackEvent() {
   var adjustEvent = new AdjustEvent("g3mfiw");
   var uniqueId = "5e85484b-1ebc-4141-aab7-25b869e54c49";
   var callbackId = "f2e728d8-271b-49ab-80ea-27830a215147";
   adjustEvent.setRevenue(0.25, "EUR");
   adjustEvent.setTransactionId(uniqueId);
   adjustEvent.setCallbackId(callbackId);
   adjustEvent.addCallbackParameter("event_token", "g3mfiw");
   adjustEvent.addCallbackParameter("revenue_amount", "0.05");
   adjustEvent.addPartnerParameter("product_id", "29");
   adjustEvent.addPartnerParameter("user_id", "835");
   Adjust.trackEvent(adjustEvent);
}
<>
   <View style={styles.container}>
      <TouchableHighlight style={styles.button} onPress={_onPress_trackEvent}>
         <Text>Send Event</Text>
      </TouchableHighlight>
   </View>
</>;
```
