## setUrlStrategy

Sets the country or region of data residency. If called with a `UrlStrategy` parameter, sets the country/region where data will be sent preferentially.

<slot slot="signature">

import SetUrlStrategySig from "@react-native-signatures/AdjustConfig/setUrlStrategy.mdx";

<SetUrlStrategySig />

</slot>

<slot slot="snippet">

import SetUrlStrategy from "@react-native-examples/AdjustConfig/setUrlStrategy.mdx";

<SetUrlStrategy />

</slot>

<slot slot="parameters">

`urlStrategy` (UrlStrategy)
: The data residency target (see available values below)

<Table>

| Value                           | Description                                               |
| ------------------------------- | --------------------------------------------------------- |
| `AdjustConfig.DataResidencyEU`  | Sets the area of data residency to the EU                 |
| `AdjustConfig.DataResidencyTR`  | Sets the area of data residency to Turkey                 |
| `AdjustConfig.DataResidencyUS`  | Sets the area of data residency to the USA                |
| `AdjustConfig.UrlStrategyChina` | Points the URL strategy to the `app.adjust.world` domain  |
| `AdjustConfig.UrlStrategyCn`    | Points the URL strategy to the `app.adjust.cn` domain     |
| `AdjustConfig.UrlStrategyIndia` | Points the URL strategy to the `app.adjust.net.in` domain |

</Table>

</slot>

</Function>
