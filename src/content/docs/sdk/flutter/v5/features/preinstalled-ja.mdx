Adjust SDKを使用して、ユーザーのデバイスにプリインストールされているアプリでのアクティビティを記録することができます。これにより、キャンペーンからアプリをダウンロードしなかったユーザーの情報を記録することが可能です。

Configオブジェクトには、この機能を制御する`bool``isPreinstallTrackingEnabled`プロパティが含まれています。プリインストール計測を有効化するには、configオブジェクトの`isPreinstallTrackingEnabled`プロパティに`bool`値を割り当てます。

```dart
AdjustConfig adjustConfig = new AdjustConfig('{YourAppToken}', AdjustEnvironment.sandbox);
//...
adjustConfig.isPreinstallTrackingEnabled = true;
//...
Adjust.initSdk(adjustConfig);
```

## デフォルトのリンクトークン {#default-link-token}

デフォルトのリンクトークンを設定すると、全てのプリインストールを定義済みのAdjustリンクにアトリビュートさせることができます。Adjustはアトリビューションソースが変更されるまで、このトークンに対して全ての情報を記録します。設定方法は次のとおりです：

1. [Campaign Labで新しいリンクを作成します。](https://help.adjust.com/ja/article/links)

   ```text
   https://app.adjust.com/{token}
   ```

2. このトークンをコピーして、configオブジェクトの[`defaultTracker`プロパティ](/ja/sdk/flutter/configuration#set-default-link-token)に割り当てます。

   ```dart
   AdjustConfig adjustConfig = new AdjustConfig('{YourAppToken}', AdjustEnvironment.sandbox);
   //...
   adjustConfig.defaultTracker = '{TrackerToken}';
   //...
   Adjust.initSdk(adjustConfig);
   ```

3. アプリをビルドして実行します。ログを有効にしている場合は、ログにメッセージが表示されます。

   ```text
   Default tracker: 'abc123'.
   ```
