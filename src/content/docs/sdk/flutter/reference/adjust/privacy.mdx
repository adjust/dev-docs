Use these methods to handle user privacy in your app.

<Function>

## disableThirdPartySharing

Disable sharing of information with third parties for all users.

import DisableThirdPartySharingSig from "@flutter-signatures/Adjust/disableThirdPartySharing.mdx"

<DisableThirdPartySharingSig />

</slot>

<slot slot="snippet">

import DisableThirdPartySharing from "@flutter-examples/Adjust/disableThirdPartySharing.mdx"

<DisableThirdPartySharing />

</slot>

</Function>

<Function>

## trackThirdPartySharing

Disable or enable sharing of information with third parties on a per-user basis. Accepts a third party sharing object initialized with a **boolean** value.

<slot slot="signature">

import TrackThirdPartySharingSig from "@flutter-signatures/Adjust/trackThirdPartySharing.mdx"

<TrackThirdPartySharingSig />

</slot>

<slot slot="snippet">

import TrackThirdPartySharing from "@flutter-examples/Adjust/trackThirdPartySharing.mdx"

<TrackThirdPartySharing />

</slot>

<slot slot="parameters">

`thirdPartySharing` (AdjustThirdPartySharing): The third party sharing object

</slot>

</Function>

<Function>

## trackMeasurementConsent

Set whether [data collection and retention rules](https://help.adjust.com/en/article/manage-data-collection-and-retention) apply to a user.

<slot slot="signature">

import TrackMeasurementConsentSig from "@flutter-signatures/Adjust/trackMeasurementConsent.mdx"

<TrackMeasurementConsentSig />

</slot>

<slot slot="snippet">

import TrackMeasurementConsent from "@flutter-examples/Adjust/trackMeasurementConsent.mdx"

<TrackMeasurementConsent />

</slot>

<slot slot="parameters">

`enabled` (bool): Whether data collection and retention rules apply for the user.

</slot>

</Function>

<Function>

## gdprForgetMe

Send an RTBF (Right To Be Forgotten) request to Adjust's servers. This erases all information about the user the source app. The SDK stops sending requests for the user.

<slot slot="signature">

import GdprForgetMeSig from "@flutter-signatures/Adjust/gdprForgetMe.mdx"

<GdprForgetMeSig />

</slot>

<slot slot="snippet">

import GdprForgetMe from "@flutter-examples/Adjust/gdprForgetMe.mdx"

<GdprForgetMe />

</slot>

</Function>
