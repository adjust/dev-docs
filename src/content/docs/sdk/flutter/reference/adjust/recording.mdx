## trackEvent

Record event information using an `Adjust` object and an [Adjust event token](https://help.adjust.com/en/article/add-events#add-event).

import TrackEventSig from "@flutter-signatures/Adjust/trackEvent.mdx"

<TrackEventSig />

</slot>

<slot slot="snippet">

import TrackEvent from "@flutter-examples/Adjust/trackEvent.mdx"

<TrackEvent />

</slot>

<slot slot="parameters">

`event` (AdjustEvent): An event object containing the information you want to send to Adjust.

</slot>

</Function>
