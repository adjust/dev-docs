Adjust SDKには、デバイス情報を返すヘルパーメソッドが含まれています。これらのメソッドを使用することでコールバックに詳細を追加し、レポートを改善させましょう。

## AdjustのデバイスID {#adjust-device-identifier}

<CodeBlock title="メソッドシグネチャー">

```dart
static Future<String?> getAdid() async
```

</CodeBlock>

Adjustは各デバイスに固有のAdjust Device ID（ADID）を生成します。`getAdid`メソッドを呼び出すと、このIDを`String`として返します。

```dart
Adjust.getAdid().then((adid) {
  // Use adid string value.
});
```

## 広告ID {#id-for-advertisers}

<CodeBlock title="メソッドシグネチャー">

```dart
static Future<String?> getIdfa() async
```

</CodeBlock>

ID for Advertisers（IDFA）は、Appleデバイスの端末固有の識別子です。`getIdfa`メソッドを呼び出すと、このIDを`String`として返します。

```dart
Adjust.getIdfa().then((idfa) {
  // Use idfa string value.
});
```

## ベンダーID {#id-for-vendor}

<CodeBlock title="メソッドシグネチャー">

```dart
static Future<String?> getIdfv() async
```

</CodeBlock>

ID for Advertisers（IDFA）は、Appleデバイス端末の固有の識別子です。コールバック関数で`Adjust.getIdfv`メソッドを呼び出すと、デバイスのIDFVを返すことができます。SDKは情報を非同期で取得し、完了ハンドラーにパスします。

```dart
Adjust.getIdfv().then((idfv) {
  // Use idfv string value.
});
```

## Google Play開発者サービス広告ID {#google-play-services-advertising-id}

<CodeBlock title="メソッドシグネチャー">

```dart
static Future<String?> getGoogleAdId() async
```

</CodeBlock>

Google Play 開発者サービス ID（GPS ADID）は、Androidデバイスのデバイス固有の識別子です。

ユーザーは、端末の設定「広告のパーソナライズをオプトアウトする」を切り替えることで、GPS ADIDの共有を拒否できます。ユーザーがこの設定を有効にしている場合、Adjust SDKはGPS ADIDの値を「0」の文字列で返します。

この値にアクセスするには、バックグラウンドスレッドで`getGoogleAdId`メソッドを呼び出す必要があります。GPS ADID値にアクセスするには、デリゲート関数を割り当てます。

```dart
Adjust.getGoogleAdId().then((googleAdId) {
  // Use googleAdId string value.
});
```
