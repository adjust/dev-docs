您在 Adjust 控制面板中[注册回传 URL](https://help.adjust.com/zh/article/recommended-placeholders-callbacks)，SDK 监测到会话后，Adjust 会向您的回传 URL 发送一个 GET 请求。

## 会话回传参数 {#session-callback-parameters}

<Callout type="note">

Adjust SDK 会合并会话回传参数和事件回传参数。事件回传参数优先级高于会话回传参数。这意味着如果将一个参数键同时添加到事件和会话，那么 SDK 会发送事件参数。

</Callout>

您可以为自己的服务器设置回传参数。配置好事件的参数后，SDK 会将参数附加至您的回传 URL。您可以利用该信息，通过自己的 BI 系统分析用户应用内行为。

### addGlobalCallbackParameters {#add-global-callback-parameters}

<CodeBlock title="方法签名">

```cs
public static void AddGlobalCallbackParameter(string key, string value);
```

</CodeBlock>

使用 `string` 键值对参数调用`Adjust.AddGlobalCallbackParameter`方法，以此向会话添加回传参数。多次调用该方法可添加多个参数。

```cs
Adjust.AddGlobalCallbackParameter("key", "value");
```

### 移除统一回传参数 {#remove-global-callback-parameters}

<CodeBlock title="方法签名">

```cs
public static void RemoveGlobalCallbackParameter(string key);
```

</CodeBlock>

您可以移除不再需要的特定统一回传参数。为此，请将参数 `key` 传递至 `Adjust.RemoveGlobalCallbackParameter` 方法。

```cs
Adjust.RemoveGlobalCallbackParameter("key");
```

### 重置统一回传参数 {#reset-global-callback-parameters}

<CodeBlock title="方法签名">

```cs
public static void RemoveGlobalCallbackParameters();
```

</CodeBlock>

如果您不再需要统一回传参数，则可以将所有参数一并移除。为此，请调用 `Adjust.RemoveGlobalCallbackParameters` 方法。

```cs
Adjust.RemoveGlobalCallbackParameters();
```

## 统一合作伙伴参数 {#global-partner-parameters}

您可以添加[合作伙伴参数](https://help.adjust.com/zh/article/data-sharing-ad-network#map-parameters)，向渠道合作伙伴发送额外的信息。

Adjust 可向您设置的[外部合作伙伴](https://help.adjust.com/zh/article/integrated-partners)发送合作伙伴参数。这些信息可用来进行更精细的数据分析，开展再营销活动。您设置好参数并为合作伙伴启用参数转发后，Adjust 服务器就会将这些参数转发给合作伙伴。

<Callout type="note">

合作伙伴参数不会默认出现在原始数据中。您可以添加`{partner_parameters}`占位符，在同一个字符串中接收参数。

</Callout>

### addGlobalPartnerParameters {#add-global-partner-parameters}

<CodeBlock title="方法签名">

```cs
public static void AddGlobalPartnerParameter(string key, string value);
```

</CodeBlock>

要发送带有会话信息的合作伙伴参数，请使用 `string` 键值参数调用 `Adjust.AddGlobalPartnerParameter` 方法。多次调用该方法可添加多个参数。

```cs
Adjust.AddGlobalPartnerParameter("key", "value");
```

### 移除统一合作伙伴参数 {#remove-global-partner-parameters}

<CodeBlock title="方法签名">

```cs
public static void RemoveGlobalPartnerParameter(string key);
```

</CodeBlock>

您可以移除不再需要的特定统一合作伙伴参数。为此，请将参数 key 传递至 `Adjust.RemoveGlobalPartnerParameter` 方法。

```cs
Adjust.RemoveGlobalPartnerParameter("key");
```

### 重置统一合作伙伴参数 {#reset-global-partner-parameters}

<CodeBlock title="方法签名">

```cs
public static void RemoveGlobalPartnerParameters();
```

</CodeBlock>

如果您不再需要统一合作伙伴参数，则可以将所有参数一并移除。为此，请调用 `Adjust.RemoveGlobalPartnerParameters` 方法。

```cs
Adjust.RemoveGlobalPartnerParameters();
```
