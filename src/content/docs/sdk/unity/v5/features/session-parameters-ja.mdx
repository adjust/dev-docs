Adjustダッシュボードで[コールバックURLを登録](https://help.adjust.com/ja/article/recommended-placeholders-callbacks)した場合、SDKがセッションを計測するとAdjustはコールバックURLにGETリクエストを送信します。

## セッションコールバックパラメーター {#session-callback-parameters}

<Callout type="note">

Adjust SDKは、セッションコールバックパラメーターをイベントコールバックパラメーターとマージします。イベントコールバックパラメーターは、セッションコールバックパラメーターより優先されます。つまり、パラメーターキーをイベントとセッションの両方に追加した場合、SDKはイベントパラメーターを送信します。

</Callout>

コールバックパラメーターをサーバーに設定できます。イベントにパラメーターを設定すると、SDKはそれらをコールバックURLに追加します。この情報を使用して、自社のBIシステムでユーザーのアプリ内行動を分析することが可能です。

### addGlobalCallbackParameters {#add-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void AddGlobalCallbackParameter(string key, string value);
```

</CodeBlock>

セッションにコールバックパラメーターを追加するには、`string` key\-value引数を指定して`Adjust.AddGlobalCallbackParameter`メソッドを呼び出してください。このメソッドを複回呼び出すことで、複数のパラメーターを追加することができます。

```cs
Adjust.AddGlobalCallbackParameter("key", "value");
```

### グローバルコールバックパラメーターを削除する {#remove-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void RemoveGlobalCallbackParameter(string key);
```

</CodeBlock>

不要になった場合は、特定のセッションコールバックパラメーターを削除できます。これを行うには、グローバルパラメーター`key`を`Adjust.RemoveGlobalCallbackParameter`メソッドにパスします。

```cs
Adjust.RemoveGlobalCallbackParameter("key");
```

### グローバルコールバックパラメーターをリセットする {#reset-global-callback-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void RemoveGlobalCallbackParameters();
```

</CodeBlock>

不要になった場合は、全てのグローバルコールバックパラメーターを削除できます。これを行うには、`Adjust.RemoveGlobalCallbackParameters`メソッドを呼び出します。

```cs
Adjust.RemoveGlobalCallbackParameters();
```

## グローバルパートナーパラメーター {#global-partner-parameters}

[パートナーパラメーター](https://help.adjust.com/ja/article/data-sharing-ad-network#map-parameters)を追加することで、ネットワークパートナーに追加情報を送信することができます。

Adjustは、設定した[外部パートナー](https://help.adjust.com/ja/article/integrated-partners)にパートナーパラメーターを送信します。この情報は、より詳細な分析やリターゲティング施策にご利用いただけます。パートナー用にパラメーターの設定が完了し、機能が有効化されると、Adjustのサーバーはデータ転送を開始します。

<Callout type="note">

パートナーパラメーターはデフォルトではローデータに表示されません。ご希望の場合は、プレースホルダー`{partner_parameters}`を追加することで、それらを単一の文字列として受信することができます。

</Callout>

### addGlobalPartnerParameters {#add-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void AddGlobalPartnerParameter(string key, string value);
```

</CodeBlock>

セッションと一緒にパートナーパラメーターを送信するには、`string` key\-value引数を指定して`Adjust.AddGlobalPartnerParameter`メソッドを呼び出してください。このメソッドを複回呼び出すことで、複数のパラメーターを追加することができます。

```cs
Adjust.AddGlobalPartnerParameter("key", "value");
```

### グローバルパートナーパラメーターを削除する {#remove-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void RemoveGlobalPartnerParameter(string key);
```

</CodeBlock>

不要になった場合は、特定のグローバルパートナーパラメーターを削除できます。これを行うには、パラメーターキーを`Adjust.RemoveGlobalPartnerParameter`メソッドにパスします。

```cs
Adjust.RemoveGlobalPartnerParameter("key");
```

### グローバルパートナーパラメーターをリセットする {#reset-global-partner-parameters}

<CodeBlock title="メソッドシグネチャー">

```cs
public static void RemoveGlobalPartnerParameters();
```

</CodeBlock>

不要になった場合は、全てのグローバルパートナーパラメーターを削除できます。これを行うには、`Adjust.RemoveGlobalPartnerParameters`メソッドを呼び出します。

```cs
Adjust.RemoveGlobalPartnerParameters();
```
