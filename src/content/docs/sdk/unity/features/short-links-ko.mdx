<SdkVersion version="v5">

Adjust의 [링크 단축 솔루션](https://help.adjust.com/ko/article/short-branded-links) 은 복잡하고 긴 링크를 간결하고 간결한 링크로 만들어줍니다. 단축 링크는 딥링크와 캠페인 정보를 유지하며, 사용자가 아직 앱을 설치하지 않은 경우 앱 스토어로 라우팅합니다.

이 섹션에서 설명된 방법을 사용하여 짧은 링크를 해결하시기 바랍니다.

## Adjust SDK에서 딥링크 설정 {#set-up-deep-linking-in-the-adjust-sdk}

Adjust SDK에서 다음에 대한 지원을 추가합니다.

-  [URI 스킴](/ko/sdk/unity/features/deep-links)
-  [디퍼드 딥링크](/ko/sdk/unity/features/deep-links#deferred-deep-linking)

<Callout type="note">

Adjust의 링크 단축 솔루션을 사용하기 위해 LinkMe 또는 Link resolution을 설정할 필요는 없습니다.

</Callout>

## 짧은 링크를 해석하기 위해 Adjust SDK 설정 {#set-up-the-adjust-sdk-to-resolve-short-links}

<CodeBlock title="메서드 서명">

```cs
public static void ProcessAndResolveDeeplink(AdjustDeeplink deeplink, Action<string> callback);
```

</CodeBlock>

단축 딥링크를 해석하려면 단축 딥링크로 `AdjustDeeplink` 객체를 인스턴스화하고, 콜백 함수를 사용하여 `Adjust.ProcessAndResolveDeeplink` 메서드에 전달합니다. 콜백 함수는 해석된 딥링크를 `string` 로 수신합니다.

```cs
AdjustDeeplink adjustDeeplink = new AdjustDeeplink("url");
Adjust.ProcessAndResolveDeeplink(adjustDeeplink, (resolvedLink) =>
{
   Debug.Log("Resolved link = " + resolvedLink);
});
```

`Adjust.ProcessAndResolveDeeplink` 메서드로 전달된 링크가 단축된 경우, 콜백 함수는 확장된 원본 링크를 수신합니다. 그렇지 않은 경우, 콜백 함수는 전달된 링크를 수신합니다.

</SdkVersion>

## 짧은 링크를 해석하기 위해 Adjust SDK 설정 {#set-up-the-adjust-sdk-to-resolve-short-links}

<CodeBlock title="메서드 서명">

```cs
private void DeeplinkResolvedCallback(string resolvedLink)
```

</CodeBlock>

`DeeplinkResolvedCallback` 메서드를 호출하여 사용자를 앱으로 딥링크한 짧은 Adjust 링크 URL을 확인합니다.

```cs
private void DeeplinkResolvedCallback(string resolvedLink)
{
    Debug.Log("Resolved link = " + resolvedLink);
}
// ...
Adjust.processDeeplink("your-deeplink", DeeplinkResolvedCallback, "game-object-name-where-Adjust-script-is-loaded");
```

`DeeplinkResolvedCallback` 메서드로 전송된 링크가 단축된 경우, `resolvedLink` 는 확장된 원본 링크를 반환합니다. 메서드에 전달한 링크가 단축되지 않은 경우 `resolvedLink` 에는 전달한 링크와 동일한 링크가 포함됩니다.

</SdkVersion>
