Use these methods to retrieve device information.

<Function>

## getAdid

<Callout type="note">

The ADID is available only after the install has been recorded.

</Callout>

Returns the <Abbr>ADID (Adjust Device ID)</Abbr> associated with the device.

<slot slot="signature">

import GetAdidSig from "@unity-signatures/Adjust/getAdid.mdx"

<GetAdidSig />

</slot>

<slot slot="snippet">

import GetAdid from "@unity-examples/Adjust/getAdid.mdx"

<GetAdid />

</slot>

<slot slot="returns">

`adid` (string)
: The Adjust identifier associated with the device

</slot>

</Function>
