<callout type="important"> 

下列步骤仅会在 Adjust SDK 中设置订阅跟踪。要启动该功能，Adjust 需要设置应用相关的具体信息。请发送邮件至 [support@adjust.com](mailto:support@adjust.com)或与您的技术客户经理联系进行设置。
</Callout> 

您可以用 Adjust SDK 记录 App Store 和 Play 应用商店的订阅，并验证这些订阅是否有效。用户购买订阅后，创建一个包含细节的 `AdjustAppStoreSubscription2dx` 或 `AdjustPlayStoreSubscription2dx` 实例。

1\. 实例化订阅对象
------------

要开始设置，先创建一个包含订阅购买细节的订阅对象。
<tabs> <tab title="App Store" sync="appstore" icon="PlatformIos"> <codeblock title="方法签名"> 

```cpp
public: AdjustAppStoreSubscription2dx(std::string price, std::string currency, std::string transactionId, std::string receipt)
```

</CodeBlock> 

创建一个 `AdjustAppStoreSubscription2dx` 对象，属性如下：
<table> 

|       参数        | 数据类型 |                                                                                                                           描述                                                                                                                           |
|-----------------|------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `price`         | 字符串  | 订阅价格                                                                                                                                                                                                                                                   |
| `currency`      | 字符串  | 订阅使用的币种。此信息格式为 [`priceLocale`](https://developer.apple.com/documentation/foundation/nslocale/1642836-currencycode?language=objc) 对象的 [`currencyCode`](https://developer.apple.com/documentation/storekit/skproduct/1506145-pricelocale?language=objc)。 |
| `transactionId` | 字符串  | 您的交易 ID                                                                                                                                                                                                                                                |
| `receipt`       | 字符串  | 收据信息                                                                                                                                                                                                                                                   |

</Table> 

```cpp
AdjustAppStoreSubscription2dx subscription = AdjustAppStoreSubscription2dx(
    price,
    currency,
    transactionId,
    receipt
);
```

</Tab> <tab title="Play 商店" sync="playstore" icon="PlatformGooglePlay"> <codeblock title="方法签名"> 

```cpp
public: AdjustPlayStoreSubscription2dx(std::string price, std::string currency, std::string sku, std::string orderId, std::string signature, std::string purchaseToken);
```

</CodeBlock> 

创建一个 `AdjustPlayStoreSubscription2dx` 对象，属性如下：
<table> 

|       参数        | 数据类型 |                                                                 描述                                                                 |
|-----------------|------|------------------------------------------------------------------------------------------------------------------------------------|
| `price`         | 数字   | 订阅价格                                                                                                                               |
| `currency`      | 字符串  | 订阅使用的币种                                                                                                                            |
| `sku`           | 字符串  | 产品 ID                                                                                                                              |
| `orderId`       | 字符串  | 您的交易 ID                                                                                                                            |
| `signature`     | 字符串  | 购买数据的签名                                                                                                                            |
| `purchaseToken` | 字符串  | 交易唯一识别码。请参考[Google 的文档](https://developer.android.com/reference/com/android/billingclient/api/Purchase#getPurchaseToken())，了解更多信息。 |

</Table> 

```cpp
AdjustPlayStoreSubscription2dx subscription = AdjustPlayStoreSubscription2dx(
    price,
    currency,
    sku,
    orderId,
    signature,
    purchaseToken
);
```

</Tab> </Tabs> 

### 记录购买日期

您可以记录用户购买订阅的日期。SDK 会返回该数据用于报告。
<tabs> <tab title="App Store" sync="appstore" icon="PlatformIos"> <codeblock title="方法签名"> 

```cpp
void setTransactionDate(std::string transactionDate);
```

</CodeBlock> 

在您的订阅对象上调用 `setTransactionDate` 方法来记录订阅的时间戳。

```cpp
AdjustAppStoreSubscription2dx subscription = AdjustAppStoreSubscription2dx(
    price,
    currency,
    transactionId,
    receipt
);
subscription.setTransactionDate(transactionDate);
```

</Tab> <tab title="Play 商店" sync="playstore" icon="PlatformGooglePlay"> <codeblock title="方法签名"> 

```cpp
void setPurchaseTime(std::string purchaseTime);
```

</CodeBlock> 

在您的订阅对象上调用 `setPurchaseTime` 方法来记录订阅的时间戳。

```cpp
AdjustPlayStoreSubscription2dx subscription = AdjustPlayStoreSubscription2dx(
    price,
    currency,
    sku,
    orderId,
    signature,
    purchaseToken
);
subscription.setPurchaseTime(purchaseTime);
```

</Tab> </Tabs> 

### 记录购买地区 \(仅限 iOS\)

<codeblock title="方法签名"> 

```cpp
void setSalesRegion(std::string salesRegion);
```

</CodeBlock> 

您可以记录用户购买订阅的地区。为此，请在订阅对象上调用 `setSalesRegion` 方法并将国家代码作为 **字符串** 传递。此信息格式需为 [`Storefront`](https://developer.apple.com/documentation/storekit/storefront/3792000-countrycode) 对象 的 [`countryCode`](https://developer.apple.com/documentation/storekit/storefront) 。

```cpp
AdjustAppStoreSubscription2dx subscription = AdjustAppStoreSubscription2dx(
    price,
    currency,
    transactionId,
    receipt
);
subscription.setSalesRegion(salesRegion);
```

### 添加回传参数

您可以向订阅对象附加回传参数。SDK 会将这些参数附加至您的回传 URL。要添加回传参数，请在订阅对象上调用 `addCallbackParameter` 方法。多次调用该方法可添加多个回传参数。
<tabs> <tab title="App Store" sync="appstore" icon="PlatformIos"> <codeblock title="方法签名"> 

```cpp
void addCallbackParameter(std::string key, std::string value);
```

</CodeBlock> 

```cpp
AdjustAppStoreSubscription2dx subscription = AdjustAppStoreSubscription2dx(
    price,
    currency,
    transactionId,
    receipt
);
subscription.addCallbackParameter("key1", "value1");
subscription.addCallbackParameter("key2", "value2");
```

</Tab> <tab title="Play 商店" sync="playstore" icon="PlatformGooglePlay"> <codeblock title="方法签名"> 

```cpp
void addCallbackParameter(std::string key, std::string value);
```

</CodeBlock> 

```cpp
AdjustPlayStoreSubscription2dx subscription = AdjustPlayStoreSubscription2dx(
    price,
    currency,
    sku,
    orderId,
    signature,
    purchaseToken
);
subscription.addCallbackParameter("key1", "value1");
subscription.addCallbackParameter("key2", "value2");
```

</Tab> </Tabs> 

### 添加合作伙伴参数

您可以向订阅对象附加合作伙伴参数。用户购买订阅时，SDK 会将这些参数发送至 Adjust 服务器。Adjust 服务器会将此信息转发至您的合作伙伴。要添加合作伙伴参数，请在您的订阅对象上调用`addPartnerParameter`方法。多次调用该方法可添加多个合作伙伴参数。
<tabs> <tab title="App Store" sync="appstore" icon="PlatformIos"> <codeblock title="方法签名">

```cpp
void addPartnerParameter(std::string key, std::string value);
```

</CodeBlock>

```cpp
    receipt

subscription.addPartnerParameter("key2", "value2");
```

2\. 记录订阅信息
-----------

设置好订阅对象后，您就可以通过 Adjust SDK 对其进行记录了。
<tabs> <tab title="App Store" sync="appstore" icon="PlatformIos"> <codeblock title="方法签名"> 

```cpp
static void trackAppStoreSubscription(AdjustAppStoreSubscription2dx subscription);
```

</CodeBlock> 

将订阅对象传送至 `trackAppStoreSubscription` 方法，以记录用户订阅购买。

```cpp
AdjustAppStoreSubscription2dx subscription = AdjustAppStoreSubscription2dx(
    price,
    currency,
    transactionId,
    receipt
);
subscription.setTransactionDate(transactionDate);
subscription.setSalesRegion(salesRegion);
subscription.addCallbackParameter("key1", "value1");
subscription.addCallbackParameter("key2", "value2");
subscription.addPartnerParameter("key1", "value1");
subscription.addPartnerParameter("key2", "value2");
Adjust2dx::trackAppStoreSubscription(subscription);
```

</Tab> <tab title="Play 商店" sync="playstore" icon="PlatformGooglePlay"> <codeblock title="方法签名"> 

```cpp
static void trackPlayStoreSubscription(AdjustPlayStoreSubscription2dx subscription);
```

</CodeBlock> 

将订阅对象传送至 `trackPlayStoreSubscription` 方法，以记录用户订阅购买。

```cpp
AdjustPlayStoreSubscription2dx subscription = AdjustPlayStoreSubscription2dx(
    price,
    currency,
    sku,
    orderId,
    signature,
    purchaseToken
);
subscription.setPurchaseTime(purchaseTime);
subscription.addCallbackParameter("key1", "value1");
subscription.addCallbackParameter("key2", "value2");
subscription.addPartnerParameter("key1", "value1");
subscription.addPartnerParameter("key2", "value2");
Adjust2dx::trackPlayStoreSubscription(subscription);
```
