Adjust SDK는 디바이스 정보를 반환하는 헬프 메서드가 포함되어 있습니다. 해당 메서드를 사용하여 콜백에 세부 정보를 추가하고 리포트를 보완하시기 바랍니다.

Adjust 디바이스 ID
--------------

<codeblock title="메서드 서명"> 

```cpp
static std::string getAdid();
```

</CodeBlock> 

Adjust는 각 기기에 대해 고유한 ADID\(Adjust 기기 ID\)를 생성합니다. `getAdid` 메서드를 호출하면 이 ID가 **문자열** 로 반환됩니다.

```cpp
std::string adid = Adjust2dx::getAdid();
```

광고주용 ID
-------

<codeblock title="메서드 서명"> 

```cpp
static std::string getIdfa();
```

</CodeBlock> 

IDFA\(광고주용 ID\)는 Apple 기기에 대한 기기별 고유 ID입니다. `getIdfa` 메서드를 호출하면 이 ID가 **문자열** 로 반환됩니다.

```cpp
std::string idfa = Adjust2dx::getIdfa();
```

Google Play Services Advertising ID
-----------------------------------

<codeblock title="메서드 서명"> 

```cpp
static void getGoogleAdId(void (*adIdCallback)(std::string adId));
```

</CodeBlock> 

GPS ADID\(Google 플레이 서비스 광고 ID\)는 Android 기기에 대한 기기별 고유 ID입니다.

사용자는 기기에서 "광고 개인 최적화 사용 안 함\(Opt out of Ads Personalization\)"을 설정하고 GPS ADID 공유를 옵트아웃할 수 있습니다. 사용자가 이 설정을 활성화하면, 앱에서 GPS ADID 읽기 시도 시 Adjust SDK가 0으로 구성된 문자열을 반환합니다.

이 값에 액세스하려면 background thread에서 `getGoogleAdId` 메서드를 호출해야 합니다. GPS ADID 값에 액세스하기 위한 델리게이트 함수를 할당합니다.

```cpp
static void adIdCallbackMethod(std::string adId) {
    CCLOG("\nAdvertising identifier = %s", adId.c_str());
    CCLOG("\n");
}

// ...

Adjust2dx::getGoogleAdId(adIdCallbackMethod);
```

</CodeBlock> 

Amazon Ad ID\(Amazon 광고 ID\)는 Android 기기에 대한 기기별 고유 ID입니다. `getAmazonAdId` 메서드를 호출하면 이 ID가 **문자열** 로 반환됩니다.
