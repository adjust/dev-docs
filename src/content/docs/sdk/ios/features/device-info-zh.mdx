Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

[Adjust 设备标识符](adjust-device-identifier)
----------------------------------------

<codeblock title="Method signature"> 

```objc
+ (void)adidWithCompletionHandler:(nonnull ADJAdidGetterBlock)completion;
```

</CodeBlock> 

Adjust generates a unique Adjust Device ID \(ADID\) for each device. You can return the device's ADID by calling the `[Adjust adidWithCompletionHandler]` method with a completion handler. The SDK fetches the information asynchronously and passes it to your completion handler.
<tabs> <tab title="Swift" sync="swift"> 

```swift
Adjust.adid { adid in
      // add your completion handler
};
```

</Tab> <tab title="Objective-C" sync="objc"> 

```objc
[Adjust adidWithCompletionHandler:^(NSString * _Nullable adid) {
      // add your completion handler
}];
```

</Tab> </Tabs>

Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

[Adjust 设备标识符](adjust-device-identifier)
----------------------------------------

<codeblock title="Method signature"> 

```objc
+ (nullable NSString *)adid;
```

</CodeBlock> 

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `adid` 方法来通过 `NSString` 的形式返回此 ID。
<tabs> <tab title="Swift" sync="swift"> 

```swift
let adid = Adjust.adid()
```

</Tab> <tab title="Objective-C" sync="objc"> 

```objc
NSString *adid = [Adjust adid];
```

</Tab> <tab title="Javascript" sync="js"> 

```js
var adid = Adjust.getAdid();
```

</Tab> </Tabs>
