Adjust SDKには、デバイス情報を返すヘルパーメソッドが含まれています。これらのメソッドを使用することでコールバックに詳細を追加し、レポートを改善させましょう。

## AdjustのデバイスID {#adjust-device-identifier}

<CodeBlock title="メソッドシグネチャー">

```objc
+ (nullable NSString *)adid;
```

</CodeBlock>

Adjustは各デバイスに固有のAdjust Device ID（ADID）を生成します。`adid`メソッドを呼び出すと、このIDを`NSString`として返します。

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let adid = Adjust.adid()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *adid = [Adjust adid];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
var adid = Adjust.getAdid();
```

</Tab>
</Tabs>

## 広告ID {#id-for-advertisers}

<CodeBlock title="メソッドシグネチャー">

```objc
+ (nullable NSString *)idfa;
```

</CodeBlock>

IDFA（ID for Advertisers）は、Appleデバイスの端末固有の識別子です。`idfa`メソッドを呼び出すと、このIDを`NSString`として返します。

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let idfa = Adjust.idfa()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *idfa = [Adjust idfa];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
Adjust.getIdfa(function (idfa) {
   // …
});
```

</Tab>
</Tabs>
