Adjust SDK는 디바이스 정보를 반환하는 헬프 메서드가 포함되어 있습니다. 해당 메서드를 사용하여 콜백에 세부 정보를 추가하고 리포트를 보완하시기 바랍니다.

## Adjust 디바이스 ID {#adjust-device-identifier}

<CodeBlock title="메서드 서명">

```objc
+ (nullable NSString *)adid;
```

</CodeBlock>

Adjust는 각 기기에 대해 고유한 ADID\(Adjust 기기 ID\)를 생성합니다. `adid` 메서드를 호출하면 이 ID가 `NSString`로 반환됩니다.

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let adid = Adjust.adid()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *adid = [Adjust adid];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
var adid = Adjust.getAdid();
```

</Tab>
</Tabs>

## 광고주용 ID {#id-for-advertisers}

<CodeBlock title="메서드 서명">

```objc
+ (nullable NSString *)idfa;
```

</CodeBlock>

IDFA\(광고주용 ID\)는 Apple 기기에 대한 기기별 고유 ID입니다. `idfa` 메서드를 호출하면 이 ID가 `NSString`로 반환됩니다.

<Tabs>
<Tab title="Swift" sync="swift">

```swift
let idfa = Adjust.idfa()
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
NSString *idfa = [Adjust idfa];
```

</Tab>
<Tab title="Javascript" sync="js">

```js
Adjust.getIdfa(function (idfa) {
   // …
});
```

</Tab>
</Tabs>
