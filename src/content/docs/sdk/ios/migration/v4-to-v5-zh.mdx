Before you begin \{\#before\-you\-begin\}
----------------------------------------------

在更新到 SDK v5 之前，您需要进行以下操作：

1. SDK v5 支持[SDK 签名验证](https://help.adjust.com/en/article/sdk-signature)。如果您当前正在使用 SDK 签名库，则需要先将其卸载。
2. 如果您的应用针对低于以下版本的 API，则需要先更新应用才能使用 SDK v5： 
   * iOS: **12\.0** 
   * tvOS: **12\.0** 

Install the beta SDK \{\#install\-the\-sdk\}
-------------------------------------------------

要开始使用 SDK v5，您需要将其添加为 XCode 项目的依赖。为此请进行下列操作：

1. 请移除`Podfile`中所有较旧版本的 SDK。

2. 添加下列行至您的 `Podfile`：

   <codeblock title="Podfile"> 

   ```rb
   pod 'Adjust', :git => 'https://github.com/adjust/ios_sdk.git', :branch => 'v5'
   ```

   </CodeBlock> 
3. 运行`pod install`来安装依赖。

Changes \{\#changes\}
------------------------

<callout type="warning"> 

SDK v5 是一个重大版本更新。也就是说，SDK 发生了重大变化，需要您更新代码。
</Callout> 

安装 v5 SDK 后，您需要更新现有的 Adjust 代码以使用新的 API。

### Initialization method \{\#init\-method\}

<minorversion changed="v5"> 

在 SDK v4 中，SDK 的初始化方法是配置`ADJConfig`对象并将其传递至`[Adjust appDidLaunch]`方法。
</MinorVersion> <tabs> <tab title="Swift" sync="swift"> 

```swift
let yourAppToken = "{YourAppToken}"
let environment = ADJEnvironmentSandbox as? String
let adjustConfig = ADJConfig(
      appToken: yourAppToken,
      environment: environment)

Adjust.appDidLaunch(adjustConfig)
```

</Tab> <tab title="Objective-C" sync="objc"> 

```objc
#import "Adjust.h"

NSString *yourAppToken = @"{YourAppToken}";
NSString *environment = ADJEnvironmentSandbox;
*adjustConfig = [ADJConfig configWithAppToken:yourAppToken;
                                  environment:environment];

[Adjust appDidLaunch:adjustConfig];
```

</Tab> </Tabs> 

在 SDK v5 中，该方法已重命名为`[Adjust initSdk]` 。
<tabs> <tab title="Swift" sync="swift"> 

```swift
let yourAppToken = "{YourAppToken}"
let environment = ADJEnvironmentSandbox as? String
let adjustConfig = ADJConfig(
      appToken: yourAppToken,
      environment: environment)

Adjust.initSdk(adjustConfig)
```

</Tab> <tab title="Objective-C" sync="objc"> 

```objc
#import "Adjust.h"

NSString *yourAppToken = @"{YourAppToken}";
NSString *environment = ADJEnvironmentSandbox;
ADJConfig *adjustConfig = [[ADJConfig alloc] initWithAppToken:yourAppToken
                                                  environment:environment];

```

</Tab> </Tabs> 

### Signature setup \{\#signature\-setup\}

SDK v5 包含[SDK 签名库](https://help.adjust.com/en/article/sdk-signature)。您可以按照[签名库集成指南](/en/sdk/ios/integrations/signature-library)中的测试步骤对此进行测试。

签名保护功能默认处于禁用状态。要启用签名，您需要[执行签名验证](https://help.adjust.com/en/article/sdk-signature#enforce-signature-validation)。

</Tab> <tab title="Objective-C" sync="objc">

adjustConfig.setEventDeduplicationIdsMaxSize(20)

<minorversion changed="v5">

</MinorVersion> <tabs> <tab title="Swift" sync="swift">

func adjustDeeplinkResponse(_ deeplink: URL?) -> Bool {
   print("Deferred deep link callback called!")

   // If you don't want it to open it, return false instead.
   return true

在 SDK v5 中，您需要将选定的一个或多个域名以数组的形式发送。您还可以进行下列设置：

                     isDataResidency:YES];
