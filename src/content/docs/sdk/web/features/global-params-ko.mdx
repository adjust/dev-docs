글로벌 콜백과 파트너 파라미터를 등록하여 각 세션과 이벤트와 함께 SDK에서 추가 정보를 보낼 수 있습니다.

글로벌 콜백 파라미터 \{\#global\-callback\-parameters\}
---------------------------------------------------

Adjust 대시보드에 [콜백 URL을 등록](https://help.adjust.com/en/article/set-up-callbacks)하면 SDK가 세션이나 이벤트를 기록할 때 Adjust가 GET 요청을 콜백 URL로 보냅니다.

글로벌 콜백 파라미터를 서버에 구성할 수 있습니다. 파라미터 구성이 완료되면 SDK는 이를 콜백 URL에 추가합니다. 해당 정보를 사용하여, BI 시스템을 통해 사용자 인앱 행동을 분석할 수 있습니다.

### 콜백 파라미터 추가 \{\#add\-callback\-parameters\}

<codeblock title="메서드 서명"> 

```js
function addGlobalCallbackParameters(params: Array<globalparams id="sl-md0000000">): void;
```

</CodeBlock> 

`string` 키\-값 쌍 객체의 배열과 함께 `addGlobalCallbackParameters` 메서드를 호출하여 세션과 이벤트에 콜백 파라미터를 추가합니다. 필요한 만큼의 객체를 이 배열에 추가할 수 있습니다.

```js
Adjust.addGlobalCallbackParameters([
   { key: "key1", value: "value1" },
   { key: "key2", value: "value2" },
]);
```

위 예시의 경우 콜백 URL은 다음과 같습니다.
<codeblock highlight="key1, value1, key2, value2"> 

```text
https://www.mydomain.com/callback?key1=value1&key2=value2
```

</CodeBlock> 

### 콜백 파라미터 삭제 \{\#remove\-callback\-parameters\}

<codeblock title="메서드 서명"> 

```js
function removeGlobalCallbackParameter(key: string): void;
```

</CodeBlock> 

글로벌 콜백 파라미터가 더 이상 필요하지 않은 경우 삭제할 수 있습니다. 삭제하려면 파라미터 키를 `removeGlobalCallbackParameter` 메서드로 전달합니다.

```js
Adjust.removeGlobalCallbackParameter("key1");
```

### 모든 콜백 파라미터 삭제 \{\#clear\-all\-callback\-parameters\}

```js
function clearGlobalCallbackParameters(): void;
```

모든 글로벌 콜백 파라미터가 더 이상 필요하지 않은 경우 삭제할 수 있습니다. 해당 기능 사용을 원하는 경우 `clearGlobalCallbackParameters` 메서드를 호출합니다.

```js
Adjust.clearGlobalCallbackParameters();
```

글로벌 파트너 파라미터 \{\#global\-partner\-parameters\}
---------------------------------------------------

Adjust SDK가 기록하는 각 세션과 이벤트에는 [파트너 파라미터](https://help.adjust.com/en/article/data-sharing-ad-network#map-parameters)를 추가하여 네트워크 파트너에게 추가 정보를 전송할 수 있습니다.

Adjust는 설정한 [외부 파트너](https://help.adjust.com/en/article/integrated-partners)에게 파트너 파라미터를 전송합니다. 이러한 정보는 보다 세분화된 분석과 리타겟팅에 유용할 것입니다. 파트너에 대해 파라미터를 설정하고 활성화하면 Adjust 서버는 해당 파라미터를 전달합니다.
<callout type="note"> 

파트너 파라미터는 기본 설정에 따라 로 데이터에 나타나지 않습니다. `{partner_parameters}` 플레이스홀더를 추가하여 단일 문자열로 수신할 수 있습니다.
</Callout>
