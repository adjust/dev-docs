Adjust SDK는 플러그인을 통해 [삼성 CloudDev SDK](https://developer.samsung.com/instant-plays/samsung-clouddev-sdk/get-started.html) 를 지원합니다.

## [시작하기 전에](before-you-begin)

1. [시작 가이드](/ko/sdk/android) 를 참조하여 Adjust SDK를 앱에 연동하시기 바랍니다.
2. 삼성 담당자에게 삼성 CloudDev SDK 사본을 요청하고 프로젝트에 추가하십시오.

## [Samsung CloudDev 플러그인 설치](install-the-samsung-clouddev-plugin)

Samsung CloudDev 플러그인은 두 가지 방법으로 설치할 수 있습니다.

### [Maven으로 설치](install-with-maven)

Adjust SDK 선언 아래의 `build.gradle` 파일에 다음의 디펜던시를 추가합니다.

<CodeBlock title="build.gradle">

```groovy
implementation 'com.adjust.sdk:adjust-android-samsung-clouddev:4.38.3'
```

</CodeBlock>

### [JAR 또는 AAR로 추가](add-as-a-jar-or-aar)

삼성 CloudDev 플러그인은 [Adjust SDK 리포지토리](https://github.com/adjust/android_sdk/releases/latest)에서 Java Archive \(JAR\) 및 Android Archive \(AAR\)로 이용 가능합니다.
