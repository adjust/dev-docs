[구매 검증](https://help.adjust.com/ko/article/purchase-verification)을 활성화한 경우 검증을 위해 구매 이벤트와 함께 추가 정보를 전송해야 합니다. Adjust의 서버가 `AdjustEvent` 객체를 통해 해당 정보를 수신하면, 해당 정보를 Google에 전달하여 구매를 검증합니다.

구매를 검증하려면 `AdjustEvent` 객체를 이벤트 토큰으로 인스턴스화하고 다음의 속성을 설정하시기 바랍니다.

-  `productId` \(`String`\): 성공적으로 구매된 아이템의 제품 ID
-  `purchaseToken` \(`String`\): 성공적으로 완료된 인앱 구매에 대해 생성된 구매 토큰

<Callout type="seealso">

`AdjustEvent` 클래스에 대한 자세한 내용은 [이벤트 정보 전송](/ko/sdk/android/features/events) 을 참조하시기 바랍니다.

</Callout>
<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
val adjustEvent = AdjustEvent("abc123")
adjustEvent.setRevenue(6.0, "EUR");
adjustEvent.setProductId("product-id");
adjustEvent.setPurchaseToken("purchase-token");
Adjust.trackEvent(adjustEvent);
```
