如果您想使用 AppLovin MAX 监测广告收入，可以借助 SDK 到 SDK 集成，将数据发送到 Adjust 服务器。

<Callout type="note">

如果您对 AppLovin MAX 广告收入监测有任何疑问，请联系您的专属技术客户经理，或发送邮件至 [support@adjust.com](mailto:support@adjust.com)。

</Callout>

## 要求 {#requirements}

-  Adjust 安卓 SDK v4\.33\.5 及更新版本

## 示例 {#examples}

<Tabs>
<Tab title="Java">

```java
@Override
public void onAdRevenuePaid(final MaxAd ad) {
    AdjustAdRevenue adjustAdRevenue = new AdjustAdRevenue( AdjustConfig.AD_REVENUE_APPLOVIN_MAX);
    adjustAdRevenue.setRevenue(ad.getRevenue(), "USD");
    adjustAdRevenue.setAdRevenueNetwork(ad.getNetworkName());
    adjustAdRevenue.setAdRevenueUnit(ad.getAdUnitId());
    adjustAdRevenue.setAdRevenuePlacement(ad.getPlacement());

    Adjust.trackAdRevenue( adjustAdRevenue);
}
```

</Tab>
</Tabs>

要了解此集成在具体情境中如何工作，请查看 AppLovin 的[示例 安卓 应用。](https://github.com/AppLovin/AppLovin-MAX-SDK-Android/blob/master/AppLovin%20MAX%20Demo%20App%20-%20Java/app/src/main/java/com/applovin/enterprise/apps/demoapp/ads/max/RewardedAdActivity.java#L133-L144)
