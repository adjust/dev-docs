Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## Adjust 设备标识符

```java
public static String getAdid()
```

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。调用 `getAdid` 方法来通过 `String` 的形式返回此 ID。

<Tabs>

 <Tab title="Kotlin" sync="kotlin">

```kotlin
val adid = Adjust.getAdid()
```

</Tab>

<Tab title="Java" sync="java" >

```java
String adid = Adjust.getAdid();
```

</Tab>

<Tab title="Javascript" sync="js" >

```js
let adid = Adjust.getAdid();
```

</Tab>

</Tabs>

## Google Play 服务广告 ID

```java
public static void getGoogleAdId(Context context, OnDeviceIdsRead onDeviceIdRead)
```

Google Play 服务广告 ID \(GPS ADID\) 是安卓设备的独特标识符。

用户可以通过启用设备的 "选择停用广告个性化功能" 设置来停止分享自己的 GPS ADID。当用户启用该设置后，Adjust SDK 试图读取 GPS ADID 时会返回一串零。

您可以通过在后台线程中调用 `getGoogleAdId` 方法来获取该值。指定一个委托 \(delgate\) 函数，以访问 GPS ADID 值。

<Tabs>

 <Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getGoogleAdId(this, object : OnDeviceIdsRead {
   override fun onGoogleAdIdRead(googleAdId: String) {}
})
```

</Tab>

<Tab title="Java" sync="java" >

```java
Adjust.getGoogleAdId(this, new OnDeviceIdsRead() {
    @Override
    public void onGoogleAdIdRead(String googleAdId) {}
});
```

</Tab>

<Tab title="Javascript" sync="js" >

```js
Adjust.getGoogleAdId(function (googleAdId) {
   // ...
});
```

</Tab>

</Tabs>

## Amazon 广告主 ID

```java
public static String getAmazonAdId(final Context context)
```

Amazon 广告 ID \(Amazon Ad ID\) 是 Android 设备的独特标识符。调用 `getAmazonAdId` 方法来通过 `String` 的形式返回此 ID。

<Tabs>

 <Tab title="Kotlin" sync="kotlin">

```kotlin
val amazonAdId = Adjust.getAmazonAdId(getApplicationContext())
```

</Tab>

<Tab title="Java" sync="java" >

```java
String amazonAdId = Adjust.getAmazonAdId(getApplicationContext());
```

</Tab>

<Tab title="Javascript" sync="js" >

```js
let amazonAdId = Adjust.getAmazonAdId();
```

</Tab>

</Tabs>
