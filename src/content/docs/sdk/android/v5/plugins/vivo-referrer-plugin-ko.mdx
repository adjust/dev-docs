Vivo 리퍼러 플러그인은 Adjust SDK가 기기에서 Vivo 앱 스토어 설치 리퍼러 값을 읽을 수 있도록 합니다. 이후 SDK는 해당 정보를 설치 페이로드의 일부로서 Adjust 서버에 전송합니다.

Vivo 설치 리퍼러는 Vivo 앱 스토어 5\.3\.0\.0 이상 버전을 사용하는 모든 디바이스에서 이용 가능합니다.

Android 시작 가이드를 참조하여 Adjust SDK를 앱에 연동하시기 바랍니다.

## Vivo 리퍼러 플러그인 설치 {#install-the-vivo-referrer-plugin}

Vivo 리퍼러 플러그인은 두 가지 방식으로 설치할 수 있습니다.

### Maven {#maven}

Adjust SDK 선언 아래의 `build.gradle` 파일에 다음의 디펜던시를 추가합니다.

<CodeBlock title="build.gradle">

```groovy
implementation 'com.adjust.sdk:adjust-android:$ANDROID_V4_VERSION'

implementation 'com.adjust.sdk:adjust-android-vivo-referrer:$ANDROID_V4_VERSION'

```

</CodeBlock>

### JAR {#jar}

Vivo 리퍼러 플러그인은 GitHub [릴리스 페이지](https://github.com/adjust/android_sdk/releases/latest) 에서 이용 가능합니다.

## 플러그인 사용하기 {#use-the-plugin}

Vivo 리퍼러 플러그인은 기본 설정에 따라 **활성화** 되어 있습니다. `AdjustVivoReferrer` 클래스 메서드를 사용하여 Vivo 설치 리퍼러 정보 읽기를 켜거나 끌 수 있습니다.

설치 리퍼러 읽기를 **비활성화** 하려면 SDK 초기화 **전에** `AdjustVivoReferrer.doNotReadVivoReferrer()` 를 호출합니다.

<Tabs>
<Tab title="Java" sync="java">

```java
AdjustVivoReferrer.doNotReadVivoReferrer();
//...
Adjust.onCreate(config);
```

</Tab>
<Tab title="Kotlin" sync="kotlin">

```kotlin
AdjustVivoReferrer.doNotReadVivoReferrer()
//...
Adjust.onCreate(config)
```

</Tab>
</Tabs>

설치 리퍼러 읽기를 **재활성화** 하려면 SDK 초기화 **전에** `AdjustVivoReferrer.readVivoReferrer(applicationContext)` 를 호출합니다.

<Tabs>
<Tab title="Java" sync="java">

```java
AdjustVivoReferrer.readVivoReferrer(applicationContext);
//...
Adjust.onCreate(config);
```

</Tab>
<Tab title="Kotlin" sync="kotlin">

```kotlin
AdjustVivoReferrer.readVivoReferrer(applicationContext)
//...
Adjust.onCreate(config)
```

</Tab>
</Tabs>
