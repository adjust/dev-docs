The Adjust SDK supports [the Samsung CloudDev SDK](https://developer.samsung.com/instant-plays/samsung-clouddev-sdk/get-started.html) with a plugin.

## Before you begin

1. Follow the [get started guide](/en/sdk/android) to integrate the Adjust SDK in your app.
2. Ask your Samsung representative for a copy of the Samsung CloudDev SDK and add it to your project.

## Install the Samsung CloudDev plugin

There are two ways to install the Samsung CloudDev plugin:

### Install with Maven

Add the following dependency to your `build.gradle` file under your Adjust SDK declaration.

```groovy title="build.gradle"
implementation 'com.adjust.sdk:adjust-android-samsung-clouddev:4.38.3'
```

### Add as a JAR or AAR

The Samsung CloudDev plugin is available as a Java Archive (JAR) and Android Archive (AAR) in the [Adjust SDK repository](https://github.com/adjust/android_sdk/releases/latest)

## Read device information

Once you've installed the Samsung CloudDev SDK and the Adjust CloudDev SDK plugin, you can read a device's Google Play Advertising ID by calling the `getGoogleAdId` method.

<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getGoogleAdId(this, object : OnDeviceIdsRead {
   override fun onGoogleAdIdRead(googleAdId: String) {}
})
```

</Tab>

<Tab title="Java" sync="java" >

```java
Adjust.getGoogleAdId(this, new OnDeviceIdsRead() {
    @Override
    public void onGoogleAdIdRead(String googleAdId) {}
});
```

</Tab>

<Tab title="Javascript" sync="js" >

```js
Adjust.getGoogleAdId(function (googleAdId) {
   // ...
});
```

</Tab>
