<sdkversion version="v5"> <callout type="note"> 

Metaインストールリファラープラグインは、 [Metaが必要とする](https://developers.facebook.com/docs/app-ads/meta-install-referrer#step-1--add-the-meta-apps)`<queries id="sl-md0000000">`要素をバンドルしています。これらを`AndroidManifest.xml`に追加する必要はありません。
</Callout> 

Metaインストールリファラープラグインは、Adjust SDKがデバイス上のMetaインストールリファラー値を読み取ることを可能にします。その後、SDKはこの情報をインストールペイロードの一部として、Adjustサーバーに送信します。

プラグインをアプリに追加する方法：

1. `build.gradle`ファイルに以下のdependencyを追加します：

   <codeblock title="build.gradle"> 

   ```groovy
   dependencies {
      implementation 'com.adjust.sdk:adjust-android-meta-referrer:5.0.0'
   }
   ```

   </CodeBlock> 
2. [App Dashboard](https://developers.facebook.com/apps)でMetaアプリIDを検索します。詳しくはMetaの[App Dashboardのドキュメント](https://developers.facebook.com/docs/development/create-an-app/app-dashboard)をご覧ください。

3. `AdjustConfig.setFbAppId`メソッドに`String`引数としてアプリIDをパスします。

<tabs> <tab title="Kotlin"> <codeblock highlight="{range: 2}"> 

```kotlin
val config = AdjustConfig(this, appToken, environment)
config.setFbAppId(<fb_app_id_string id="sl-md0000000">)
Adjust.initSdk(config)
```

</CodeBlock> </Tab> <tab title="Java"> <codeblock highlight="{range: 2}"> 

```java
AdjustConfig config = new AdjustConfig(this, appToken, environment);
config.setFbAppId(<fb_app_id_string id="sl-md0000000">);
Adjust.initSdk(config);
```

</CodeBlock> </Tab> </Tabs> </SdkVersion> <sdkversion version="v4"> 

MetaインストールリファラープラグインはSDK v5でのみ利用可能です。詳細は[移行ガイド](/en/sdk/android/migration/v4-to-v5)をご覧ください
</SdkVersion>
