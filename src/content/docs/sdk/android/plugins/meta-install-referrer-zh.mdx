<sdkversion version="v5"> <callout type="note"> 

Meta Install Referrer 插件捆绑了[Meta 要求](https://developers.facebook.com/docs/app-ads/meta-install-referrer#step-1--add-the-meta-apps)的`<queries id="sl-md0000000">`元素。您无需将这些元素添加至`AndroidManifest.xml`。
</Callout> 

通过 Meta Install Referrer 插件，Adjust SDK 可以读取设备上的Meta Install Referrer 值。SDK 随后会将此信息作为安装 payload 的一部分发送至 Adjust 服务器。

要将插件添加至您的应用：

1. 将以下依赖添加至您的 `build.gradle` 文件：

   <codeblock title="build.gradle"> 

   ```groovy
   dependencies {
      implementation 'com.adjust.sdk:adjust-android-meta-referrer:5.0.0-SNAPSHOT'
   }
   ```

   </CodeBlock> 
2. 在[应用控制面板](https://developers.facebook.com/apps)中找到您的 Meta app ID。更多信息，请参考 Meta 的[应用控制面板文档](https://developers.facebook.com/docs/development/create-an-app/app-dashboard)。

3. 将应用 ID 作为 `String` 参数传递至 `AdjustConfig.setFbAppId` 方法。

<tabs> <tab title="Kotlin"> <codeblock highlight="{range: 2}"> 

```kotlin
val config = AdjustConfig(this, appToken, environment)
config.setFbAppId(<fb_app_id_string id="sl-md0000000">)
Adjust.initSdk(config)
```

</CodeBlock> </Tab> <tab title="Java"> <codeblock highlight="{range: 2}"> 

```java
AdjustConfig config = new AdjustConfig(this, appToken, environment);
config.setFbAppId(<fb_app_id_string id="sl-md0000000">);
Adjust.initSdk(config);
```

</CodeBlock> </Tab> </Tabs> </SdkVersion> <sdkversion version="v4">

</SdkVersion>
