<SdkVersion version="v5">
<Callout type="note">

The Meta Install Referrer plugin bundles the `<queries id="sl-md0000000">` elements [required by Meta](https://developers.facebook.com/docs/app-ads/meta-install-referrer#step-1--add-the-meta-apps). You don't need to add these to your `AndroidManifest.xml`.

</Callout>

The Meta Install Referrer plugin enables the Adjust SDK to read Meta Install Referrer values on a device. The SDK then sends this information to Adjust's servers as part of the install payload.

To add the plugin to your app:

1. Add the following dependency to your `build.gradle` file:

   <CodeBlock title="build.gradle">

   ```groovy
   dependencies {
      implementation 'com.adjust.sdk:adjust-android-meta-referrer:$ANDROID_V5_VERSION'
   }
   ```

   </CodeBlock>

2. 在[应用控制面板](https://developers.facebook.com/apps)中找到您的 Meta app ID。更多信息，请参考 Meta 的[应用控制面板文档](https://developers.facebook.com/docs/development/create-an-app/app-dashboard)。

3. 将应用 ID 作为 `String` 参数传递至 `AdjustConfig.setFbAppId` 方法。

<Tabs>
<CodeBlock highlight="{range: 2}">

```kotlin
val config = AdjustConfig(this, appToken, environment)
config.setFbAppId(<fb_app_id_string id="sl-md0000000">)
Adjust.initSdk(config)
```

</CodeBlock>
<CodeBlock highlight="{range: 2}">

```java
AdjustConfig config = new AdjustConfig(this, appToken, environment);
config.setFbAppId(<fb_app_id_string id="sl-md0000000">);
Adjust.initSdk(config);
```

</CodeBlock>
