스마트 배너는 초기화 이후 숨김 설정이 가능하며, 필요한 경우 다시 표시할 수 있습니다. 해당 기능 사용을 원하는 경우 `AdjustSmartBanner.hide` 메서드를 호출합니다.

<Callout type="note">

스마트 배너를 숨겨도 DOM에서 스마트 배너가 제거되는 것은 아닙니다.

</Callout>

```js
AdjustSmartBanner.hide();
```

스마트 배너를 다시 표시하려면 `AdjustSmartBanner.show` 메서드를 호출합니다.

<Callout type="important">

웹 앱이 SPA\(싱글 페이지 어플리케이션\)인 경우, 네비게이션 및 페이지 URL 변경 이후에 이 메서드를 호출해야 합니다. 이 경우 SDK는 강제로 페이지의 업데이트된 URL을 읽어 현재 페이지에 적합한 배너를 찾거나, 다이나믹 딥링크로 링크를 빌드할 때 업데이트된 GET 파라미터를 사용할 수 있습니다.

</Callout>

```js
AdjustSmartBanner.show();
```
