You can integrate the Adjust Smart Banner SDK into Shopify to distribute smart banners via Shopify tags. Follow the steps in this guide to set this up.

## [Integrate the SDK in Shopify](integrate-the-sdk-in-shopify)

1. Navigate to your store admin tools at https://admin.shopify.com.
2. Select **Online Store -> Themes**.
3. Select the three dot menu on your active theme and select **Edit code**.

   ![A screenshot of the Shopify theme overview page](@images/smart-banner-sdk/shopify/1-overview.png)

4. Find and open the `theme.liquid` file.
5. Add the [SDK loading snippet](/en/sdk/smart-banner#using-a-cdn) inside the theme’s `<head></head>` tags.

   ![A screenshot of the Shopify theme editor](@images/smart-banner-sdk/shopify/2-loading-snippet.png)

6. Scroll down to the theme’s `<body>` tag and enter your SDK initialization call. This call must include an appToken. Check out the [SDK initialization options](/en/sdk/smart-banner/init-options) to configure the SDK using the `init` method.

   ```html
   <!-- Adjust Smart Banner SDK -->
   <div id="AdjSbRoot"></div>
   <script>
      AdjustSmartBanner.init({
         appToken: "YOUR_TOKEN",
         logLevel: "verbose",
         bannerParent: document.getElementById("AdjSbRoot"),
      });
   </script>
   <!-- Adjust Smart Banner SDK -->
   ```

   ![A screenshot of the Shopify theme editor showing the Smart Banner SDK init options](@images/smart-banner-sdk/shopify/3-init-options.png)

7. Save your changes.

<Callout type="note">

The snippet in this example adds a new divider with the ID `AdjSbRoot`. This divider is used as the parent of the smart banner. You should set this to prevent the Shopify header from overlaying the smart banner. You can apply styles to the banner parent or choose a different parent element. If you don’t add a parent element, the smart banner attaches to `document.body`.

</Callout>
