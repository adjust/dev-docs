Adjust 스마트 배너 SDK를 Adobe Launch에 연동하여, Adobe Launch 태그를 통해 스마트 배너를 배포할 수 있습니다. 설정을 위해 본 가이드의 단계를 수행하시기 바랍니다.

## [Adobe Launch에 SDK 연동](integrate-the-sdk-in-adobe-launch)

1. [https://launch.adobe.com](https://launch.adobe.com)에서 스토어 관리자 도구로 이동합니다.

2. **Rules** 탭을 엽니다.

3. **Create New Rule** 을 선택합니다. **Name** 필드에 새 규칙의 이름을 입력하여 이름을 지정합니다.

   ![Adobe 실행 규칙 생성 화면 스크린샷](@images/smart-banner-sdk/adobe-launch/1-create-rule.png)

4. **Events** 섹션 아래 **Add** 를 선택하여 새 이벤트 트리거를 설정합니다.

5. 화면의 이벤트 양식을 작성하여 스마트 배너 트리거 이벤트를 정의합니다. 필요에 따라 모든 페이지 또는 특정 페이지에 대해 이 이벤트를 구성할 수 있습니다.

   ![Adobe 실행 규칙 구성 화면 스크린샷](@images/smart-banner-sdk/adobe-launch/2-trigger-event.png)

6. **Actions** 섹션 아래 **Add** 를 선택하여 새 액션을 설정합니다.

   ![Adobe의 Action Configuration 실행 화면 스크린샷](@images/smart-banner-sdk/adobe-launch/3-action-setup.png)

7. **Action Type** 드롭다운 메뉴에서 **Custom Code** 를 선택합니다.

8. **Language** 섹션에서 **Javascript** 를 선택하고, **Open Editor** 를 선택하여 Javascript 코드를 수정합니다.

9. [SDK loading snippet](/en/sdk/smart-banner#using-a-cdn)을 액션에 추가하여 이벤트가 트리거될 때 스마트 배너 SDK를 로딩합니다.

10.   **Save Rule** 을 선택하여 규칙을 저장합니다.

Adobe Launch에서의 규칙 설정에 대한 자세한 정보는 [Adobe 규칙 문서](https://experienceleague.adobe.com/docs/experience-platform/tags/ui/rules.html)를 확인하시기 바랍니다.
