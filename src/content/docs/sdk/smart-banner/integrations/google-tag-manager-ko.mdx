Adjust 스마트 배너 SDK를 Google Tag Manager에 연동하면 GTM 컨테이너를 통해 스마트 배너를 배포할 수 있습니다. 설정을 위해 본 가이드의 단계를 수행하시기 바랍니다.

## [Google Tag Manager에 SDK 연동](integrate-the-sdk-in-google-tag-manager)

1. Google Tag Manager로 이동합니다.

2. **Tags → New** 를 선택합니다.

3. **Tag Configuration** 을 선택한 다음 **Custom HTML** 을 선택합니다.

4. [HTML](/en/sdk/smart-banner#using-a-cdn) 필드에 **SDK loading snippet** 을 추가하고, **Firing Triggers** 를 **Initialization\-All Pages** 로 설정합니다.

   ![GTM 태그 구성 화면 스크린샷](@images/smart-banner-sdk/gtm/1-gtm-configuration.png)

5. 또 다른 커스텀 HTML 태그를 추가하고 `<script></script>` 태그 사이에 SDK 초기화 콜을 입력합니다. 이 콜은 반드시 `appToken`를 포함해야 합니다. `init` 메서드를 사용하여 SDK를 구성하려면 [SDK 초기화 옵션](/en/sdk/smart-banner/init-options)을 확인합니다.

   ```js
   window.AdjustSmartBanner.init({ appToken: "TOKEN" });
   ```

   ![초기화 옵션이 있는 커스텀 HTML 편집기의 스크린샷](@images/smart-banner-sdk/gtm/2-init-options.png)

6. 화면 우측 상단의 더하기 아이콘\(`+`\)을 선택하여 새 트리거를 생성합니다.

7. **Trigger Configuration** 으로 이동한 다음 **Page View** 를 선택하여 각 페이지 뷰의 스크립트를 로딩합니다. 초기화를 모든 페이지에서 트리거할지 또는 특정 호스트 이름을 입력할지 구성할 수 있습니다.

   ![실행 트리거 구성 스크린샷](@images/smart-banner-sdk/gtm/3-trigger-config.png)

8. 트리거 구성을 저장하고 **Tag Configuration** 으로 돌아갑니다.

9. 모든 설정이 올바른지 확인하고, GTM 컨테이너를 퍼블리시합니다.
