您可以将 Adjust 智能横幅 SDK 集成到 Shopify 并通过 Shopify 标签发送智能横幅。请按照本指南中的步骤进行设置。

## [在 Shopify 中集成 SDK](integrate-the-sdk-in-shopify)

1. 访问[https://admin.shopify.com](https://admin.shopify.com)找到您的商店管理员工具。

2. 选择 **Online Store → Themes** 。

3. 选择活跃主题上的三点菜单，然后选择 **Edit code** 。

   ![A screenshot of the Shopify theme overview page](@images/smart-banner-sdk/shopify/1-overview.png)

4. 找到并打开`theme.liquid`文件。

5. 将[SDK 加载片段](/en/sdk/smart-banner#using-a-cdn)添加进主题的`<head></head>`标签中。

   ![A screenshot of the Shopify theme editor](@images/smart-banner-sdk/shopify/2-loading-snippet.png)

6. 向下滚动至主题的`<body>`标签，输入 SDK 初始化调用。该调用必须包含 appToken。请查看[SDK 初始化选项](/en/sdk/smart-banner/init-options)来使用`init`方法配置 SDK。

   ```html
   <!-- Adjust Smart Banner SDK -->
   <div id="AdjSbRoot"></div>
   <script>
      AdjustSmartBanner.init({
         appToken: "YOUR_TOKEN",
         logLevel: "verbose",
         bannerParent: document.getElementById("AdjSbRoot"),
      });
   </script>
   <!-- Adjust Smart Banner SDK -->
   ```

   ![A screenshot of the Shopify theme editor showing the Smart Banner SDK init options](@images/smart-banner-sdk/shopify/3-init-options.png)

7. 保存变更。

<Callout type="note">

本示例中的代码片段添加了 ID 为`AdjSbRoot`的新 divider 。此 divider 被用作智能横幅的父级。您需要进行此设置，以防止 Shopify 标头与智能横幅重叠。您可以为横幅父级应用样式或选择不同的父级元素。如果您不添加父级元素，智能横幅就会附加到`document.body`上。

</Callout>
