Adjust Smart Banner SDK를 Shopify에 연동하면 Shopify 태그를 통해 스마트 배너를 배포할 수 있습니다. 설정을 위해 본 가이드의 단계를 수행하시기 바랍니다.

## [Shopify에 SDK 연동](integrate-the-sdk-in-shopify)

1. [https://admin.shopify.com](https://admin.shopify.com)에서 스토어 관리자 도구로 이동합니다.

2. **Online Store → Themes** 를 선택합니다.

3. 활성 테마에서 점 3개로 된 메뉴를 선택하고 **코드 편집** 을 선택합니다.

   ![Shopify 테마 개요 페이지 스크린샷](@images/smart-banner-sdk/shopify/1-overview.png)

4. `theme.liquid` 파일을 찾아 엽니다.

5. 테마의 `<head></head>` 태그에 [SDK loading snippet](/en/sdk/smart-banner#using-a-cdn)을 추가합니다.

   ![Shopify 테마 편집기 스크린샷](@images/smart-banner-sdk/shopify/2-loading-snippet.png)

6. 테마의 `<body>` 태그로 스크롤 다운하여 SDK 초기화 콜을 입력합니다. 이 콜은 반드시 앱토큰을 포함해야 합니다. `init` 메서드를 사용하여 SDK를 구성하려면 [SDK 초기화 옵션](/en/sdk/smart-banner/init-options)을 확인하시기 바랍니다.

   ```html
   <!-- Adjust Smart Banner SDK -->
   <div id="AdjSbRoot"></div>
   <script>
      AdjustSmartBanner.init({
         appToken: "YOUR_TOKEN",
         logLevel: "verbose",
         bannerParent: document.getElementById("AdjSbRoot"),
      });
   </script>
   <!-- Adjust Smart Banner SDK -->
   ```

   ![스마트 배너 SDK 초기화 옵션을 보여주는 Shopify 테마 편집기의 스크린샷](@images/smart-banner-sdk/shopify/3-init-options.png)

7. 변경 내용을 저장합니다.

<Callout type="note">

이 예시에서 스니펫은 ID가 `AdjSbRoot`인 새로운 구분자를 추가합니다. 해당 구분자는 스마트 배너의 상위\(parent\) 요소로 사용됩니다. Shopify 헤더가 스마트 배너를 오버레이 하는 것을 방지하려면 이를 반드시 설정해야 합니다. 배너 상위 요소에 스타일을 적용하거나 다른 상위 요소를 선택할 수 있습니다. 상위 요소를 추가하지 않은 경우, 스마트 배너가 `document.body`에 첨부됩니다.

</Callout>
