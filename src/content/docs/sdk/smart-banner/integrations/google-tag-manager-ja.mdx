AdjustスマートバナーSDKをGoogle タグ マネージャーに実装し、GTMコンテナでスマートバナーを配信できます。このガイドの手順に従って設定してください。

## Google タグ マネージャーにSDKを実装する {#integrate-the-sdk-in-google-tag-manager}

1. Google タグ マネージャーに移動します。

2. **Tags → New** を選択します。

3. **Tag Configuration** を選択し、 **Custom HTML** を選択します。

4. [HTML](/ja/sdk/smart-banner#using-a-cdn)フィールドに **SDK loading snippet** を追加し、 **Firing Triggers** フィールドが **Initialization \- All Pages** に設定されていることを確認します。

   ![GTMタグ設定画面のスクリーンショット](@images/smart-banner-sdk/gtm/1-gtm-configuration.png)

5. 別のカスタムHTMLタグを追加し、`<script></script>`タグの間にSDK初期化コールを入力します。このコールには`appToken`を含める必要があります。`init`メソッドを使用してSDKを設定するには、[SDKの初期化オプション](/ja/sdk/smart-banner/init-options)をご覧ください。

   ```js
   window.AdjustSmartBanner.init({ appToken: "TOKEN" });
   ```

   ![カスタムHTMLエディターとinitオプションのスクリーンショット](@images/smart-banner-sdk/gtm/2-init-options.png)

6. 画面の右上隅にあるプラス（`+`）アイコンを選択して、新しいトリガーを作成します。

7. **Trigger Configuration** に移動し、 **Page View** を選択して、各ページビューにスクリプトを読み込みます。全てのページで初期化をトリガーするか、特定のホスト名を入力するかを設定できます。

   ![トリガーの発火設定のスクリーンショット](@images/smart-banner-sdk/gtm/3-trigger-config.png)

8. トリガー設定を保存し、 **Tag Configuration** に戻ります。

9. 全てが正しく設定されていることを確認してから、GTMコンテナを公開します。
