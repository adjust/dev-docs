Adjust SDK는 디바이스 정보를 반환하는 헬프 메서드가 포함되어 있습니다. 해당 메서드를 사용하여 콜백에 세부 정보를 추가하고 리포트를 보완하시기 바랍니다.

Adjust 디바이스 ID \{\#adjust\-device\-identifier\}
----------------------------------------------------

<codeblock title="메서드 서명"> 

```cs
public static string GetAdid();
```

</CodeBlock> 

Adjust는 각 기기에 대해 고유한 ADID\(Adjust 기기 ID\)를 생성합니다. `GetAdid` 메서드를 호출하면 이 ID가 `string`로 반환됩니다.

```cs
string adid = Adjust.GetAdid();
```

Windows 광고 ID \{\#windows\-advertising\-identifier\}
---------------------------------------------------------

<codeblock title="메서드 서명"> 

```cs
public static string GetWindowsAdId();
```

</CodeBlock> 

Windows ADID\(Windows 광고 ID\)는 Windows 기기 용 특정 ID입니다. `GetWindowsAdId` 메서드를 호출하면 이 ID가 `string`로 반환됩니다.

```cs
var windowsAdid = Adjust.GetWindowsAdId();
```
