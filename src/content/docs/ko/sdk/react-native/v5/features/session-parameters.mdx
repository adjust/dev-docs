Adjust 대시보드에 [콜백 URL을 등록](https://help.adjust.com/ko/article/recommended-placeholders-callbacks)하면 SDK가 세션을 측정할 때 Adjust가 GET 요청을 콜백 URL로 보냅니다.

## 글로벌 콜백 파라미터 \{\#global\-callback\-parameters\}

<Callout type="note">

Adjust SDK는 글로벌 콜백 파라미터와 이벤트 콜백 파라미터를 병합합니다. 이벤트 콜백 파라미터는 글로벌 콜백 파라미터보다 우선시 됩니다. 즉, 이벤트와 글로벌 콜백 모두에 파라미터 키를 추가한 경우, SDK는 이벤트 파라미터를 전송합니다.

</Callout>

콜백 파라미터를 서버에 구성할 수 있습니다. 글로벌 파라미터 구성이 완료되면 SDK는 이를 콜백 URL에 추가합니다. 해당 정보는 BI 시스템에서 사용자의 인앱 행동을 분석하는 데 사용될 수 있습니다.

### 글로벌 콜백 파라미터 추가 {#add-global-callback-parameters}

<CodeBlock title="메서드 서명">

```ts
addGlobalCallbackParameter: (key: string, value: string) => void
```

</CodeBlock>

`string` 키\-값 인수와 함께 `addGlobalCallbackParameter` 메서드를 호출하여 이벤트에 콜백 파라미터를 추가합니다. 이 메서드를 여러 번 호출함으로써 여러 파라미터를 추가할 수 있습니다.

```jsx
Adjust.addGlobalCallbackParameter("key", "value");
```

### 글로벌 콜백 파라미터 삭제 {#remove-global-callback-parameters}

<CodeBlock title="메서드 서명">

```ts
removeGlobalCallbackParameter: (key: string) => void
```

</CodeBlock>

특정 글로벌 콜백 파라미터가 더 이상 필요하지 않은 경우 이를 삭제할 수 있습니다. 삭제하려면 파라미터 `key`를 `removeGlobalCallbackParameter` 메서드로 전달합니다.

```jsx
Adjust.removeGlobalCallbackParameter("key");
```

### 글로벌 콜백 파라미터 초기화 {#reset-global-callback-parameters}

<CodeBlock title="메서드 서명">

```ts
removeGlobalCallbackParameters: () => void
```

</CodeBlock>

글로벌 파라미터가 더 이상 필요하지 않은 경우 모두 삭제할 수 있습니다. 해당 기능 사용을 원하는 경우 `removeGlobalCallbackParameters` 메서드를 호출합니다.

```jsx
Adjust.removeGlobalCallbackParameters();
```

## 글로벌 파트너 파라미터 \{\#global\-partner\-parameters\}

[파트너 파라미터](https://help.adjust.com/ko/article/data-sharing-ad-network#map-parameters)를 추가하여 네트워크 파트너에게 추가 정보를 전송할 수 있습니다.

Adjust는 설정한 [외부 파트너](https://help.adjust.com/ko/article/integrated-partners)에게 파트너 파라미터를 전송합니다. 이러한 정보는 보다 세분화된 분석과 리타겟팅에 유용할 것입니다. 파트너에 대해 파라미터를 설정하고 활성화하면 Adjust 서버는 해당 파라미터를 전달합니다.

<Callout type="note">

파트너 파라미터는 기본 설정에 따라 로 데이터에 나타나지 않습니다. `{partner_parameters}` 플레이스홀더를 추가하여 단일 문자열로 수신할 수 있습니다.

</Callout>

### 글로벌 파트너 파라미터 추가 {#add-global-partner-parameters}

<CodeBlock title="메서드 서명">

```ts
addGlobalPartnerParameter: (key: string, value: string) => void
```

</CodeBlock>

`string` 키\-값 인수와 함께 `addGlobalPartnerParameter` 메서드를 호출하여 세션의 파트너 파라미터를 전송합니다. 이 메서드를 여러 번 호출함으로써 여러 파라미터를 추가할 수 있습니다.

```jsx
Adjust.addGlobalCallbackParameter("key", "value");
```

### 글로벌 파트너 파라미터 삭제 {#remove-global-partner-parameters}

<CodeBlock title="메서드 서명">

```ts
removeGlobalPartnerParameter: (key: string) => void
```

</CodeBlock>

특정 글로벌 파트너 파라미터가 더 이상 필요하지 않은 경우 이를 삭제할 수 있습니다. 삭제하려면 파라미터 키를 `removeGlobalPartnerParameter` 메서드로 전달합니다.

```jsx
Adjust.removeGlobalPartnerParameter("key");
```

### 글로벌 파트너 파라미터 초기화 {#reset-global-partner-parameters}

<CodeBlock title="메서드 서명">

```ts
removeGlobalPartnerParameters: () => void
```

</CodeBlock>

모든 글로벌 파트너 파라미터가 더 이상 필요하지 않은 경우 삭제할 수 있습니다. 해당 기능 사용을 원하는 경우 `resetGlobalPartnerParameters` 메서드를 호출합니다.

```jsx
Adjust.removeGlobalPartnerParameters();
```
