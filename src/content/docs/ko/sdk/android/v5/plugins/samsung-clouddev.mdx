Adjust SDK는 플러그인을 통해 [삼성 CloudDev SDK](https://developer.samsung.com/instant-plays/samsung-clouddev-sdk/get-started.html) 를 지원합니다.

## 시작에 앞서 {#before-you-begin}

1. 앱에 Adjust SDK를 연동하려면 [시작 가이드](/ko/sdk/android)를 참조하시기 바랍니다.
2. Samsung CloudDev SDK 사본을 삼성 담당자에게 요청하고 해당 SDK를 프로젝트에 추가하세요.

## 삼성 CloudDev 플러그인 설치하기 {#install-the-samsung-clouddev-plugin}

삼성 CloudDev 플러그인을 설치하는 방법은 두 가지가 있습니다.

### Maven으로 설치하기 {#install-with-maven}

Adjust SDK 선언 아래의 `build.gradle` 파일에 다음의 디펜던시를 추가합니다.

<CodeBlock title="build.gradle">

```groovy
implementation 'com.adjust.sdk:adjust-android-samsung-clouddev:$ANDROID_V5_VERSION'
```

</CodeBlock>

### JAR 또는 AAR로 추가하기 {#add-as-a-jar-or-aar}

Samsung CloudDev 플러그인은 [Adjust SDK 리포지토리](https://github.com/adjust/android_sdk/releases/latest)에서 Java Archive\(JAR\) 및 Android Archive\(AAR\) 형식으로 사용할 수 있습니다.
