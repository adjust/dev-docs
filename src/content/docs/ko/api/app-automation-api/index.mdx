Adjust App Automation API를 사용하면 앱을 생성 및 업데이트할 수 있으며, 해당 엔드포인트는 앱과 앱 설정 관리 과정을 자동화해줍니다.

대시보드에서 개별 앱을 생성하는 대신 캠페인에 대한 모든 필수 설정을 포함하는 템플릿 앱을 생성한 후, Adjust App Automation API를 사용해 템플릿 앱에 기반한 새 앱을 생성할 수 있습니다.

개별 신규 앱은 템플릿 앱 또는 선택한 다른 앱의 설정을 그대로 복제합니다. 템플릿 앱에 덮어쓰기 하거나 새 정보를 추가하기 위해 앱에 특화된 설정도 전달할 수 있습니다.

App Automation API를 사용하면 각 새 앱에 대해 요청을 1번만 생성하면 되기 때문에 시간을 크게 절약할 수 있습니다.

## 사용 사례 {#use-cases}

Adjust Automation API는 새 앱을 많이 생성하는 경우 유용합니다. 새 앱과 캠페인 출시를 위해 앱별 설정을 한다면 상당한 간접 비용이 발생하는데, Automation API를 사용하면 다음과 같은 효과를 얻을 수 있습니다.

1. 템플릿의 설정을 그대로 복제하여 앱 **생성** 과정을 간소화할 수 있습니다. Adjust의 테크니컬 어카운트 스페셜리스트 팀이 템플릿 설정을 도와드립니다.
2. 앱 **업데이트** 과정을 간소화할 수 있습니다. 파트너나 앱에 특화된 설정을 API로 전달하면 언제든지 앱을 업데이트할 수 있습니다.
3. 앱 생성 과정의 상태를 확인할 수 있습니다. 각 앱의 진행 단계를 정확히 파악할 수 있습니다.
4. 앱에 대한 정보를 프로그래매틱한 방식으로 확인할 수 있습니다. 대시보드에서 개별 앱의 링크를 확인하는 대신에 API를 사용해 정보를 얻을 수 있습니다.

## 시작하기 {#get-started}

### 요청 인증하기 {#authenticate-your-requests}

App Automation API를 사용하려면 콜을 인증해야 합니다. [인증 가이드](/ko/api/app-automation-api/authentication)에 나온 설명을 참조하시기 바랍니다.

### 새 앱 생성하기 {#create-new-apps}

<Callout type="note">

Adjust의 테크니컬 어카운트 스페셜리스트 팀이 템플릿 앱을 설정해드립니다. 담당 매니저에게 연락하시기 바랍니다.

</Callout>

[`/app` 엔드포인트](/ko/api/app-automation-api/app-endpoint)를 사용하면 앱에 대한 정보와 함께 POST 요청을 전송하여 앱을 생성할 수 있습니다. Adjust는 새 앱 정보를 수신하면 이에 대한 엔트리를 생성하고, 새 앱 토큰을 반환합니다.

파트너 모듈을 설정하려면 [파트너에 특화된 설정](/ko/api/app-automation-api/channel-setup)을 전달해야 합니다.

Adjust는 템플릿 앱에서 설정을 복사하여 새 앱에 이를 입력합니다.

### 새 앱의 상태 확인하기 {#check-the-status-of-your-new-app}

[`/status` 엔드포인트](/ko/api/app-automation-api/status-endpoint)로 GET 요청을 전송하면 새 앱의 상태를 확인할 수 있습니다.이는 템플릿 앱에서 설정을 복사한 새 앱에 대한 백그라운드 작업의 상태를 보여줍니다.

### 앱에 대한 정보 받기 {#get-information-about-your-app}

새 앱이 생성되면 [`/app/{adjust_app_token}` 엔드포인트](/ko/api/app-automation-api/app-endpoint#app-token-parameters)로 GET 요청을 전송하여 앱에 관한 정보를 조회할 수 있습니다.

앱의 스토어 ID와 플랫폼과 함께 [`/app` 엔드포인트](/ko/api/app-automation-api/app-endpoint)로 GET 요청을 전송하여 앱 정보를 받을 수도 있습니다.

### 앱 업데이트하기 {#update-your-app}

템플릿 앱의 설정을 변경한 경우, 새 정보로 앱을 업데이트할 수 있습니다. 변경할 설정과 함께 `force_update` 요청을 [`/app` 엔드포인트](/ko/api/app-automation-api/app-endpoint)로 전송하면 앱이 업데이트됩니다. 파트너 모듈을 설정하려면 [파트너에 특화된 설정](/ko/api/app-automation-api/channel-setup)으로 앱을 업데이트하시기 바랍니다.
