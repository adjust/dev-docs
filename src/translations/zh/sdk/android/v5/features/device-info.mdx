Adjust SDK 包含能够返回设备信息的辅助方法。用这些方法来向回传添加细节信息，优化报告。

## [Adjust 设备标识符](adjust-device-identifier)

<CodeBlock title="Method signature">

```java
public static void getAdid(final OnAdidReadListener onAdidReadListener)
```

</CodeBlock>

Adjust 会针对每台设备生成一个独特的 Adjust 设备 ID \(ADID\)。您可以通过向`Adjust.getAdid`方法传递回传方法来从 Adjust SDK 异步请求此信息。

<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getAdid { adid ->
   // Your callback function
}
```

</Tab>
<Tab title="Java" sync="java" >

```java
Adjust.getAdid(new OnAdidReadListener() {
    @Override
    public void onAdidRead(String adid) {
      // Your callback function
    }
});

```

</Tab>
</Tabs>

## [Google Play 服务广告 ID](google-play-services-advertising-id)

<CodeBlock title="Method signature">

```java
public static void getGoogleAdId(Context context, OnDeviceIdsRead onDeviceIdRead)
```

</CodeBlock>

Google Play 服务广告 ID \(GPS ADID\) 是安卓设备的独特标识符。

用户可以通过启用设备的 "选择停用广告个性化功能" 设置来停止分享自己的 GPS ADID。当用户启用该设置后，Adjust SDK 试图读取 GPS ADID 时会返回一串零。

您可以通过在后台线程中调用 `getGoogleAdId` 方法来获取该值。指定一个委托 \(delgate\) 函数，以访问 GPS ADID 值。

<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getGoogleAdId(this, object : OnDeviceIdsRead {
   override fun onGoogleAdIdRead(googleAdId: String) {}
})
```

</Tab>
<Tab title="Java" sync="java" >

```java
Adjust.getGoogleAdId(this, new OnDeviceIdsRead() {
    @Override
    public void onGoogleAdIdRead(String googleAdId) {}
});
```

</Tab>
</Tabs>

## [Amazon 广告主 ID](amazon-advertiser-id)

<CodeBlock title="Method signature">

```java
public static void getAmazonAdId(final Context context,final OnAmazonAdIdReadListener onAmazonAdIdReadListener)
```

</CodeBlock>

Amazon 广告 ID \(Amazon Ad ID\) 是安卓设备的独特标识符。您可以通过向`Adjust.getAmazonAdId`方法传递回传方法来从 Adjust SDK 异步请求此信息。

<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getAmazonAdId(applicationContext, object : OnAmazonAdIdReadListener {
   override fun onAmazonAdIdRead(amazonAdId: String) {}
   override fun onFail(message: String) {}
})
```

</Tab>
<Tab title="Java" sync="java" >

```java
Adjust.getAmazonAdId(getApplicationContext(), new OnAmazonAdIdReadListener() {
    @Override
    public void onAmazonAdIdRead(String amazonAdId) {
    }

    @Override
    public void onFail(String message) {
    }
});

```

</Tab>
</Tabs>

## [Adjust SDK 版本 getter](sdk-version-getter)

要读取 Adjust SDK 的版本，请将回传函数传递至`Adjust.getSdkVersion`方法。版本号在回传函数中以`String`形式提供。

<Tabs>
<Tab title="Kotlin" sync="kotlin">

```kotlin
Adjust.getSdkVersion {sdkVersion ->
   // Your callback function
}
```

</Tab>
<Tab title="Java" sync="java">

```java
Adjust.getSdkVersion(new OnSdkVersionReadListener() {
   @Override
   public void onSdkVersionRead(String sdkVersion) {
      // Your callback function
   }
})
```

</Tab>
</Tabs>
