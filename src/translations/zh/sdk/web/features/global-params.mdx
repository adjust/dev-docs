您可以通过注册统一回传和合作伙伴参数来随每次会话和事件发送来自 SDK 的更多信息。

## [统一回传参数](global-callback-parameters)

您在 Adjust 控制面板中[注册回传 URL](https://help.adjust.com/zh/article/set-up-callbacks)，SDK 记录到会话或事件后，Adjust 会向您的回传 URL 发送一个 GET 请求。

您可以为自己的服务器设置统一回传参数。配置好参数后，SDK 会将参数附加至您的回传 URL。您可以利用该信息，通过自己的 BI 系统分析用户应用内行为。

### [添加回传参数](add-callback-parameters)

<CodeBlock title="方法签名">

```js
function addGlobalCallbackParameters(params: Array<globalparams>): void;
```

</CodeBlock>

使用 `string` 键值对对象组成的数组调用 `addGlobalCallbackParameters` 方法，以此向会话和事件添加回传参数。您可以向数组中添加任意数量的对象。

```js
Adjust.addGlobalCallbackParameters([
   { key: "key1", value: "value1" },
   { key: "key2", value: "value2" },
]);
```

对于上述示例，回传 URL 形式如下：

<CodeBlock highlight="key1, value1, key2, value2">

```text
https://www.mydomain.com/callback?key1=value1&key2=value2
```

</CodeBlock>

### [移除回传参数](remove-callback-parameters)

<CodeBlock title="方法签名">

```js
function removeGlobalCallbackParameter(key: string): void;
```

</CodeBlock>

您可以移除不再需要的统一回传参数。为此，请将参数 key 传递至 `removeGlobalCallbackParameter` 方法。

```js
Adjust.removeGlobalCallbackParameter("key1");
```

### [清除所有回传参数](clear-all-callback-parameters)

```js
function clearGlobalCallbackParameters(): void;
```

如果您不再需要统一回传参数，则可以将所有参数一并移除。为此，请调用 `clearGlobalCallbackParameters` 方法。

```js
Adjust.clearGlobalCallbackParameters();
```

## [统一合作伙伴参数](global-partner-parameters)

您可以为 Adjust SDK 记录的每个会话和事件添加[合作伙伴参数](https://help.adjust.com/zh/article/data-sharing-ad-network#map-parameters)，向渠道合作伙伴发送额外的信息。

Adjust 可向您设置的[外部合作伙伴](https://help.adjust.com/zh/article/integrated-partners)发送合作伙伴参数。这些信息可用来进行更精细的数据分析，开展再营销活动。您设置好参数并为合作伙伴启用参数转发后，Adjust 服务器就会将这些参数转发给合作伙伴。

<Callout type="note">

合作伙伴参数不会默认出现在原始数据中。您可以添加`{partner_parameters}`占位符，在同一个字符串中接收参数。

</Callout>
