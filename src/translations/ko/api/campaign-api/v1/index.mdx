캠페인 API를 사용하면 링크, 하위 링크 및 파트너 정보를 얻을 수 있습니다. 새 링크를 생성하고 기존 링크를 새 파트너 정보로 업데이트하시기 바랍니다.

## [작동 방식](how-it-works)

1. **콜 인증**

   캠페인 API 엔드포인트를 사용하려면 Adjust API 토큰이 필요합니다. [인증 가이드](/en/api/campaign-api/authentication) 내 설명을 참조하여 해당 토큰을 API와 함께 사용하는 방법을 확인하시기 바랍니다.

2. **앱과 관련된 네트워크 링크 목록 수신**

   `trackers` 엔드포인트를 사용하여 앱과 관련된 모든 링크에 대한 세부 정보를 가져옵니다. [최상위 링크](/en/api/campaign-api/get-links) 또는 [하위 링크](/en/api/campaign-api/get-sublinks) 목록을 가져올 수 있습니다.

3. **새 네트워크 링크 생성**

   [`trackers` 엔드포인트에 POST 요청](/en/api/campaign-api/create-link)을 전송하여 앱에 새 링크를 추가할 수 있습니다. 이 엔드포인트는 최상위 링크와 하위 링크 생성에 사용됩니다.

4. **파트너 목록 수신**

   [`partners` 엔드포인트](/en/api/campaign-api/get-partners)를 사용하면 파트너 목록을 받을 수 있으며, 링크 업데이트 시 활용할 수 있습니다.

5. **새 파트너로 링크 업데이트**

   [`trackers` 엔드포인트에 PATCH 요청](/en/api/campaign-api/update-link)을 전송하여 기존 링크를 새 파트너와 비용 데이터 정보로 업데이트할 수 있습니다.
