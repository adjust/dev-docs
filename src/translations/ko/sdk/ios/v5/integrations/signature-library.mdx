Adjust SDK v5는 [SDK Signature\(SDK 서명\) 인증](https://help.adjust.com/ko/article/sdk-signature)을 기본적으로 지원합니다. SDK 서명 라이브러리를 이전에 연동한 경우 이것을 삭제하고 관련된 모든 코드를 제거해야 합니다.
