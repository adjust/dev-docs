Adjust SDKには、デバイス情報を返すヘルパーメソッドが含まれています。これらのメソッドを使用することでコールバックに詳細を追加し、レポートを改善させましょう。

## [AdjustのデバイスID](adjust-device-identifier)

<CodeBlock title="メソッドシグネチャー">

```objc
+ (void)adidWithCompletionHandler:(nonnull ADJAdidGetterBlock)completion;
```

</CodeBlock>

Adjustは各デバイスに固有のAdjust Device ID（ADID）を生成します。完了ハンドラーで`[Adjust adidWithCompletionHandler]`メソッドを呼び出すと、デバイスのADIDを返すことができます。SDKは情報を非同期で取得し、完了ハンドラーにパスします。

<Tabs>
<Tab title="Swift" sync="swift">

```swift
Adjust.adid { adid in
      // add your completion handler
};
```

</Tab>
<Tab title="Objective-C" sync="objc">

```objc
[Adjust adidWithCompletionHandler:^(NSString * _Nullable adid) {
      // add your completion handler
}];
```

</Tab>
</Tabs>
