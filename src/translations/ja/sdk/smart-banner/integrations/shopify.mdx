AdjustのスマートバナーSDKをShopifyに実装し、Shopifyのタグを使用してスマートバナーを配信できます。このガイドの手順に従って設定してください。

## [SDKをShopifyに実装する](integrate-the-sdk-in-shopify)

1. [https://admin.shopify.com](https://admin.shopify.com)でストア管理ツールに移動します。

2. **Online Store → Themes** を選択します。

3. 3つの点をクリックし、 **Edit code** を選択します。

   ![Shopifyテーマオーバービューページのスクリーンショット](@images/smart-banner-sdk/shopify/1-overview.png)

4. `theme.liquid`ファイルを開きます。

5. テーマの`<head></head>`タグ内に[SDK loading snippet](/en/sdk/smart-banner#using-a-cdn)を追加します。

   ![Shopifyテーマエディタのスクリーンショット](@images/smart-banner-sdk/shopify/2-loading-snippet.png)

6. テーマの`<body>`タグまでスクロールし、SDKの初期化コールを入力します。このコールにはappTokenを含める必要があります。`init`メソッドを使用してSDKを設定するには、[SDKの初期化オプション](/en/sdk/smart-banner/init-options)をご覧ください。

   ```html
   <!-- Adjust Smart Banner SDK -->
   <div id="AdjSbRoot"></div>
   <script>
      AdjustSmartBanner.init({
         appToken: "YOUR_TOKEN",
         logLevel: "verbose",
         bannerParent: document.getElementById("AdjSbRoot"),
      });
   </script>
   <!-- Adjust Smart Banner SDK -->
   ```

   ![スマートバナーSDK initオプションを表示したShopifyテーマエディタのスクリーンショット](@images/smart-banner-sdk/shopify/3-init-options.png)

7. 変更を保存します。

<Callout type="note">

この例のスニペットには、ID `AdjSbRoot`の新しい区切り線が追加されています。この区切り線は、スマートバナーの親として使用されます。これを設定し、Shopifyのヘッダーがスマートバナーに重ならないようにする必要があります。バナーの親にスタイルを適用するか、別の親要素を選択することができます。親要素を追加しない場合、スマートバナーは`document.body`に付与されます。

</Callout>
