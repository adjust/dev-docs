Adjust SDK v5は、[SDKシグネチャーの検証](https://help.adjust.com/ja/article/sdk-signature)をネイティブでサポートしています。以前にSDKシグネチャーライブラリを連携した場合は、アンインストールし、関連するコードを全て削除する必要があります。
